# 第六版运行调试归档说明

本轮版本目标：

```text
加入 chat wrapper，让 MiniGPT 可以用 system/user/assistant 的对话模板运行 one-shot 或交互式聊天。
```

本轮不改变 Transformer 模型结构，重点是把“基础语言模型续写”包装成“简易助手回复”。

---

# 1. 01-unit-tests.png

对应命令：

```powershell
python -B -m unittest discover -s tests -v
```

结果：

```text
chat、attention、prediction、tokenizer、dataset、history、model 测试全部通过。
```

意义：

```text
证明 v6 新增的 chat prompt、停止符和上下文截断逻辑没有破坏已有训练/生成能力。
```

# 2. 02-chat-train-smoke.png

对应命令：

```powershell
python -B scripts/train.py --device cpu --out-dir tmp/v6-chat-smoke --max-iters 2 --eval-interval 1 --eval-iters 1 --batch-size 2 --block-size 32 --n-layer 1 --n-head 1 --n-embd 32 --sample-tokens 20
```

结果：

```text
训练可以正常完成并写出 checkpoint、tokenizer、metrics 和 sample 产物。
```

意义：

```text
提供一个真实 checkpoint 给 chat 脚本使用。
```

---

# 3. 03-chat-one-shot.png

对应命令：

```powershell
python -B scripts/chat.py --device cpu --checkpoint tmp/v6-chat-smoke/checkpoint.pt --message "解释 token 是什么" --max-new-tokens 30 --out tmp/v6-chat-smoke/transcript.json
```

结果：

```text
脚本输出 tokenizer、context token 数量、是否截断、assistant 回复，并写出 transcript.json。
```

意义：

```text
证明 MiniGPT 可以通过 chat prompt 模板完成最小聊天入口。
```
