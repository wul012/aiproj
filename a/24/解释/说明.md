# MiniGPT v24 验证说明

## 本版标记

`v24.0.0`：新增 model cards。

简单解释：在 v23 的单 run experiment card 基础上，v24 可以从 `registry.json` 和多张 `experiment_card.json` 汇总出项目级 `model_card.json`、`model_card.md`、`model_card.html`，用于展示整个 MiniGPT 实验系列的最佳 run、覆盖度、限制和后续建议。

## 本版新增内容

- 新增 `src/minigpt/model_card.py`。
- 新增 `scripts/build_model_card.py`。
- 新增 `tests/test_model_card.py`。
- model card 自动从 registry run path 查找 `experiment_card.json`。
- model card 汇总 intended use、limitations、top runs、coverage、quality counts、tag counts 和 recommendations。
- model card 支持 JSON、Markdown、HTML 三种输出。

## 关键命令

```powershell
python -B -m unittest discover -s tests -v
python -B -m compileall src scripts tests
python scripts/register_runs.py tmp\v24-model-card-smoke\run-a tmp\v24-model-card-smoke\run-b --name ready-candidate --name review-baseline --out-dir tmp\v24-model-card-smoke\registry
python scripts/build_experiment_card.py --run-dir tmp\v24-model-card-smoke\run-a --registry tmp\v24-model-card-smoke\registry\registry.json --out-dir tmp\v24-model-card-smoke\run-a
python scripts/build_model_card.py --registry tmp\v24-model-card-smoke\registry\registry.json --out-dir tmp\v24-model-card-smoke\model-card
powershell -ExecutionPolicy Bypass -File scripts\playwright_chrome_smoke.ps1 -UrlOrPath tmp\v24-model-card-smoke\model-card\model_card.html -Out tmp\v24-model-card-smoke\model-card.png
```

## 本版意义

这一版把项目从“单个实验能说明自己”推进到“整个实验系列能说明自己”。`registry` 负责机器可比对，`experiment_card` 负责单 run 复盘，`model_card` 负责项目级展示和交付。
