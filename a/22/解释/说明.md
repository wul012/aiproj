# MiniGPT v22 验证说明

## 本版标记

`v22.0.0`：新增 registry leaderboards。

简单解释：registry 现在会按 `best_val_loss` 给多个 run 排名，计算每个 run 相对当前最佳 run 的 loss 差值，并把排行榜同步写入 JSON、CSV、SVG、HTML 和命令行输出。

## 本版新增内容

- `build_run_registry` 新增 `loss_leaderboard`。
- 每个 run 新增 `best_val_loss_rank`、`best_val_loss_delta`、`is_best_val_loss`。
- `registry.csv` 新增排名、delta 和 best 标记列。
- `registry.svg` 每行展示 rank 和 loss delta。
- `registry.html` 新增 Rank 列、Loss Leaderboard 面板、rank/delta 排序项。
- `scripts/register_runs.py` 终端输出新增 `loss_leaderboard`。

## 本版意义

这一版让实验登记表具备“横向判断”的能力。指标不再只是散落在每一行，而是会自动形成排名和差值，方便快速选择 baseline、candidate 或需要复查的数据质量异常 run。
