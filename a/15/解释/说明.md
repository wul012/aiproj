# v15 归档说明

v15 新增 dataset quality 能力，让训练语料拥有稳定 fingerprint，并能在准备数据和训练产物中看到基础质量提示。

## 截图说明

`01-unit-tests.png`：展示 dataset quality 新测试和既有回归测试通过。

`02-prepare-quality-smoke.png`：展示 `prepare_dataset.py` 生成 `dataset_quality.json` 和 `dataset_quality.svg`。

`03-quality-json-check.png`：检查 quality report 的 status、fingerprint、warning、duplicate source 和 repeated line。

`04-train-dashboard-quality.png`：检查训练目录、dashboard、playground 和 manifest 都能携带 dataset quality 信息。

`05-docs-check.png`：检查 README、代码讲解记录和 v15 归档说明是否存在。

## 本版重点

- 新增 `src/minigpt/data_quality.py`
- 新增 `tests/test_data_quality.py`
- `data_prep.py` 增加 corpus/source sha256
- `prepare_dataset.py` 自动输出 quality JSON/SVG
- `train.py` 复制或生成 dataset quality artifact
- Dashboard、Playground、Manifest 接入 dataset quality
- 代码讲解记录合并为一篇 `30-v15-dataset-quality.md`

## 一句话总结

v15 让数据不只“能被训练”，还开始“能被识别、比较和检查”。
