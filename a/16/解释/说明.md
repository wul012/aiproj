# v16 归档说明

v16 新增固定 prompt eval suite，让不同 checkpoint 可以运行同一组提示词，并导出可比较的 JSON、CSV 和 SVG 报告。

`03-eval-suite-json-check.png`：检查 `eval_suite.json`、`eval_suite.csv` 和 `eval_suite.svg`。

`04-dashboard-playground-eval-suite.png`：检查 dashboard/playground 能识别 eval suite 产物。

`05-docs-check.png`：检查 README、代码讲解记录和 v16 归档说明是否存在。

## 本版重点

- 新增 `data/eval_prompts.json`
- 新增 `src/minigpt/eval_suite.py`
- 新增 `scripts/eval_suite.py`
- 新增 `tests/test_eval_suite.py`
- Dashboard 增加 Eval Suite 区块
- Playground 增加 eval suite artifact 链接

## 一句话总结

v16 让 checkpoint 的生成表现可以用同一套 prompts 反复评估，为后续 run registry 和多版本对比打基础。
