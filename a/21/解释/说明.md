# MiniGPT v21 验证说明

## 本版标记

`v21.0.0`：新增 registry annotations。

简单解释：每个 run 目录可以放一个 `run_notes.json`，写入人工备注和 tags。`register_runs.py` 会自动读取这些注释，并把它们写进 registry JSON/CSV/SVG/HTML。HTML 页面也能搜索 tag 或 note。

## 本版新增内容

- `RegisteredRun` 新增 `note` 和 `tags` 字段。
- `summarize_registered_run` 自动读取 `run_notes.json`。
- `build_run_registry` 新增 `tag_counts`。
- `registry.csv` 新增 `note`、`tags` 列。
- `registry.svg` 展示 note/tag 摘要。
- `registry.html` 新增 Notes 列，tags 渲染为 chip，搜索范围包含 note/tag。
- `scripts/register_runs.py` 终端输出新增 `tag_counts`。

## 本版意义

这一版让 registry 能记录人的实验判断。模型指标只能说明一部分问题，notes 和 tags 可以标记 baseline、candidate、review、keep 等语义，方便以后复盘和筛选。
