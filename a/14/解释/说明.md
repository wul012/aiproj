# v14 归档说明

v14 新增 run manifest 能力，让每次 MiniGPT 训练都自动记录代码版本、环境、数据来源、模型配置、训练指标和产物索引。

## 截图说明

`01-unit-tests.png`：展示 manifest 新测试和既有回归测试通过。

`02-train-manifest-smoke.png`：展示 `scripts/train.py` 训练结束后写出 `run_manifest.json` 和 `run_manifest.svg`。

`03-manifest-json-check.png`：检查 manifest 中的 schema、Git、环境、数据、模型、训练、结果和 artifact 字段。

`04-dashboard-playground-manifest.png`：检查 dashboard/playground 能识别并链接 manifest 产物。

`05-docs-check.png`：检查 README、代码讲解记录和 v14 归档说明是否存在。

## 本版重点

- 新增 `src/minigpt/manifest.py`
- 新增 `tests/test_manifest.py`
- `scripts/train.py` 自动写出 `run_manifest.json` 和 `run_manifest.svg`
- Dashboard 增加 Run Manifest 区块和 manifest artifact
- Playground 增加 manifest 链接
- README 更新到 v14
- 代码讲解记录合并为一篇 `29-v14-run-manifest.md`

## 一句话总结

v14 给每次训练补上实验身份证，为后续数据 fingerprint、质量检查和实验注册表打基础。
