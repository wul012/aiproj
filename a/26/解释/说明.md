# MiniGPT v26 验证说明

## 本版标记

`v26.0.0`：新增 release bundle。

简单解释：在 registry、experiment card、model card、project audit 之上，v26 新增最终交付证据包，输出 `release_bundle.json`、`release_bundle.md`、`release_bundle.html`。它汇总 release 状态、审计分数、最佳 run、top runs、证据产物和建议。

## 本版新增内容

- 新增 `src/minigpt/release_bundle.py`。
- 新增 `scripts/build_release_bundle.py`。
- 新增 `tests/test_release_bundle.py`。
- release bundle 自动汇总 registry、model card、project audit。
- release bundle 根据 audit 状态生成 `release-ready`、`review-needed`、`blocked`、`needs-audit`。
- release bundle 收集 registry/model card/project audit 的 JSON、Markdown、HTML、CSV、SVG 等 evidence artifacts。

## 本版意义

这一版让项目从“能自检”推进到“能交付”。`release_bundle` 是 registry、model card、project audit 的统一入口，适合和版本 tag 一起作为阶段性成果保存。
