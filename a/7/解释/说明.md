# 第七版运行调试归档说明

本轮版本目标：

```text
加入 model report，让 MiniGPT 可以导出 checkpoint 的模型配置、参数分布、每层参数、关键张量形状和 SVG 架构图。
```

本轮不改变训练目标，重点是让模型结构变得可解释、可检查。

---

# 1. 01-unit-tests.png

对应命令：

```powershell
python -B -m unittest discover -s tests -v
```

结果：

```text
model_report、chat、attention、prediction、tokenizer、dataset、history、model 测试全部通过。
```

意义：

```text
证明 v7 新增的参数统计、张量形状和 SVG 导出工具没有破坏已有能力。
```

---

# 2. 02-model-report-train-smoke.png

对应命令：

```powershell
python -B scripts/train.py --device cpu --out-dir tmp/v7-model-report-smoke --max-iters 2 --eval-interval 1 --eval-iters 1 --batch-size 2 --block-size 32 --n-layer 1 --n-head 1 --n-embd 32 --sample-tokens 20
```

结果：

```text
训练可以正常完成并写出 checkpoint、tokenizer、metrics 和 sample 产物。
```

意义：

```text
提供一个真实 checkpoint 给 inspect_model.py 使用。
```

---

# 3. 03-inspect-model.png

对应命令：

```powershell
python -B scripts/inspect_model.py --device cpu --checkpoint tmp/v7-model-report-smoke/checkpoint.pt --sequence-length 16 --out-dir tmp/v7-model-report-smoke/model_report
```

结果：

```text
脚本输出总参数量、参数分组、关键张量形状，并写出 model_report.json 和 model_architecture.svg。
```

意义：

```text
可以直接观察 checkpoint 的结构和参数量分布。
```
