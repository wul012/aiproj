# MiniGPT v20 验证说明

## 本版标记

`v20.0.0`：新增 registry saved views 与可见行 CSV 导出。

简单解释：v19 让 `registry.html` 可以搜索、筛选和排序；v20 把当前视图写进 URL hash，并允许导出当前可见行 CSV，方便分享某个实验视图或把筛选结果交给表格工具继续分析。

## 本版新增内容

- `registry.html` 新增 `Share` 按钮，复制或保留当前视图链接。
- `registry.html` 新增 `CSV` 按钮，只导出当前可见行。
- 搜索词、质量筛选、排序字段和升降序方向会写入 URL hash。
- 打开带 hash 的链接时，页面会自动恢复控件状态并重新筛选。
- CSV 导出优先使用 `innerText`，让带 `<br>` 的表格单元格导出后更易读。
- 单测覆盖 Share、CSV、URL 状态和导出脚本关键标记。

## 本版意义

这一版继续把实验管理能力做实。一个筛选后的 registry 视图现在可以变成链接，也可以导出成 CSV，适合讨论、复盘和二次分析。
