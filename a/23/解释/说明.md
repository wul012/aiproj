# MiniGPT v23 验证说明

## 本版标记

`v23.0.0`：新增 experiment cards。

简单解释：单个 run 目录现在可以生成 `experiment_card.json`、`experiment_card.md`、`experiment_card.html`。实验卡会汇总 run 状态、备注和 tags、数据质量、训练配置、评估结果、registry 排名、产物清单和后续建议。

## 本版新增内容

- 新增 `src/minigpt/experiment_card.py`。
- 新增 `scripts/build_experiment_card.py`。
- 新增 `tests/test_experiment_card.py`。
- `manifest.py` 把 experiment card 三个输出纳入 artifact 规格。
- `dashboard.py` 能识别 experiment card 产物。
- `playground.py` 增加 experiment card 命令和链接。
- `registry.py` 把 experiment card 纳入 artifact count，并在 HTML Links 中显示 `card` 链接。

## 关键命令

```powershell
python -B -m unittest discover -s tests -v
python -B -m compileall src scripts tests
python scripts/register_runs.py tmp\v23-experiment-card-smoke\run-a tmp\v23-experiment-card-smoke\run-b --name card-candidate --name baseline --out-dir tmp\v23-experiment-card-smoke\registry
python scripts/build_experiment_card.py --run-dir tmp\v23-experiment-card-smoke\run-a --registry tmp\v23-experiment-card-smoke\registry\registry.json --out-dir tmp\v23-experiment-card-smoke\run-a
powershell -ExecutionPolicy Bypass -File scripts\playwright_chrome_smoke.ps1 -UrlOrPath tmp\v23-experiment-card-smoke\run-a\experiment_card.html -Out tmp\v23-experiment-card-smoke\experiment-card.png
```

## 本版意义

这一版把单个 run 的“交付说明”补齐了。registry 负责横向比较多个实验，experiment card 负责把一个实验浓缩成可读、可传、可复盘的卡片，适合后续写报告、答辩或作品集展示。
