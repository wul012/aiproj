# 第四版运行调试归档说明

本轮版本目标：

```text
加入 attention inspection，让 MiniGPT 可以导出指定 prompt、layer、head 的注意力矩阵 JSON 和 SVG 热力图。
```

本轮不改变训练目标，不引入 Web UI，先把 GPT 核心机制中的 self-attention 做成可观察学习材料。

---

# 1. 01-unit-tests.png

对应命令：

```powershell
python -B -m unittest discover -s tests -v
```

结果：

```text
attention、tokenizer、dataset、history、model 测试全部通过。
```

意义：

```text
证明 attention capture 能返回每层矩阵，且 causal mask 仍然屏蔽未来 token。
```

# 2. 02-attention-train-smoke.png

对应命令：

```powershell
python -B scripts/train.py --device cpu --out-dir tmp/v4-attention-smoke --max-iters 2 --eval-interval 1 --eval-iters 1 --batch-size 2 --block-size 16 --n-layer 1 --n-head 2 --n-embd 32 --sample-tokens 20
```

结果：

```text
训练可以正常完成并写出 checkpoint.pt、tokenizer.json、metrics.jsonl、loss_curve.svg 和 sample.txt。
```

意义：

```text
提供一个真实 checkpoint 给 attention inspection 使用。
```

# 3. 03-attention-export.png

对应命令：

```powershell
python -B scripts/inspect_attention.py --device cpu --checkpoint tmp/v4-attention-smoke/checkpoint.pt --prompt 人工智能模型 --layer 0 --head 0 --out-dir tmp/v4-attention-smoke/attention
```

结果：

```text
脚本输出 token 列表、attention.json、attention.svg 和最后一个 token 的 top attention links。
```

意义：

```text
self-attention 矩阵已经可以从模型中导出并解释。
```
