# v12 归档说明

v12 新增本地 Playground Server，让 `playground.html` 可以通过 `/api/generate` 调用本地 MiniGPT checkpoint 生成文本。

## 截图说明

`01-unit-tests.png`：展示 server API、playground UI 和既有回归测试通过。

`02-server-train-smoke.png`：展示为了 server smoke 准备的小 checkpoint 训练过程。

`03-serve-playground.png`：展示本地 server 启动、`/api/health` 和 `/api/generate` 请求成功。

`04-server-artifacts-check.png`：检查 server smoke 的 checkpoint、tokenizer、playground、health 和 generate 输出结构。

`05-docs-check.png`：检查 README、代码讲解记录和 v12 归档说明是否存在。

## 本版重点

- 新增 `src/minigpt/server.py`
- 新增 `scripts/serve_playground.py`
- 新增 `tests/test_server.py`
- `playground.html` 增加 live generate 区块
- README 增加 v12、server 命令、项目结构和截图说明
- 代码讲解记录增加 `25-v12-playground-server.md` 和 `26-version-12-tests-docs.md`

## 一句话总结

v12 把 MiniGPT Playground 从静态学习页面推进成一个可以调用本地模型的实验入口。
