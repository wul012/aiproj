# v13 归档说明

v13 新增数据准备与数据报告能力，让 MiniGPT 的训练语料可以从多个 `.txt` 文件合并、统计并复用于训练。

## 截图说明

`01-unit-tests.png`：展示数据准备测试和既有回归测试通过。

`02-prepare-dataset.png`：展示 `scripts/prepare_dataset.py` 合并多个文本文件，并导出 `corpus.txt`、`dataset_report.json`、`dataset_report.svg`。

`03-train-prepared-data.png`：展示 `scripts/train.py --prepared-data ...` 可以直接从准备好的 corpus 训练。

`04-dataset-artifacts-check.png`：检查 dataset report、prepared corpus、训练产物、dashboard/playground 入口是否存在。

`05-docs-check.png`：检查 README、代码讲解记录和 v13 归档说明是否存在。

## 本版重点

- 新增 `src/minigpt/data_prep.py`
- 新增 `scripts/prepare_dataset.py`
- 新增 `tests/test_data_prep.py`
- `scripts/train.py` 支持 `--prepared-data` 和 `--data-dir`
- Dashboard 和 Playground 增加 dataset report 入口
- README 增加 v13、数据准备命令、项目结构和截图说明
- 代码讲解记录增加 `27-v13-dataset-preparation.md` 和 `28-version-13-tests-docs.md`

## 一句话总结

v13 把训练数据变成可检查、可报告、可复用的工程产物，为后续更大的语料训练和实验管理打基础。
