# 第八版运行调试归档说明

本轮版本目标：

```text
加入静态 dashboard，把训练、评估、模型结构、预测、attention、chat 和样例文本汇总到一个 dashboard.html。
```

本轮不改变模型训练逻辑，重点是把实验产物整理成可复盘、可展示的本地报告页面。

---

# 1. 01-unit-tests.png

对应命令：

```powershell
python -B -m unittest discover -s tests -v
```

结果：

```text
dashboard、model_report、chat、attention、prediction、tokenizer、dataset、history、model 测试全部通过。
```

意义：

```text
证明 v8 新增的 HTML 汇总、路径链接、转义和 warnings 逻辑没有破坏已有能力。
```
