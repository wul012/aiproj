# MiniGPT v19 验证说明

## 本版标记

`v19.0.0`：新增 registry HTML 交互控件。

简单解释：v18 让 registry 可以导出 `registry.html`；v19 继续增强这个页面，增加搜索、质量筛选、排序、升降序切换和可见行计数，让多个实验 run 更容易浏览和比较。

## 本版新增内容

- `registry.html` 的 run 行增加 `data-search`、`data-quality`、`data-best-val`、`data-params`、`data-artifacts`、`data-eval` 等属性。
- HTML 顶部增加搜索框、质量筛选下拉框、排序字段下拉框、升降序按钮和计数器。
- 浏览器端原生 JavaScript 根据搜索词、质量状态和排序字段即时更新表格。
- 数值排序会把缺失值放到最后，避免 `missing` 影响 loss、参数量和 artifact 排序。
- 单测覆盖控件存在性、质量选项生成、行数据属性和 HTML 转义。
- Playwright 继续使用本机 Google Chrome (`--channel chrome`) 做浏览器级验证。

## 本版意义

这一版继续加强实验管理能力。随着 run 数量增加，静态表格会越来越难扫；搜索、筛选和排序让 registry 更接近真实工程里的实验追踪页面。
