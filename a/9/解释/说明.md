# 第九版运行调试归档说明

本轮版本目标：

```text
加入 run comparison，把多个 MiniGPT 实验的 loss、perplexity、参数量和模型配置横向比较，并导出 JSON/CSV/SVG。
```

本轮不改变训练逻辑，重点是补上多实验管理能力。
