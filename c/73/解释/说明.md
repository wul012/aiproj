# v73 运行截图和解释

## 本版主题

v73 新增 training scale run comparison。它读取多份 v72 `training_scale_run.json`，比较 allowed/blocked、gate pass/warn/fail、batch started/skipped、readiness score delta 和相对 baseline 的变化说明。

本版不执行训练，不重新运行 gate 或 batch，也不替代 run registry。它只比较已有 gated scale run 总控产物。

## 关键命令

```powershell
python -m unittest tests.test_training_scale_run_comparison
python -m unittest discover -s tests
python -m compileall src tests scripts
python scripts/compare_training_scale_runs.py tmp\v73-smoke\scale-run-allowed\training_scale_run.json tmp\v73-smoke\scale-run-blocked\training_scale_run.json --name allowed --name blocked --baseline allowed --out-dir tmp\v73-smoke\scale-run-comparison
python scripts/compare_training_scale_runs.py tmp\v73-smoke\scale-run-blocked\training_scale_run.json tmp\v73-smoke\scale-run-allowed\training_scale_run.json --name blocked --name allowed --baseline blocked --out-dir tmp\v73-smoke\scale-run-comparison-blocked-baseline
powershell -ExecutionPolicy Bypass -File scripts/playwright_chrome_smoke.ps1 -UrlOrPath tmp\v73-smoke\scale-run-comparison\training_scale_run_comparison.html -Out c\73\图片\05-playwright-training-scale-run-comparison-html.png
```

## 证据说明

`training_scale_run_comparison.json` 是完整比较证据，包含 baseline、runs、baseline deltas、summary、best_by_readiness 和 recommendations。

`training_scale_run_comparison.csv` 是横向表格证据，适合查看 allowed_delta、readiness_delta、gate_relation 和 batch_relation。

`training_scale_run_comparison.html` 是浏览器审阅入口，方便快速看哪些 run 被 gate 阻止、哪些到达 batch 层。
