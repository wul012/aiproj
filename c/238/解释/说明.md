# v238 dependabot automation 运行说明

## 本版主题

v238 给 aiproj 增加 Dependabot 自动更新入口。

本版采用“4 项目加 dependabot.yml”的建议中对 aiproj 最低风险、收益明确的部分：监控 GitHub Actions 和 Python 依赖，让依赖更新由 GitHub 自动开 PR。

## 本版边界

本版不自动合并依赖更新。

本版不改变 CI workflow、依赖版本约束和测试命令。

本版只新增配置、轻量测试、文档和运行证据。

## Tag 含义

`v238.0.0` 表示 dependabot automation 版本。

这版不是模型能力提升，而是仓库维护自动化补强。

## 一句话总结

v238 让 aiproj 的 GitHub Actions 和 Python 依赖更新有了自动 PR 入口，并用测试保护配置不被误删或弱化。
