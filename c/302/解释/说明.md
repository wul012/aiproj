# v302 运行截图说明

## 本版主题

v302 把 training portfolio comparison 的 CI regression review summary 继续传入 training portfolio batch。目标是让 batch 总览、HTML 报告和 CLI stdout 直接显示 CI-regressed portfolios，而不是只能打开嵌套 comparison JSON 查看。

本版不改变训练、比较或 maturity narrative 判定，只增强 batch 层的证据可见性。

## tag 说明

`v302.0.0` 的含义是：MiniGPT v302 training portfolio batch CI regression summary。

这一版把 `maturity_ci_regression_count`、`maturity_ci_regression_names` 和 `best_score_ci_regressed` blocker reason 提升到 batch 报告层。

一句话总结：v302 让批量训练组合入口也能直接看见 CI regression review 风险。
