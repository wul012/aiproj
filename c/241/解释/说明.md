# v241 project audit coverage gate context 运行说明

## 本版主题

v241 把 v240 的 coverage gate 结果接入 project audit。

`scripts/audit_project.py` 现在可以通过 `--test-coverage-report` 读取 `test_coverage_report.json`，并在 audit summary、checks、context、Markdown/HTML 中展示 coverage gate 状态。

## 关键改动

- `project_audit_contexts.py` 新增 coverage check/context helper。
- `project_audit.py` 新增 `test_coverage_report_path` 参数、自动发现、summary 字段和 audit check。
- `project_audit_artifacts.py` 在 Markdown/HTML 里展示 coverage 状态和百分比。
- `scripts/audit_project.py` 新增 `--test-coverage-report`。
- `tests/test_project_audit.py` 新增 coverage pass/fail/missing 语义测试。

## 本版边界

coverage gate 进入 project audit 后仍是治理证据，不是模型质量证据。

project audit 对 coverage 缺失或失败给 `warn`，不直接替代 CI 的覆盖率失败退出码。
