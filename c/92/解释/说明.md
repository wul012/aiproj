# v92 training scale workflow report utility migration 运行截图说明

## 本版主题

v92 继续 `report_utils` 收口路线，把 v75 的 `training_scale_workflow.py` 迁移到公共报告工具上。

这次仍然是轻量、定向质量优化：只回收 workflow 里的重复 JSON 写出、Markdown/HTML 转义和 list/dict 归一化 helper，不改变 profiles、baseline、allow_warn、allow_fail、decision_require_gate_pass 或 decision_require_batch_started 的业务语义。

迁移后，plan、gate、run、comparison、decision、workflow 六层训练规模证据共享同一套报告基础设施。

## 关键代码

```text
src/minigpt/training_scale_workflow.py
src/minigpt/report_utils.py
tests/test_training_scale_workflow.py
tests/test_report_utils.py
scripts/run_training_scale_workflow.py
```

`training_scale_workflow.py` 仍然负责串联 planner、gated run、run comparison 和 run decision。变化只在 workflow 自己写 JSON/Markdown/HTML 时使用公共 helper。
