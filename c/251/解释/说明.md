# v251 portfolio comparison review split 运行截图说明

## 本版主题

v251 做 training portfolio comparison 的收束型拆分。

v250 把 coverage regression review 接进 portfolio comparison 后，`training_portfolio_comparison.py` 的 review action 和 recommendation 逻辑已经明显变厚。v251 不新增展示层，也不改变 JSON/CSV/Markdown/HTML 输出契约，而是把 review 规则迁到独立模块。

## 证据边界

v251 是 contract-preserving refactor。

它不改变 portfolio comparison 的字段、输出文件名、CLI 行为、review action reason、severity 或 recommendation 文案。

它只改变职责边界：

```text
training_portfolio_comparison.py
  artifact loading
  portfolio summary
  baseline delta
  comparison summary

training_portfolio_comparison_review.py
  review statuses
  coverage regression predicate
  review actions
  recommendations
```

## 一句话总结

v251 把 portfolio comparison 的 review 规则从主比较器里拆出来，让 v250 新增的 coverage blocker 能继续演进而不把 comparison builder 再次撑大。
