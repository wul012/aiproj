# v254 run decision batch review propagation 运行证据

## 本版目标

v254 继续 v253 的 batch comparison review 下游传播，把 scale-run comparison 中已经聚合的 batch review/blocker/coverage regression 摘要传入 `training_scale_run_decision`。

本版不新增训练流程、不改变 gate 规则，也不默认阻止带 review action 的候选 run。它只在选中 run 带 batch blocker action 时，把决策状态从 `ready` 降为 `review`，确保 execute command 不被误读为完全干净。

## 关键改动

- `src/minigpt/training_scale_run_decision.py`：decision summary 新增 selected batch review status、选中 run 的 review/blocker/coverage 计数、comparison 级别的 batch review 总数和 blocker reasons。
- `src/minigpt/training_scale_run_decision_artifacts.py`：CSV、Markdown、HTML 输出同步展示这些字段。
- `tests/test_training_scale_run_decision.py`：验证普通 review action 能被带入 recommendation，验证 blocker action 会把 pass-gate run 降为 `review`。
- `README.md`、`c/README.md`、代码讲解索引：更新 v254 版本定位。

## 证据边界

这些截图证明 v254 的执行决策证据链能保留上游 batch comparison 风险。它们不是模型能力提升证明，也不是实际大规模训练结果。

## 一句话总结

v254 让执行候选决策不再只看 readiness 分数，也能看到被选中 run 背后的 batch comparison review/blocker 风险。
