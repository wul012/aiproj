# v113 运行截图说明：Request History Core Split

## 本版主题

v113 把 `server.py` 中的 request-history 核心逻辑抽到 `src/minigpt/request_history.py`，覆盖 JSONL 写入/读取、过滤、详情查询、query 参数解析和 CSV 导出。

这版属于轻量、定向的质量优化：不改 `/api/request-history`、`/api/request-history-detail`、playground、推理生成或 pair artifact 行为，只减少 `server.py` 的职责宽度。
