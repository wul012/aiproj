# v225 training scale run numeric helper consolidation 运行说明

## 本版目标

v225 延续数值 helper 收口，把 training-scale run decision / comparison / promotion index / promoted comparison / promoted decision 里的 `_int()` 接到 `report_utils.number_or_default()`。

## 边界

本版不改变比较、索引或决策 schema，不改变 suite guard 逻辑，不改变 promote/review/block 规则。

它只收敛重复的默认归零转换，保留这些模块对“无法读取时按 0 处理”的保守语义。

`c/225` 保存最终截图和本说明。临时测试日志、source encoding 输出和 smoke 文本保存在 `runs/` 下，提交前按清理门禁删除。

## Tag 含义

`v225.0.0` 表示 training-scale run / promoted decision 链路开始复用 `report_utils` 的公共默认归零 helper，减少局部重复定义。

## 一句话总结

v225 把默认归零数值 helper 收口推进到训练规模 run、比较和晋级证据链，让这些层共享统一的转换规则。
