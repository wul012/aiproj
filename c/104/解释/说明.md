# v104 运行截图与解释

## 本版主题

v104 是 data preparation report-utils migration。

本版把 `src/minigpt/data_prep.py` 里的通用报告 helper 迁移到 `report_utils`，让 data preparation 复用公共 UTC 时间、JSON 写出、HTML 转义和 dict 归一化工具。

本版不改变：

- source discovery 规则。
- 文本归一化规则。
- prepared corpus 内容生成规则。
- dataset report 字段结构。
- dataset version manifest 字段结构。
- dataset quality 的调用和交接方式。
- dataset report SVG 或 dataset version HTML 布局。
- run manifest、dataset card、experiment card、model card、project audit、release bundle 或 release gate 逻辑。

## 证据边界

这些截图证明的是 report-utils 迁移没有改变 data preparation 的外部行为。

它们不是新的数据清洗策略证据，也不是模型训练质量证据。
