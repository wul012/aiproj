# v90 training scale gate report utility migration 运行截图说明

## 本版主题

v90 继续 `report_utils` 收口路线，把 v71 的 `training_scale_gate.py` 迁移到公共报告工具上。迁移后，training scale gate、gated training scale run、training scale run comparison、training scale run decision、promoted baseline decision、promoted seed generation、promoted seed handoff 和 controlled execution handoff 都开始复用同一套 JSON 写出、CSV cell 转换、命令展示、Markdown/HTML 转义和 list/dict 归一化工具。

本版不改变 gate policy 和 check 语义：review/standard/strict profile、pass/warn/fail 汇总、token budget、corpus pass estimate、baseline variant 和 batch handoff 的判断规则保持不变。

## 关键代码

```text
src/minigpt/training_scale_gate.py
src/minigpt/report_utils.py
tests/test_training_scale_gate.py
tests/test_report_utils.py
```

`training_scale_gate.py` 继续负责读取 training scale plan、按 profile policy 生成 checks、汇总 overall status、写出 JSON/CSV/Markdown/HTML。变化在于 JSON 写出、CSV details 转换、命令展示和渲染基础层复用 `report_utils.py`。
