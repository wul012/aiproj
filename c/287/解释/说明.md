# v287 运行截图和解释

## 本版主题

v287 把 promoted seed handoff embedded receipt-check 校验扩展到 sidecar 文件完整性。

新增能力：

- 校验器会打开 `receipt_check.receipt_path` 指向的 automation receipt JSON。
- 校验器会打开 `receipt_check_outputs.json` 指向的 receipt-check JSON。
- 校验器会打开 `receipt_check_outputs.text` 指向的 receipt-check text。
- 缺失、过期或被篡改的 sidecar 会让 embedded check 失败。

本版不改变 receipt schema，不改变 handoff 执行脚本默认行为，也不改变模型训练或模型质量判断。
