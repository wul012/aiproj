# v252 portfolio batch comparison review summary 运行截图说明

## 本版主题

v252 把 training portfolio comparison 的 review action 摘要回传到 training portfolio batch。

v250-v251 让 comparison 能识别 coverage-regressed best-score blocker，并把 review 规则拆成独立模块。但 batch 层只写出 comparison artifact 路径，没有把 review/action/blocker 计数放到 batch 摘要里。v252 解决这个下游可见性问题。

## 证据边界

v252 不改变 portfolio comparison 的判定规则。

v252 不改变 batch variant 执行流程，也不改变是否自动训练。

它只让 batch 报告消费 comparison 已有的 review actions，并把摘要写入：

```text
comparison_review_summary
execution.comparison_review_action_count
execution.comparison_blocker_action_count
Markdown Comparison 表
HTML Comparison Hook 面板
recommendations
```

## 一句话总结

v252 让 training portfolio batch 不再只“生成 comparison”，而是直接暴露 comparison 的 review/blocker 风险，避免批量训练矩阵掩盖后续基线选择问题。
