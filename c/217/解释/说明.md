# v217 promoted seed handoff clean evidence gate diagnostics 运行说明

## 本版目标

v217 给 v216 的 `--require-clean-evidence` 增加结构化诊断输出。

新增 CLI 输出：

```text
clean_evidence_required_status=...
clean_evidence_required_ready=...
clean_evidence_required_detail=...
```

原有 `clean_evidence_required=pass/fail` 和退出码语义保持不变。

## 边界

本版不改变默认 seed handoff 行为，不改变 `--execute`，不改变 `_handoff_allowed()`，也不改变 v216 的 opt-in gate 判定。

它只让 gate pass/fail 的原因更容易在 CI、smoke 和人工日志里直接读取。
