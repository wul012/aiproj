# v187 运行截图和解释

## 本版主题

v187 是 training run evidence eval summary。

本版接着 v186 的真实训练证据推进：`training_run_evidence` 不再只检查 `eval_suite/eval_suite.json` 是否存在，而是读取固定评估集摘要，把 case 数、suite metadata、任务类型覆盖、难度覆盖、平均生成长度和平均唯一字符数写进 JSON/CSV/Markdown/HTML 与 CLI 输出。

## 关键改动

- 更新 `src/minigpt/training_run_evidence.py`，新增 `evaluation` section 和 eval-suite-aware readiness check。
- 更新 `src/minigpt/training_run_evidence_artifacts.py`，在 CSV/Markdown/HTML 中展示 eval summary。
- 更新 `scripts/build_training_run_evidence.py`，终端输出 `eval_suite_cases` 和 `eval_task_types`。
- 更新 `tests/test_training_run_evidence.py`，覆盖完整 eval 摘要和缺 eval 的非阻塞 review 边界。
- 更新 README、`c/README.md` 和项目成熟度阶段代码讲解索引。
