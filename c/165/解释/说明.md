# v165 training scale plan artifact split 运行说明

## 本版主题

v165 把 `src/minigpt/training_scale_plan.py` 的 JSON、variants JSON、CSV、Markdown、HTML 写出和渲染逻辑，拆到 `src/minigpt/training_scale_plan_artifacts.py`。

这一版不改变 scale plan report schema，不改变 `training_scale_variants.json` 给 batch runner 的消费格式，也不改变 `minigpt.training_scale_plan` 的旧 facade 导出。它只让 scale planner 回到“构建训练规模计划”的核心职责。

## 关键改动

- 新增 `src/minigpt/training_scale_plan_artifacts.py`，集中保存 plan artifact 写出和页面渲染。
- `src/minigpt/training_scale_plan.py` 从 artifact 模块重新导出旧函数，保持脚本和下游模块兼容。
- `training_scale_plan.py` 从 541 行降到 313 行。
- `tests/test_training_scale_plan.py` 新增 facade identity 测试，确认旧入口仍指向 artifact 模块实现。
- README、`c/README.md` 和项目成熟度阶段代码讲解索引同步到 v165。
