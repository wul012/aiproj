# v289 运行截图和解释

## 本版主题

v289 把 promoted seed handoff inline embedded receipt-check 的结果写回主报告。

新增能力：

- `execute_promoted_training_scale_seed.py` 在 inline embedded check 写出 JSON/text 后，会把结果保存到主 handoff report。
- 主 JSON 新增 `embedded_receipt_check` 和 `embedded_receipt_check_outputs`。
- 主 CSV、Markdown、HTML 展示 embedded check 状态、decision、sidecar 状态、文件存在性和输出路径。
- stop 决策退出前也会先写出并嵌入 embedded check 结果。

本版不改变 receipt schema，不改变 embedded checker 的判定算法，也不改变默认执行行为。
