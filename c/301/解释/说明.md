# v301 运行截图说明

## 本版主题

v301 把 maturity narrative 里的 CI workflow regression 和 CI order regression 继续接到 training portfolio comparison。目标是让 best-score baseline promotion 在比较层就能被 blocker action 拦下，而不是只在 narrative 里被动显示 review 风险。

本版不改训练、评测或 maturity narrative 的计算逻辑，只补 comparison 层的消费和 artifact 表达。
