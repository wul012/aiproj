# v234 portfolio maturity recommendation precision 运行说明

## 本版目标

v234 精修 training portfolio comparison 的 maturity recommendation。

它解决的问题是：此前只要任一 portfolio 是 review，就会提示检查 best-scoring portfolio，可能误伤最高分但 maturity ready 的组合。

## 边界

本版不改变比较排序和 summary 统计。

本版只调整 recommendations 文案和测试覆盖。

## Tag 含义

`v234.0.0` 表示 training portfolio comparison 的 maturity recommendation 已经区分 best-score review 和 non-leading review。

## 一句话总结

v234 让 maturity review 提示更精准，既保留治理边界，也避免把非领先组合的 review 误投射到最高分组合上。
