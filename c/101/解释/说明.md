# v101 运行截图与解释

## 本版主题

v101 是 experiment card report-utils migration。

本版把 `src/minigpt/experiment_card.py` 里的通用报告 helper 迁移到 `report_utils`，让 experiment card 复用公共 UTC 时间、JSON 写出、HTML 转义和 dict 归一化工具。

本版不改变：

- experiment card 字段结构。
- run summary、data、training、evaluation 和 registry context 的生成策略。
- artifact 收集和输出 artifact 标记逻辑。
- recommendations 和 warnings 规则。
- Markdown/HTML 报告布局。
- model card、project audit、release bundle 或 release gate 逻辑。
