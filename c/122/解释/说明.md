# v122 training portfolio comparison artifact split 运行截图说明

## 本版主题

v122 按 v110 module pressure audit 的维护路线，拆分 `training_portfolio_comparison.py` 的 artifact 输出边界。portfolio 读取、baseline 选择、delta 计算、summary 和 recommendations 仍留在 `training_portfolio_comparison.py`；JSON/CSV/Markdown/HTML 写入和展示 helper 移到 `training_portfolio_comparison_artifacts.py`。

本版不改变 comparison schema、CLI 参数、输出文件名、batch runner 调用方式和 `minigpt.training_portfolio_comparison` 的旧 public artifact 导出。
