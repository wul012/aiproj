# v222 benchmark scorecard numeric helper consolidation 运行说明

## 本版目标

v222 延续 v221 的轻量质量优化，把 benchmark scorecard 主链中重复的 `_number()` / `_int()` 转换 helper 接到 `report_utils`。

本版覆盖：

```text
benchmark_scorecard.py
benchmark_scorecard_scoring.py
benchmark_scorecard_comparison_helpers.py
benchmark_scorecard_decision.py
```

## 边界

本版不改变 scorecard JSON/CSV/Markdown/HTML schema，不改变评分规则，不改变 promotion decision 的 promote/review/block 判定。

它只把局部重复转换逻辑替换为公共 helper，同时保留局部适配函数作为业务语义边界。
