# v286 运行截图和解释

## 本版主题

v286 给 promoted seed handoff 主报告新增 embedded receipt-check 校验。

新增能力：

- `scripts/check_promoted_seed_handoff_embedded_receipt.py` 可读取 handoff 输出目录或主 JSON。
- 校验器会从主报告重新生成 automation receipt，再运行 receipt checker。
- 校验器把重算结果和报告里的 `receipt_check` 做字段级对比。
- 可写出 `promoted_training_scale_seed_handoff_embedded_receipt_check.json` 和 `.txt`。

本版不改变 receipt schema，不改变 handoff 执行脚本默认行为，也不改变模型训练或模型质量判断。
