# v218 promoted seed handoff clean evidence gate artifacts 运行说明

## 本版目标

v218 把 v216/v217 的 clean-evidence CLI gate 结果写入 seed handoff artifacts。

新增报告对象：

```text
clean_evidence_requirement
```

它记录本次 CLI 是否要求 clean evidence、gate 状态是 `not-required` / `pass` / `fail`、readiness status 和 detail。

## 边界

本版不改变默认 seed handoff 行为，不改变 `--execute`，不改变 `--require-clean-evidence` 的退出码策略。

它只把 gate 决策从 stdout 进一步持久化到 JSON/CSV/Markdown/HTML 证据中。
