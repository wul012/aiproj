# v71 运行截图和解释

## 本版主题

v71 新增 training scale gate。它读取 v70 生成的 `training_scale_plan.json`，用 `review`、`standard`、`strict` profile 检查语料规模、质量 warning、variant 数量、baseline、dataset version、token budget 和 estimated corpus passes，输出执行 batch 前的 pass/warn/fail 证据。

本版不执行训练，不重新扫描原始语料，也不替代 release gate。它只回答“这份训练规模计划是否适合交给 batch runner 执行”。

## 关键命令

```powershell
python -m unittest tests.test_training_scale_gate
python -m unittest discover -s tests
python -m compileall src tests scripts
python scripts/plan_training_scale.py data --out-dir tmp\v71-smoke\scale-plan --dataset-name sample-zh --dataset-version-prefix v71
python scripts/check_training_scale_gate.py --plan tmp\v71-smoke\scale-plan\training_scale_plan.json --out-dir tmp\v71-smoke\scale-gate-review --profile review
python scripts/check_training_scale_gate.py --plan tmp\v71-smoke\scale-plan\training_scale_plan.json --out-dir tmp\v71-smoke\scale-gate-standard --profile standard --no-fail
powershell -ExecutionPolicy Bypass -File scripts/playwright_chrome_smoke.ps1 -UrlOrPath tmp\v71-smoke\scale-gate-review\training_scale_gate.html -Out c\71\图片\05-playwright-training-scale-gate-html.png
```

## 证据说明

`training_scale_gate.json` 是执行前 gate 证据，记录 profile、policy、plan summary、checks、overall status、recommendations 和 batch handoff。

`training_scale_gate.csv` 是逐项检查表，适合后续 CI 或人工审阅。

`training_scale_gate.html` 是浏览器审阅入口，不是训练结果；真正训练结果仍由 v69 batch 和 v67 portfolio 产物负责。

## Tag 说明

`v71.0.0` 表示项目新增训练规模准入层：从“能生成 batch-compatible 训练矩阵”推进到“能在执行矩阵前用 profile gate 判断 pass/warn/fail”。
