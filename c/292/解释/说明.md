# v292 运行截图和解释

## 本版主题

v292 把 promoted seed handoff assurance 链路提升为 GitHub Actions smoke。

新增能力：

- 新增 `scripts/check_promoted_seed_handoff_assurance_smoke.py`。
- smoke 脚本会生成最小 promoted seed tree。
- smoke 脚本会执行 handoff、receipt-check、embedded receipt-check 和 assurance。
- smoke 脚本会检查主 handoff report 中的 `handoff_assurance` 与 sidecar 路径。
- GitHub Actions 在 coverage 前运行该 smoke，提前发现核心交接证据链断裂。

本版不改变模型训练逻辑，不改变 handoff schema，不改变 coverage 阈值。

## tag 含义

`v292.0.0` 表示 promoted seed handoff assurance 已经从本地执行入口的一等证据，推进为 GitHub Actions coverage 前的 CI smoke gate。

## 一句话总结

v292 让 promoted seed handoff 的自动化交接证据链不再只靠人工或本地脚本确认，而是在 CI 中先行 smoke 验证。
