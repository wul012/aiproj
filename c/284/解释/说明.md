# v284 运行截图和解释

## 本版主题

v284 把 promoted seed handoff 的 automation receipt checker 接入执行脚本。

新增能力：

- `scripts/execute_promoted_training_scale_seed.py --receipt-check-out-dir <dir>`
- 执行脚本在写出 handoff receipt 后，可以立即校验该 receipt。
- 校验结果写入 `promoted_training_scale_seed_handoff_automation_receipt_check.json` 和 `.txt`。
- 即使 automation summary 最终决定 `stop`，receipt-check 证据也会先写出。

本版不改变默认执行行为；不传 `--receipt-check-out-dir` 时，执行脚本和 v283 保持一致。
