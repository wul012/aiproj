# v88 run comparison report utility migration 运行截图说明

## 本版主题

v88 继续 `report_utils` 收口路线，把 v73 的 `training_scale_run_comparison.py` 迁移到公共报告工具上。迁移后，training scale run comparison、training scale run decision、promoted baseline decision、promoted seed generation、promoted seed handoff 和 controlled execution handoff 都开始复用同一套 JSON 写出、CSV cell 转换、Markdown/HTML 转义和 list/dict 归一化工具。

本版不改变 comparison 的业务语义：readiness score、baseline delta、gate relation、batch relation、best-by-readiness 和 recommendations 的规则保持不变。

## 关键代码

```text
src/minigpt/training_scale_run_comparison.py
src/minigpt/report_utils.py
tests/test_training_scale_run_comparison.py
tests/test_report_utils.py
```

`training_scale_run_comparison.py` 继续负责读取多份 gated training scale run、生成 run summary、选择 baseline、计算 baseline deltas、写出 JSON/CSV/Markdown/HTML。变化在于输出和渲染基础层复用 `report_utils.py`。
