# v224 release gate numeric helper consolidation 运行说明

## 本版目标

v224 延续数值 helper 收敛，把 release-gate policy checks 与 artifact 渲染里的 `_number()` / `_integer()` 接到 `report_utils`。

## 边界

本版不改变 release gate profile，不改变 pass/warn/fail 判定，不改变 JSON/Markdown/HTML schema。

它只收敛重复转换逻辑，并保留局部 helper 作为 policy threshold、score label 和默认归零的业务语义边界。

## Tag 含义

`v224.0.0` 表示 release-gate 链路开始复用 `report_utils` 的公共数值转换 helper，减少局部重复定义。

## 一句话总结

v224 把数值 helper 收敛推进到 release-gate policy 和 artifact 层，让发布闸门证据继续复用统一转换规则。
