# v82 promoted training scale seed handoff 运行说明

## 本版目标

v82 读取 v81 的 promoted training scale seed，验证或执行下一步 `plan_training_scale.py` 命令，把 seed 真正落成 plan 产物，并暴露下一层 batch command。

本版不训练模型，不跑 portfolio batch，只把 seed 命令转成计划证据。

## 运行链路

```text
v80 promoted training scale baseline decision
 -> v81 promoted training scale next-cycle seed
 -> v82 promoted training scale seed handoff
```

v82 的 CLI 是：

```powershell
python scripts/execute_promoted_training_scale_seed.py runs/training-scale-workflow/promoted-seed --execute --out-dir runs/training-scale-workflow/promoted-seed-handoff
```

输出包括：

```text
promoted_training_scale_seed_handoff.json
promoted_training_scale_seed_handoff.csv
promoted_training_scale_seed_handoff.md
promoted_training_scale_seed_handoff.html
```

## 验收标准

- 新单测覆盖 planned、review blocked、执行成功、执行失败、HTML 转义。
- 全量 unittest 回归通过。
- `src`、`tests`、`scripts` 能通过 `compileall`。
- ready smoke 能生成 plan 产物并输出 next batch command。
- blocked smoke 不会把缺失前提伪装成可执行计划。
- Playwright/Chrome 能打开 HTML 报告并生成非空截图。
- 提交前删除临时 `tmp/`、`__pycache__/` 和 `.pytest_cache/`。

## 一句话总结

v82 把 promoted baseline 的下一步从“命令”推进到“计划证据”，让下一轮训练规模链路能继续向 batch 层收口。
