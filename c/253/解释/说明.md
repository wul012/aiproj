# v253 scale run batch review propagation 运行证据

## 本版目标

v253 继续 v252 的 portfolio batch comparison review 链路，把 batch 层已经计算出的 review action、blocker action、coverage regression 和 blocker reason 传到 gated scale run 与 scale-run comparison。

本版不新增训练策略、不改变 gate 是否放行的规则，也不把 review action 升级成默认 blocker。它只让下游 readiness 和 promotion 证据继续看见 batch comparison 的风险摘要。

## 关键改动

- `src/minigpt/training_scale_run.py`：`batch_summary` 新增 comparison review/blocker/coverage 字段，JSON/CSV/Markdown/HTML 和 recommendations 都能消费这些字段。
- `src/minigpt/training_scale_run_comparison.py`：run comparison 汇总每个 scale run 的 batch review/blocker 字段，输出到 CSV、Markdown、HTML，并在 readiness recommendation 中保留风险提示。
- `tests/test_training_scale_run.py`：验证 scale run 从 batch report 读取 review action，并在渲染输出中暴露字段。
- `tests/test_training_scale_run_comparison.py`：验证 scale-run comparison 汇总 batch review 字段，并在渲染输出中保留可见证据。

## 证据边界

这些截图是 v253 的本地运行证据，用来证明字段传播、渲染输出、测试回归和编码卫生。它们不是模型能力提升证明，也不是大规模真实训练结果。

## 一句话总结

v253 把 batch comparison review 从“批量矩阵内部摘要”推进为“scale-run readiness 和 promotion 输入可见的下游风险信号”。
