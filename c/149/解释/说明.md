# v149 运行截图说明

## 本版主题

v149 把 v148 已经进入 `release_readiness` 的 CI workflow hygiene 状态继续带入 `release_readiness_comparison`。

本版新增跨版本 CI workflow 状态比较、failed check delta、CI workflow regression summary 和推荐语。它仍然是比较证据，不改变 release gate 的硬阻断规则。

  - delta 新增 `baseline_ci_workflow_status`、`compared_ci_workflow_status`、`ci_workflow_failed_check_delta`、`ci_workflow_status_changed`。

  - 新增 CI workflow regression 测试。
  - 输出测试检查新增 CSV 字段。
- `README.md`
  - Current version、tag、使用命令、截图索引和能力说明更新到 v149。

## 证据边界

release readiness comparison 的 CI workflow delta 证明的是跨版本 CI 策略健康度变化，例如 failed check 是否增加、panel 是否从 pass 变成 warn。

它不证明模型能力、训练效果、数据质量或 checkpoint 成熟度。它也不改变 release gate 的 hard block 策略。

## tag 含义

`v149.0.0` 标记的是 “release readiness comparison CI workflow deltas”：跨版本 readiness comparison 可以识别 CI workflow hygiene 状态变化和退化。

一句话总结：v149 让 CI workflow hygiene 从当前发布总览进入跨版本比较，方便发现 CI 策略漂移。
