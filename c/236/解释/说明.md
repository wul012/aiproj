# v236 portfolio comparison review actions 运行说明

## 本版主题

v236 把 training portfolio comparison 里的风险提示升级为结构化 `review_actions`。

它承接 v232-v235 的 maturity review 链路：前面已经知道 best-score maturity status、推荐语和 review portfolio 名称；本版进一步生成 action row，让后续自动化或人工复核可以直接消费。

## 本版边界

本版不改变训练、评分、排序、baseline 选择和 recommendations 的基本语义。

本版新增的是证据表达：

- `report.review_actions`
- `summary.review_action_count`
- `summary.blocker_action_count`
- Markdown/HTML 的 `Review Actions` 表
