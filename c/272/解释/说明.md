# v272 运行说明

## 本版主题

v272 把 promotion clean batch-review evidence 带到 training-scale promotion index。此前 v271 已经能在 promotion acceptance 阶段阻断不满足 clean requirement 的 handoff；本版让 index 也保存该 guard，并且把未满足 clean requirement 的 promoted report 排除出 compare inputs。

## 关键改动

- promotion index row 新增 `handoff_require_clean_batch_review` 和 `handoff_clean_batch_review_status`。
- index summary 新增 clean-required、clean、unclean 三类 handoff 计数。
- compare input selection 现在要求 promoted report 同时满足 clean batch-review requirement。
- `scripts/index_training_scale_promotions.py` 打印新增 clean batch-review counts。

## Tag 含义

`v272.0.0` 标记 training scale promotion index clean batch gate。它表示 clean batch-review requirement 已经从 decision、workflow、handoff、promotion 继续推进到 promotion index 和 compare input selection。

## 一句话总结

v272 让 promotion index 只把满足 clean batch-review requirement 的 promoted run 放入 compare inputs，避免不干净证据进入模型能力对比。
