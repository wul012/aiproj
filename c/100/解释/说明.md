# v100 运行截图与解释

## 本版主题

v100 是 model card report-utils migration。

本版把 `src/minigpt/model_card.py` 里的通用报告 helper 迁移到 `report_utils`，让 model card 复用公共 UTC 时间、JSON 写出、HTML 转义和 list/dict 归一化工具。

本版不改变：

- model card 字段结构。
- ready/review run 统计。
- intended use 和 limitations 文案生成。
- coverage 和 recommendations 规则。
- Markdown/HTML 报告布局。
- project audit、release bundle 或 release gate 逻辑。

## 证据边界

这些截图证明的是 report-utils 迁移没有改变 model card 的外部行为。

它们不是模型训练质量证据，也不是新 release policy 证据。
