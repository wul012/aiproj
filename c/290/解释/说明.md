# v290 运行截图和解释

## 本版主题

v290 新增 promoted seed handoff assurance checker。

新增能力：

- `scripts/check_promoted_seed_handoff_assurance.py` 可以检查 handoff 输出目录或主报告 JSON。
- assurance checker 会重新计算 embedded receipt-check。
- 它会比较主报告里的 `embedded_receipt_check`，并打开 `embedded_receipt_check_outputs` 指向的 JSON/text sidecar。
- 它可以写出 `promoted_training_scale_seed_handoff_assurance.json` 和 `.txt`，作为 CI/交接用的总体验证证据。

本版不改变 handoff 决策，不改变 receipt 或 embedded checker 的基础 schema，也不新增模型训练能力。
