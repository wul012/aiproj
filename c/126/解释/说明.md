# v126 comparison artifact split 运行截图说明

## 本版主题

v126 按 v110 module pressure audit 的维护路线，拆分 `comparison.py` 的 artifact 输出边界。run 摘要、baseline 选择、delta 计算、summary 和 recommendations 仍留在 `comparison.py`；JSON/CSV/SVG/Markdown/HTML 写入和展示 helper 移到 `comparison_artifacts.py`。

本版不改变 comparison schema、CLI 参数、输出文件名、README 示例和 `minigpt.comparison` 的旧 public artifact 导出。

## tag 含义

`v126.0.0` 表示 MiniGPT 的 baseline run comparison 完成一次低风险、定向收口：不是新增比较类型，也不是改变训练能力，而是把 comparison 的证据发布层从比较计算层拆开。

## 一句话总结

v126 的价值是让 baseline run comparison 也具备清晰的“比较计算层 / artifact 发布层”边界，减少单文件膨胀，同时保护现有 CLI、schema、输出文件名和旧导出契约。
