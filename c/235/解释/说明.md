# v235 portfolio maturity review names 运行说明

## 本版主题

v235 让 training portfolio comparison 的 maturity review 证据从“只有数量”升级为“数量 + 具体 portfolio 名称”。

这版承接 v232-v234 的 maturity review 链路：前几版已经让 best-score maturity status 进入 summary、Markdown/HTML 和 recommendations；本版补上可定位性，方便 reviewer 直接知道是哪一个 portfolio 触发 review。

## 本版边界

本版不改变 portfolio 排序、best-score 选择、recommendation 分支和比较评分。

本版只增加：

- `summary.maturity_review_names`
- Markdown 的 `Maturity review portfolios`
- HTML stats 的 `Maturity reviews`
- 对应测试与运行证据

## Tag 含义

`v235.0.0` 表示 portfolio maturity review names 版本。

它不是新的模型能力版本，而是训练组合比较证据链的可追踪性增强版本。
