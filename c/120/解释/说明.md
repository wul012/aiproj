# v120 运行截图说明：Benchmark Scorecard Scoring Split

## 本版主题

v120 把 `benchmark_scorecard.py` 中的 case 合并、rubric 打分和 task/difficulty drilldown 逻辑拆成 `benchmark_scorecard_scoring.py`。

这版不改变 scorecard schema v3、rubric 权重、drilldown 公式、CLI 参数、artifact 输出文件名或旧导入路径，只把评分计算从 scorecard 主编排模块中抽离出来。

## 标签含义

`v120.0.0` 表示 MiniGPT 完成 benchmark scorecard scoring split。它延续 v110-v119 的维护收口路线，把评估 scorecard 从“大文件同时负责证据编排和评分计算”推进到 scorecard orchestration 与 scoring logic 分离的结构。
