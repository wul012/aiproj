# v273 运行说明

## 本版主题

v273 把 clean batch-review guard 从 promotion index 继续带到 promoted training-scale comparison。此前 v272 已经让 index 只把干净的 promoted report 放入 compare inputs；本版让 comparison 层自己也会重检 clean requirement，防止旧 index、手工 index 或兼容输入把 dirty clean-required promoted row 混进模型能力对比。

## 关键改动

- promoted comparison row 新增 `handoff_require_clean_batch_review` 和 `handoff_clean_batch_review_status`。
- comparison input selection 会排除 clean requirement 未满足的 promoted row。
- summary 新增 handoff clean-required、clean、unclean 以及 comparison-ready 对应计数。
- CSV、Markdown、HTML 和 CLI 输出新增 clean batch-review 字段。
- 测试覆盖 clean-required clean、clean-required review、不要求 clean 三类输入。

## Tag 含义

`v273.0.0` 标记 promoted training scale comparison clean batch gate。它表示 clean batch-review requirement 已经从 decision、workflow、handoff、promotion、promotion index 继续推进到 promoted comparison。

## 一句话总结

v273 让 promoted comparison 自己拒绝不满足 clean batch-review requirement 的 promoted input，避免不干净证据被误用为模型能力对比。
