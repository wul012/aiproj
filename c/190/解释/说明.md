# v190 运行截图和解释

## 本版主题

v190 是 benchmark scorecard generation quality path compatibility。

本版修掉 v189 真实链路中的目录摩擦：`build_benchmark_scorecard.py` 现在能识别 `generation_quality/` 和 `generation-quality/` 两种 generation quality 目录，也能识别它们位于 `eval_suite/` 下的变体。真实链路不再需要把 `generation_quality` 复制成 `generation-quality`。

## 关键改动

- 更新 `src/minigpt/benchmark_scorecard.py`，让 `_read_generation_quality()` 识别四种 quality report 路径。
- 更新 `_evidence_completeness_component()`，让 evidence completeness 与读取逻辑保持一致。
- 更新 `tests/test_benchmark_scorecard.py`，新增 underscore-style generation quality 路径兼容测试。
- 更新 README、`c/README.md` 和项目成熟度阶段代码讲解索引。
