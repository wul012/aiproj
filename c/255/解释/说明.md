# v255 handoff batch review propagation 运行证据

## 本版目标

v255 继续 v254 的 selected batch review status 链路，把 run decision 中的 batch review/blocker/coverage 摘要传到 controlled training scale handoff。

本版不改变 handoff 是否允许执行的规则，不新增训练动作，也不自动执行命令。它只让 handoff 层在验证或执行 reviewed command 时，继续保留 selected run 的 batch comparison 风险。

## 关键改动

- `src/minigpt/training_scale_handoff.py`：summary 新增 selected batch review status、selected review/blocker/coverage 计数、comparison 级别 batch review/blocker/coverage 总数和 blocker reasons。
- `training_scale_handoff` CSV、Markdown、HTML 输出同步显示这些字段。
- handoff recommendations 区分 selected batch review 和 selected batch blocker：review 提醒人工复核，blocker 提醒先解决 blocker action。
- `tests/test_training_scale_handoff.py`：验证 review/blocker 两种状态都能进入 handoff summary、renderers 和 recommendation。

## 证据边界

这些截图证明 handoff 证据能保留上游 batch comparison 风险。它们不是模型能力提升证明，也不是实际大规模训练结果。

## 一句话总结

v255 让 controlled handoff 在交接 execute command 时继续保留 selected run 的 batch review/blocker 风险。
