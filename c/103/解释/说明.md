# v103 运行截图与解释

## 本版主题

v103 是 run manifest report-utils migration。

本版把 `src/minigpt/manifest.py` 里的通用报告 helper 迁移到 `report_utils`，让 run manifest 复用公共 UTC 时间、JSON 写出和 HTML 转义工具。

本版不改变：

- run manifest 字段结构。
- git、environment、data、model、training、results 和 artifacts 的生成策略。
- artifact 规格列表。
- SHA-256 计算规则和超大文件跳过 digest 的边界。
- run manifest SVG 布局。
- dataset card、experiment card、model card、project audit、release bundle 或 release gate 逻辑。
