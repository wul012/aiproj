# v295 运行截图和解释

## 本版主题

v295 把 promoted seed handoff assurance smoke 的 workflow 顺序检查升级为行号证据。

新增能力：

- `ci_workflow_hygiene` 会记录 smoke 行号和 coverage 行号。
- `required_order` check 的 `actual` 变成 `before_line=<n>;after_line=<n>`。
- `required_order` check 的 detail 会写出 smoke 与 coverage 的具体行号。
- tests 会验证正常 workflow 的行号证据和错误顺序 workflow 的诊断文案。

本版不改变 GitHub Actions 文件本身，不改变 smoke 脚本，不改变 coverage 阈值。
