# v105 运行截图与解释

## 本版主题

v105 是 data quality report-utils migration。

本版把 `src/minigpt/data_quality.py` 里的通用报告 helper 迁移到 `report_utils`，让 data quality 复用公共 JSON 写出和 HTML 转义工具。

本版不改变：

- 数据质量判断规则。
- `DatasetQualityIssue` 字段结构。
- `status`、`warning_count`、`issue_count`、`duplicate_line_count` 的计算方式。
- duplicate source 检测。
- repeated line 检测。
- fingerprint 计算。
- dataset quality SVG 布局。
- data preparation、run manifest、dataset card、experiment card、model card、project audit、release bundle 或 release gate 逻辑。

## 证据边界

这些截图证明的是 report-utils 迁移没有改变 data quality 的外部行为。

它们不是新的质量算法证据，也不是模型训练质量证据。

## tag 含义

- data quality 已接入 `report_utils`。
