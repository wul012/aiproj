# v293 运行截图和解释

## 本版主题

v293 让 promoted seed handoff assurance smoke 写出顶层 summary artifacts。

新增能力：

- `scripts/check_promoted_seed_handoff_assurance_smoke.py` 写出 `promoted_seed_handoff_assurance_smoke_summary.json`。
- 同时写出 `promoted_seed_handoff_assurance_smoke_summary.txt`。
- stdout 保留原有 `status`、`decision`、`handoff_assurance_*` 字段。
- stdout 新增 `summary_json` 和 `summary_text`，方便 CI 定位产物。
- summary 记录 smoke 状态、命令、日志路径、目录路径、handoff report 路径、assurance output 路径和 checks。

本版不改变 GitHub Actions 步骤，不改变 handoff schema，不改变模型训练逻辑。
