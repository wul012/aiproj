# v189 运行截图和解释

## 本版主题

v189 是 training run evidence benchmark scorecard summary。

本版接着 v188 的真实训练证据推进：`training_run_evidence` 不只读取 fixed eval suite 和 generation quality 摘要，还会读取 `benchmark-scorecard/benchmark_scorecard.json`，把综合 benchmark 状态、总分、rubric 状态和平均分、最弱 rubric case、最弱任务类型、最弱难度和生成质量 flag 上下文写入训练证据。

## 关键改动

- 更新 `src/minigpt/training_run_evidence.py`，新增 `scorecard` section 和 benchmark-scorecard-aware check。
- 更新 `src/minigpt/training_run_evidence_artifacts.py`，在 CSV/Markdown/HTML 中展示 benchmark scorecard summary。
- 更新 `scripts/build_training_run_evidence.py`，终端输出 `benchmark_scorecard_status` 和 `benchmark_scorecard_score`。
- 更新 `tests/test_training_run_evidence.py`，覆盖完整 scorecard 摘要和缺 scorecard 的非阻塞 review 边界。
- 更新 README、`c/README.md` 和项目成熟度阶段代码讲解索引。
