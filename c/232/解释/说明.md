# v232 portfolio comparison maturity review guard 运行说明

## 本版目标

v232 让 training portfolio comparison 在 best-score 选择旁边保留 maturity narrative 的 review 状态。

它解决的问题是：一个组合可以分数最高，但如果 maturity narrative 是 review，就不应该被直接当成 clean baseline。

## 边界

本版不改变训练、评估、scorecard 或 portfolio 排序逻辑。

本版只增加 summary 字段和 recommendations 提醒。

## Tag 含义

`v232.0.0` 表示 training portfolio comparison 开始显式携带 best-score portfolio 的 maturity status，防止高分但 review-level 的组合被误当成 clean baseline。

## 一句话总结

v232 把 maturity narrative 的 review 边界继续推进到组合比较层，让“分数最高”和“证据干净”不被混为一谈。
