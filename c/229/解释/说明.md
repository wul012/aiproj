# v229 scorecard decision eval readiness 运行说明

## 本版目标

v229 把 scorecard comparison 中的 eval-suite readiness 带入 benchmark scorecard promotion decision。

它解决的问题是：候选 scorecard 即使分数提升，如果 eval suite 不是 comparison-ready，也不应该被当作 clean promotion 证据。

## 边界

本版不改变 regression blocker 规则。

本版把 non-comparison-ready 候选变成 review-level evidence，不直接 block。
