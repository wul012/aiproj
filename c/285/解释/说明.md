# v285 运行截图和解释

## 本版主题

v285 把 inline receipt-check 结果写回 promoted seed handoff 主报告。

新增能力：

- 主 handoff JSON 包含 `receipt_check` 和 `receipt_check_outputs`。
- CSV/Markdown/HTML 同步展示 receipt-check 状态、decision、exit code、blocking source 和 sidecar 路径。
- stop 场景仍然先写 receipt-check 证据，再按 automation summary 退出。

本版不改变默认执行行为，不改变 receipt schema，也不改变 automation decision。
