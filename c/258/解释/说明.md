# v258 promoted comparison batch review propagation 运行说明

## 本版主题

v258 把 v257 promotion index 中的 handoff selected batch review/blocker context 继续传到 promoted training scale comparison。这样 promoted-only comparison 不只显示哪些 promoted runs 被比较，也会保留这些 comparison-ready inputs 是否仍带 selected batch review/blocker 风险。

本版不改变 promoted-only compare 规则：只有 `promoted_for_comparison=True` 的输入进入比较。selected batch review/blocker 只是风险上下文和 recommendation，不会自动把 comparison 变成 blocked。

## 关键改动

- `src/minigpt/promoted_training_scale_comparison.py`：promoted input row 接住 handoff batch review 字段，并在 summary 中聚合 comparison-ready review/blocker counts、action totals 和 blocker reasons。
- `src/minigpt/promoted_training_scale_comparison_artifacts.py`：CSV/Markdown/HTML 输出新增 batch review/blocker 字段和统计卡片。
- `scripts/compare_promoted_training_scale_runs.py`：CLI 输出新增 comparison-ready batch review/blocker totals，方便 CI 日志直接检查。
- `tests/test_promoted_training_scale_comparison.py`：新增 compare-ready batch review/blocker propagation 测试，确认 promoted-only comparison 语义不变但风险字段可见。
- `README.md` 和 `代码讲解记录_项目成熟度阶段/272-v258-promoted-comparison-batch-review-propagation.md`：同步说明本版边界、输入输出和证据链角色。

## tag 含义

`v258.0.0` 表示 promoted comparison 已经接住 promotion-index handoff selected batch review/blocker context，并在 promoted-only comparison 阶段保持风险可见。

## 一句话总结

v258 让 promoted comparison 从“只比较 promoted inputs”推进到“比较 promoted inputs 时保留 comparison-ready selected batch review/blocker 风险”。
