# v84 controlled handoff report utility migration 运行截图说明

## 本版主题

v84 继续 v83 的代码质量收口，把早期的 `training_scale_handoff.py` 迁移到 `report_utils.py`。这让公共报告工具不只服务最新的 promoted seed handoff，也开始服务 v76 的 controlled execution handoff。

本版不改变执行策略：默认仍然只做 planned validation，只有显式 `--execute` 才运行命令。

## 关键代码

```text
src/minigpt/training_scale_handoff.py
src/minigpt/report_utils.py
tests/test_training_scale_handoff.py
tests/test_report_utils.py
```

`training_scale_handoff.py` 继续负责读取 workflow decision、判断 ready/review/blocked、执行或验证命令、检查 training/batch/variant/checkpoint 产物。变化在于 JSON/CSV 写出、artifact row、command display、Markdown/HTML 转义和 list/dict 归一化都复用 `report_utils.py`。
