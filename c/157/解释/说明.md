# v157 registry leaderboard split 运行说明

## 本版主题

v157 把 registry HTML 里的 loss、rubric、pair delta 和 release readiness delta leaderboard 渲染，从 `src/minigpt/registry_render.py` 拆到 `src/minigpt/registry_leaderboards.py`。

这是一版轻量、定向的质量优化。它不改变 registry payload，不改变 `render_registry_html()` 入口，不改变 `write_registry_outputs()` 产物路径，只把独立的 leaderboard HTML 片段拿出来。

## 关键改动

- 新增 `src/minigpt/registry_leaderboards.py`，集中保存四个 leaderboard HTML helper。
- 更新 `src/minigpt/registry_render.py`，继续负责整页 HTML 编排，但改为调用新模块渲染 leaderboard 区块。
- 新增 `tests/test_registry_leaderboards.py`，覆盖 HTML 转义、空状态文案和相对 report link。
- README、`c/README.md` 和项目成熟度阶段代码讲解索引同步到 v157。
