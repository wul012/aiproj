# v214 promoted seed handoff clean evidence readiness 运行说明

## 本版目标

v214 把 v212/v213 的 promoted seed handoff suite alignment verdict，继续沉淀成可机器消费的 clean-evidence readiness 字段。

本版新增的核心输出是：

```text
seed_handoff_clean_evidence_ready
seed_handoff_clean_evidence_status
seed_handoff_clean_evidence_detail
```

这些字段说明当前 seed handoff 是否可以作为 clean comparison evidence 使用。

## 边界

本版不改变 seed handoff 的执行策略，不改变 planned mode，不改变 `--execute`，也不把 `mismatch` 或 `missing` 变成 blocker。

clean-evidence readiness 是证据元数据，不是执行门禁。
