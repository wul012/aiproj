# v117 运行截图说明：Server Contract Split

## 本版主题

v117 把 `server.py` 中的纯契约和 payload 逻辑拆成 `server_contracts.py`。

这版不改变本地 HTTP API、playground 请求、流式生成、pair generation 或 request log，只把安全配置、请求/响应 dataclass、请求解析、checkpoint discovery、health/model-info/checkpoint-compare payload、SSE/timeout payload 和 pair payload 从 server runtime 中抽离出来。

## 关键代码

- `src/minigpt/server_contracts.py`
  - 新增 server 契约层，负责 safety profile、request/response dataclass、request parser、checkpoint metadata、API payload 和 SSE/pair payload。
- `src/minigpt/server.py`
  - 保留旧 public API 的兼容入口，继续负责 HTTP handler、真实生成、SSE 写入、请求日志和本地文件服务。

## 标签含义

`v117.0.0` 表示 MiniGPT 完成 server contract split。它延续 v109-v116 的维护收口路线，把本地推理服务从“大文件混合运行时和契约逻辑”推进到 server runtime 与 server contracts 分离的结构。
