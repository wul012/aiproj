# v99 运行截图与解释

## 本版主题

v99 是 project audit report-utils migration。

本版把 `src/minigpt/project_audit.py` 里的通用报告 helper 迁移到 `report_utils`，让 project audit 复用公共 UTC 时间、JSON 写出、HTML 转义和 list/dict 归一化工具。

本版不改变：

- 审计评分权重。
- pass/warn/fail 判定。
- request history summary 审计要求。
- Markdown/HTML 报告布局。
- release bundle 或 release gate 逻辑。
