# v158 release gate comparison artifact split 运行说明

## 本版主题

v158 把 release gate profile comparison 的 JSON、CSV、delta CSV、Markdown 和 HTML 写出/渲染逻辑，从 `src/minigpt/release_gate_comparison.py` 拆到 `src/minigpt/release_gate_comparison_artifacts.py`。

这是一版 contract-preserving refactor。`build_release_gate_profile_comparison()` 的 report schema 不变，`scripts/compare_release_gate_profiles.py` 的调用方式不变，旧模块里的 writer/renderer 导出也保持兼容。

## 关键改动

- 新增 `src/minigpt/release_gate_comparison_artifacts.py`，集中保存 comparison report 的 artifact 写出和 HTML/Markdown 渲染。
- 更新 `src/minigpt/release_gate_comparison.py`，保留 profile comparison 计算、delta explanation 和 recommendation，导入并继续导出 artifact 函数。
- 更新 `tests/test_release_gate_comparison.py`，增加 facade identity 测试，确认旧导出指向新 artifact 实现。
- README、`c/README.md` 和项目成熟度阶段代码讲解索引同步到 v158。
