# v185 运行截图和解释

## 本版主题

v185 是 training scale promotion index helper split。

本版把 promotion index 里的名称解析、promotion row 构造、primary variant / artifact 提取、baseline 选择、compare command inputs、summary 和 recommendations 辅助逻辑，从 `training_scale_promotion_index.py` 拆到 `training_scale_promotion_index_helpers.py`。`training_scale_promotion_index.py` 继续保留加载 promotion、构建 index、渲染 Markdown/HTML 和写出 JSON/CSV/Markdown/HTML 的公共入口。

## 关键改动

- 新增 `src/minigpt/training_scale_promotion_index_helpers.py`，集中训练规模 promotion index 的内部归一化和比较输入计算。
- `src/minigpt/training_scale_promotion_index.py` 回到 index facade 和 artifact presentation 角色，外部函数名、输出文件名和 report schema 不变。
- `tests/test_training_scale_promotion_index.py` 增加 helper smoke，确认 helper 计算出的 compare inputs 与 public builder 的 promotions/summary 保持一致。
- README、`c/README.md` 和项目成熟度阶段代码讲解索引追加 v185，同时补齐 v184 的代码讲解文件。

## tag 含义

`v185.0.0` 表示 MiniGPT 第 185 个可回看版本：training scale promotion index 的内部 helper plumbing 已经从主 index 模块中拆出，公共 API 和证据归档保持兼容，运行截图与解释归档在 `c/185`。

## 一句话总结

v185 用一次定向 helper 拆分把 promotion index 的数据归一化和比较输入计算从 artifact/report facade 中分离出来，让训练规模 promotion 索引链路更清楚。
