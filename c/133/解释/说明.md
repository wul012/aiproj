# v133 registry ranking split 运行说明

## 本版主题

v133 把 `registry_data.py` 中的 loss leaderboard、benchmark rubric leaderboard、pair delta 聚合、release readiness delta 聚合拆到 `registry_rankings.py`，让原模块继续负责 run 发现、单 run 证据读取、`RegisteredRun` 构造和 registry dict 组装。

这是一版轻量收口，不改变 registry schema，不改变 `build_run_registry()` 的输出字段，不改变 `scripts/register_runs.py`，也不改变 `minigpt.registry` facade。

## 证据边界

`c/133` 证明的是 registry 数据层拆分后仍能跑通：旧 public API 仍可用，registry JSON schema 没变，HTML 仍能展示 leaderboards 和 delta panels。它不证明模型质量提升，也不代表 registry 本身等同生产实验平台；它证明的是本地 AI 工程治理证据链的聚合逻辑更容易维护和回归。
