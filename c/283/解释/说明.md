# v283 promoted seed handoff receipt checker artifacts

## 本版主题

v283 增强 v282 的 receipt checker，使它更适合直接接在 handoff 输出目录后面。

v282 checker 只能显式传入 receipt JSON。v283 允许传入 handoff 输出目录，自动定位：

```text
promoted_training_scale_seed_handoff_automation_receipt.json
```

同时新增 `--out-dir`，把 checker 自己的结果写成 JSON/text artifact。

## 关键改动

- `src/minigpt/promoted_training_scale_seed_handoff_receipt.py`
  - 新增 receipt 文件名常量。
  - 新增目录到 receipt JSON 的 resolver。
  - 新增 check JSON/text 输出 writer。

- `scripts/check_promoted_seed_handoff_receipt.py`
  - 位置参数现在可传 receipt JSON 或 handoff 输出目录。
  - 新增 `--out-dir`。
  - CLI 输出 resolved receipt path 和 check artifact path。

- `tests/test_promoted_training_scale_seed_handoff_receipt.py`
  - 覆盖目录输入、`--out-dir` 输出、library writer 输出。

## Tag 含义

`v283.0.0` 表示：promoted seed handoff receipt checker artifacts。它说明 receipt checker 已经能自动发现 handoff 目录中的 receipt，并把检查结果作为独立 artifact 留存。

## 一句话总结

v283 让 receipt checker 从“检查一个 JSON 文件”升级为“检查 handoff 输出目录并产出可归档检查结果”的 CI 小工具。
