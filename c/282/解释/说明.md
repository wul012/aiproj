# v282 promoted seed handoff receipt checker

## 本版主题

v282 在 v281 的 automation receipt 之上新增 checker CLI。

v281 已经写出 compact receipt，但 CI 仍然需要自己解析 JSON、判断结构是否有效、再按 decision 退出。v282 新增：

```text
scripts/check_promoted_seed_handoff_receipt.py
```

它只读 receipt JSON，不重新计算 handoff 决策。

## 关键改动

- `src/minigpt/promoted_training_scale_seed_handoff_receipt.py`
  - 加载 receipt JSON。
  - 校验 receipt type、schema version、decision、exit code、blocking source、failed requirements。
  - 渲染稳定 key/value 检查结果。

- `scripts/check_promoted_seed_handoff_receipt.py`
  - CLI 入口。
  - 默认：receipt 无效或 decision 是 `stop` 时非零退出。
  - `--allow-stop`：允许结构有效的 stop receipt 以 0 退出。

- `tests/test_promoted_training_scale_seed_handoff_receipt.py`
  - 覆盖 continue receipt、stop receipt、不一致 receipt、生成 receipt 后调用 checker、`--allow-stop`。
