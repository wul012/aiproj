# v228 scorecard comparison eval readiness 运行说明

## 本版目标

v228 把 v227 的 scorecard eval-suite readiness 继续带入 benchmark scorecard comparison。

它解决的问题是：跨 run 比较报告以前只看 overall/rubric/generation flag/case delta；如果某个 scorecard 的 eval suite 本身不是 comparison-ready，比较结果仍可能看起来像干净的模型质量变化。

## 边界

本版不改变 scorecard decision 的 promote/block 规则，不改变单个 benchmark scorecard 的评分策略。

本版只在 comparison 的 runs、summary、CSV、Markdown、HTML 和 recommendations 中暴露 readiness 证据。

## Tag 含义

`v228.0.0` 表示 benchmark scorecard comparison 开始显示和消费 eval-suite comparison readiness，让跨 run 分数变化带上评估可比性边界。

## 一句话总结

v228 让 scorecard comparison 不再只比较分数，也能提示参与对比的 scorecard 是否具备干净的 eval-suite comparison-ready 证据。
