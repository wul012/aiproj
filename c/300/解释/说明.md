# v300 运行截图说明

## 本版主题

v300 把 CI workflow order regression 从 maturity summary 继续传入 maturity narrative。目标是让最后面向人阅读的 portfolio review 也能直接看到 smoke-before-coverage 顺序漂移，而不是只在 registry 或 maturity 数值层里发现。

本版不新增新的报告家族，不改变训练、模型推理或 release readiness comparison 的计算方式。它只补齐 maturity narrative 对现有 CI order regression 字段的读取、展示和 review 判定。

## tag 说明

`v300.0.0` 的含义是：MiniGPT v300 maturity narrative CI order regression review。

这一版让 `release_readiness_ci_workflow_order_regression_count` 和 `release_readiness_max_ci_workflow_order_violation_delta` 从 maturity summary 继续进入 maturity narrative。

一句话总结：v300 让 CI workflow order regression 成为 portfolio narrative 级别的 review 信号，最终成熟度叙事也能感知 smoke-before-coverage 漂移。
