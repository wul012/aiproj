# v192 运行截图和解释

## 本版主题

v192 是 external pair baseline portfolio。

v191 让 training portfolio 自动生成 pair batch，但默认只能做同 checkpoint 可复现自检。v192 在这个基础上增加外部 baseline checkpoint 参数，让一次 candidate 训练可以直接和已有 baseline checkpoint 做固定 prompt pair batch，再由 scorecard 消费这组跨 checkpoint 证据。

## 关键改动

- 更新 `scripts/run_training_portfolio.py`，新增 `--pair-baseline-checkpoint`、`--pair-baseline-tokenizer`、`--pair-baseline-id`、`--pair-candidate-id`。
- 更新 `src/minigpt/training_portfolio.py`，新增 `pair_config`，默认是 `same_checkpoint_baseline`，传入 baseline checkpoint 时切换为 `external_baseline`。
- 更新 `src/minigpt/training_portfolio_artifacts.py`，让 Markdown 和 HTML 都显示 Pair mode。
- 更新 `src/minigpt/pair_batch.py`，让 `comparison.same_checkpoint` 优先比较真实 checkpoint 路径，而不是只看展示 id。
- 更新 portfolio 和 pair batch tests，覆盖外部 baseline 命令、pair_config 输出和 same-checkpoint 路径判断。
- 更新 README、`c/README.md` 和项目成熟度阶段代码讲解索引。

## tag 含义

`v192.0.0` 表示 MiniGPT 第 192 个可回看版本：training portfolio 可以在默认 same-checkpoint 自检之外，接收外部 baseline checkpoint 并生成真实 baseline-vs-candidate pair evidence。运行截图与解释归档在 `c/192`。

## 一句话总结

v192 把 v191 的 pair 自检升级为可选的跨 checkpoint 真实比较入口，让 training portfolio 从“证明可复现”继续推进到“能比较 candidate 与 baseline 的生成差异”。
