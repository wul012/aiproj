# v93 training scale promotion report utility migration 运行截图说明

## 本版主题

v93 继续 `report_utils` 收口路线，把 v77 的 `training_scale_promotion.py` 迁移到公共报告工具上。

这次仍然是轻量、定向质量优化：只回收 promotion 里的重复 JSON 写出、Markdown/HTML 转义和 list/dict 归一化 helper，不改变 handoff、scale run、batch、variant portfolio 和 required artifacts 的 promoted/review/blocked 判定。

迁移后，训练规模执行后的 promotion acceptance 也开始和前置执行链路共享同一套报告基础设施。

## 关键代码

```text
src/minigpt/training_scale_promotion.py
src/minigpt/report_utils.py
tests/test_training_scale_promotion.py
tests/test_report_utils.py
scripts/build_training_scale_promotion.py
```

`training_scale_promotion.py` 仍然负责读取 handoff、scale run、batch、variant portfolio，检查 required artifacts，并输出 promoted/review/blocked 验收证据。变化只在报告基础 helper 的来源。

- promotion 私有 JSON/Markdown/HTML/list/dict helper 被公共 `report_utils` 替代。
