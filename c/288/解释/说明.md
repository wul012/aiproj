# v288 运行截图和解释

## 本版主题

v288 把 promoted seed handoff embedded receipt-check 接回执行脚本。

新增能力：

- `execute_promoted_training_scale_seed.py` 新增 `--embedded-receipt-check-out-dir`。
- 执行脚本在写出主 handoff report、receipt JSON/text 和 receipt-check JSON/text 后，立即运行 embedded receipt-check。
- embedded check 会校验主报告里的 `receipt_check` 以及引用的 sidecar JSON/text。
- stop 决策退出前也会先写出 embedded check JSON/text，方便 CI 留存诊断。

本版不改变 receipt schema，不改变默认执行行为，也不改变模型训练或模型质量判断。
