# v275 运行说明

## 本版主题

v275 把 v274 promoted baseline decision 中的 clean batch-review evidence 继续带到 promoted next-cycle seed。此前 seed 已经能继承 selected suite guard 和 selected batch review；本版让 seed 也能保留 selected clean requirement、selected clean status、clean/unclean handoff counts 和 comparison-ready clean counts。

## 关键改动

- promoted seed 新增 `handoff_clean_batch_review` 区块。
- seed summary 新增 selected clean status、clean-required/clean/unclean counts、comparison-ready clean counts。
- seed recommendation 会提醒 rejected dirty clean-required decision inputs 不能进入 next-cycle baseline。
- CSV、Markdown、HTML 和 CLI 输出新增 clean batch-review 字段。
- 测试覆盖 decision -> seed 的 clean batch-review 字段传递、artifact 输出和 CLI stdout。
