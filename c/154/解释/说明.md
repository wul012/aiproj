# v154 release readiness artifact split 运行说明

## 本版主题

v154 把 release readiness dashboard 的 JSON/Markdown/HTML 渲染写出层从 `release_readiness.py` 拆到 `release_readiness_artifacts.py`。

这是一版维护收口，不改变 readiness 的核心判定逻辑。

## 关键改动

- `src/minigpt/release_readiness_artifacts.py`
  - 新增 artifact 输出模块，负责 release readiness 的 JSON、Markdown、HTML 写出和展示 helper。
- `src/minigpt/release_readiness.py`
  - 保留 dashboard 构建、panel 判定、路径解析、JSON 输入读取。
  - 从 artifact 模块 re-export 原来的 render/write 函数，保持旧 import 可用。
- `tests/test_release_readiness.py`
  - 增加 facade identity 测试，证明旧模块导出和新 artifact 模块函数是同一对象。
