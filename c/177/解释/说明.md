# v177 dashboard section helper split 运行说明

## 本版目标

v177 把 `dashboard_render.py` 里的 style、stats grid、artifact card、各内容 section、链接、图片、格式化和 HTML escape helper 拆到 `dashboard_sections.py`。这一版解决的是 dashboard HTML 入口同时承担页面编排和所有局部 section 渲染的问题。

本版明确不改 dashboard payload schema、artifact collection、`write_dashboard()`、`scripts/build_dashboard.py` 用法、旧的 `minigpt.dashboard.render_dashboard_html` 导出，以及公开的 `minigpt.dashboard_render.render_dashboard_html` 入口。

## 关键改动

- `src/minigpt/dashboard_sections.py`：新增 section/helper 模块，负责 `style()`、`stats_grid()`、`artifact_grid()`、manifest/dataset/model/training/eval/pair/prediction/attention/chat/text/warning sections，以及 `image()`、`artifact_anchor()`、format 和 `escape()`。
- `src/minigpt/dashboard_render.py`：保留 `render_dashboard_html()` 入口，只负责 title、summary stats、section 顺序和 HTML 外壳编排。
- `tests/test_dashboard_render.py`：新增 section helper 断言，确认 dashboard render 入口使用新模块 helper，且 helper 自身保持 HTML escaping。
- `README.md`、`c/README.md` 和项目成熟度阶段讲解索引：记录 v177 的版本能力、证据位置和讲解入口。

## 运行验证

```powershell
python -B -m unittest tests.test_dashboard_render tests.test_dashboard -v

```

## tag 含义

`v177.0.0` 表示 MiniGPT 的 dashboard section helper split 版本。这个 tag 对应的能力是：dashboard HTML 页面入口仍保持稳定，而局部 section 和 helper 渲染由独立模块承接。

一句话总结：v177 把 dashboard 的“页面外壳编排”和“局部 section/helper 渲染”分开，让后续调整某个 dashboard 区块不再挤占主入口。
