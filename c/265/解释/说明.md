# v265 运行截图和解释

## 本版主题

v265 继续收口 promoted decision。v264 已经把 selected handoff review helper 拆出主文件，但 `promoted_training_scale_decision.py` 仍保留 JSON、CSV、Markdown、HTML 渲染与写出逻辑。本版新增 `promoted_training_scale_decision_artifacts.py`，把 artifact writer 和 HTML helper 从 baseline decision builder 中移走。

## 明确边界

本版不改变 promoted baseline selection、rejection rules、`decision_status`、summary 字段、recommendations 或 CLI 输出。`promoted_training_scale_decision.py` 仍然 re-export 原 writer/render 函数，所以现有测试、脚本和调用方可以继续从原模块导入。
