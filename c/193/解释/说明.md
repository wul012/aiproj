# v193 运行截图和解释

## 本版主题

v193 是 training portfolio batch external baseline。

v192 让单个 `run_training_portfolio.py` 可以接收外部 baseline checkpoint，并把 pair batch 从 same-checkpoint 自检推进到 baseline-vs-candidate 比较。v193 补齐批量入口：`run_training_portfolio_batch.py` 现在也能接收同样的 baseline 参数，并把它们透传给每个 variant 的 portfolio plan。

## 关键改动

- 更新 `scripts/run_training_portfolio_batch.py`，新增 `--pair-baseline-checkpoint`、`--pair-baseline-tokenizer`、`--pair-baseline-id`、`--pair-candidate-id`。
- 更新 `src/minigpt/training_portfolio_batch.py`，把 pair baseline 参数纳入 common config 和 variant override。
- 当 variant 自己覆盖 `pair_baseline_checkpoint` 但不覆盖 tokenizer 时，默认 tokenizer 跟随该 variant checkpoint 所在目录。
- 更新 `src/minigpt/training_portfolio_batch_artifacts.py`，在 JSON summary、CSV、Markdown 和 HTML 中显示 pair mode。
- 更新 `tests/test_training_portfolio_batch.py`，覆盖全局 baseline 透传、variant baseline 覆盖、pair mode 输出和默认行为不变。
- 更新 README、`c/README.md` 和项目成熟度阶段代码讲解索引。

## tag 含义

`v193.0.0` 表示 MiniGPT 第 193 个可回看版本：training portfolio batch 可以使用共享或 variant 覆盖的外部 baseline checkpoint，让候选矩阵不再只能做同 checkpoint 自检。运行截图与解释归档在 `c/193`。

## 一句话总结

v193 把 v192 的外部 baseline pair 能力从单次 portfolio 扩展到 batch variant 矩阵，让多候选训练计划也能明确比较 baseline 与 candidate。
