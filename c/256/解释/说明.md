# v256 promotion batch review propagation 运行证据

## 本版目标

v256 继续 v255 的 handoff batch review 链路，把 controlled handoff 中的 selected batch review/blocker/coverage 摘要传到 training scale promotion。

本版不改变 promotion 三态的判定规则，不把 selected batch review/blocker 默认升级成 promotion blocker。它只让 promotion evidence 在接受一个已完成 handoff 时，继续保留该 handoff 背后的 batch comparison 风险。

## 关键改动

- `src/minigpt/training_scale_promotion.py`：summary 新增 handoff selected batch review status、selected review/blocker/coverage 计数、handoff batch comparison totals 和 blocker reasons。
- `src/minigpt/training_scale_promotion_artifacts.py`：CSV、Markdown、HTML 输出同步展示这些 handoff batch review 字段。
- `tests/test_training_scale_promotion.py`：验证 clean/review/blocker 三种 handoff batch review 状态都能进入 promotion evidence，并验证 blocker recommendation。
- `README.md`、`c/README.md`、代码讲解索引：更新 v256 版本定位。

## 证据边界

这些截图证明 promotion evidence 能保留 handoff 上游的 batch comparison 风险。它们不是模型能力提升证明，也不是实际大规模训练结果。

## 一句话总结

v256 让 promotion acceptance 不再只看 handoff 完成和 variant artifacts，也能看到 handoff 背后的 selected batch review/blocker 风险。
