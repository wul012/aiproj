# v198 standard Chinese benchmark suite 运行说明

## 本版目标

v198 从 helper 收束回到轻功能增量：给 MiniGPT 的固定评估链路增加一个更完整的标准中文 benchmark prompt suite。

现有 `data/eval_prompts.json` 和默认 5 条用例保持兼容；新套件通过 `standard_zh_prompt_suite()`、`builtin:standard-zh`、`data/standard_zh_eval_prompts.json` 和 CLI 的 `--suite-name standard-zh` 提供，方便后续真实 checkpoint 做更稳定的重复比较。

## 关键改动

- `src/minigpt/eval_suites.py`
  - 新增 `standard_zh_prompt_suite()`，包含 10 个中文任务。
  - 新增 `named_prompt_suite()`，支持 `standard-zh` 和默认套件名称解析。
- `src/minigpt/eval_suite.py`
  - 新增 `load_builtin_prompt_suite()`。
  - `load_prompt_suite("builtin:standard-zh")` 可直接加载内置标准套件。
- `data/standard_zh_eval_prompts.json`
  - 文件版标准套件，内容与内置套件一致。
- `scripts/eval_suite.py` / `scripts/pair_batch.py`
  - 新增 `--suite-name standard-zh`，让 eval 和 pair-batch 不必手写文件路径也能使用标准套件。
- `tests/test_eval_suite.py`
  - 覆盖 built-in suite、`builtin:` URI、数据文件一致性和公共导出。

## 标准套件覆盖

标准套件共有 10 条中文用例，覆盖：

- continuation
- qa
- summary
- structured
- factual-consistency
- classification
- rewrite
- safety-boundary
- self-check
- comparison

它不是生产级大模型 benchmark，而是一个稳定、可重复、适合 MiniGPT 学习项目的中文评估输入集。

## tag 含义

`v198.0.0` 表示 MiniGPT v198 standard Chinese benchmark suite：完成标准中文 prompt suite 的内置 API、文件版、CLI 入口、测试和证据归档。

一句话总结：v198 让后续模型比较不再只依赖 5 条默认 prompt，而可以显式使用一个更完整、可复用的 10 条中文标准套件。
