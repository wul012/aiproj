# v294 运行截图和解释

## 本版主题

v294 把 promoted seed handoff assurance smoke 纳入 CI workflow hygiene。

新增能力：

- CI hygiene 要求 `.github/workflows/ci.yml` 包含 `scripts/check_promoted_seed_handoff_assurance_smoke.py`。
- CI hygiene 要求 assurance smoke 在 `scripts/run_test_coverage.py` 前执行。
- CI hygiene summary 新增 `required_order_count` 和 `order_violation_count`。
- CI hygiene Markdown/HTML summary 展示顺序违规计数。
- 测试覆盖 smoke 在 coverage 后的错误 workflow。

本版不改变 GitHub Actions 文件本身，不改变 smoke 脚本行为，不改变 coverage 阈值。
