# v115 运行截图说明：Playground Asset Split

## 本版主题

v115 把 `playground.py` 里的静态 CSS/JavaScript 页面资产拆到 `src/minigpt/playground_assets.py`。

这版不改变 playground payload、静态 HTML 结构、本地推理 API、streaming generation、request history、checkpoint comparison 或 pair generation 行为，只让页面资产和 Python 侧 payload/链接/HTML 组装分开维护。

## 关键代码

- `src/minigpt/playground_assets.py`
  - 新增 playground CSS/JavaScript 资产模块。
- `src/minigpt/playground.py`
  - 保留 payload、链接发现和 HTML section 组装，并把 `_style()`、`_script()` 委托给资产模块。
- `tests/test_playground_assets.py`
  - 新增资产层测试，锁住布局、交互字符串和 render 集成。

## 标签含义

`v115.0.0` 表示 MiniGPT 完成 playground asset split。它延续 v110-v114 的厚模块治理路线，把 UI 大文件治理落在稳定展示资产边界上，同时把 `registry.py` 保留为后续候选压力点。
