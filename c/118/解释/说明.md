# v118 运行截图说明：Benchmark Comparison Artifact Split

## 本版主题

v118 把 `benchmark_scorecard_comparison.py` 中的 artifact 输出和页面渲染逻辑拆成 `benchmark_scorecard_comparison_artifacts.py`。

这版不改变 scorecard comparison 的 report schema、baseline 选择、delta 计算、CLI 参数、输出文件名或旧导入路径，只把 JSON/CSV/case delta CSV/Markdown/HTML 写出和展示 helper 从比较计算模块中抽离出来。
