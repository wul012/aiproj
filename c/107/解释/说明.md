# v107 运行截图说明

## 本版主题

v107 是 release readiness comparison report-utils migration。

本版把 `src/minigpt/release_readiness_comparison.py` 中语义通用的 JSON 写出、UTC 时间、HTML 转义、dict/list 归一化和 CSV 标量 cell 处理迁到 `report_utils`，同时保留 release readiness comparison 自己的 baseline、delta、recommendation、layout 和 list-to-semicolon CSV 语义。

## 证据边界

这些截图证明的是 report-utils migration 没有破坏 release readiness comparison 的核心输出和展示链路。

它们不证明模型能力提升，也不证明新的发布策略更严格。v107 是一次轻量、定向的代码质量收束。

## Tag 含义

`v107.0.0` 表示 release readiness comparison 已经接入公共报告基础设施，但跨版本发布就绪对比的业务判断仍然留在本模块内部。

## 一句话总结

v107 让发布就绪总览和跨版本发布质量对比共享同一套报告基础工具，同时保持 baseline、delta、recommendation 和输出布局的原有语义。
