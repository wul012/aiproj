# v291 运行截图和解释

## 本版主题

v291 把 promoted seed handoff assurance checker 接回执行脚本。

新增能力：

- `execute_promoted_training_scale_seed.py` 新增 `--assurance-out-dir`。
- 执行脚本在写出 receipt-check 与 embedded receipt-check 后，可以继续运行 handoff assurance。
- assurance JSON/text 会写入指定目录。
- 主 handoff report 会嵌入 `handoff_assurance` 和 `handoff_assurance_outputs`。
- 主 CSV、Markdown、HTML 展示 assurance 状态、sidecar 状态、issue count 和输出路径。

本版不改变 handoff 决策，不改变 receipt schema，不改变默认执行行为。
