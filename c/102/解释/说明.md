# v102 运行截图与解释

## 本版主题

v102 是 dataset card report-utils migration。

本版把 `src/minigpt/dataset_card.py` 里的通用报告 helper 迁移到 `report_utils`，让 dataset card 复用公共 UTC 时间、JSON 写出、HTML 转义和 dict 归一化工具。

本版不改变：

- dataset card 字段结构。
- dataset identity、summary、provenance、quality、artifacts、recommendations 和 warnings 的生成策略。
- dataset quality 判断规则。
- Markdown/HTML 报告布局。
- experiment card、model card、project audit、release bundle 或 release gate 逻辑。
