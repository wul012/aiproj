# v87 run decision report utility migration 运行截图说明

## 本版主题

v87 继续 `report_utils` 收口路线，把 v74 的 `training_scale_run_decision.py` 迁移到公共报告工具上。迁移后，training scale run decision、promoted baseline decision、promoted seed generation、promoted seed handoff 和 controlled execution handoff 都开始复用同一套 JSON/CSV 写出、命令展示、Markdown/HTML 转义和 list/dict 归一化工具。

本版不改变 run decision 的业务语义：候选选择、rejected reasons、`ready/review/blocked` 状态和 execute command 生成规则保持不变。

## 关键代码

```text
src/minigpt/training_scale_run_decision.py
src/minigpt/report_utils.py
tests/test_training_scale_run_decision.py
tests/test_report_utils.py
```

`training_scale_run_decision.py` 继续负责读取 gated scale run comparison、筛选可执行候选、记录 rejected runs、选择 selected run、生成 execute command 和输出 decision report。变化在于输出和渲染基础层复用 `report_utils.py`。
