# v299 运行截图说明

## 本版主题

v299 把 v298 的 CI workflow order violation delta 从 release readiness comparison 继续传入 registry 和 maturity summary。目标是让项目级总览也能看见 smoke-before-coverage 顺序漂移，而不是只能在单个 comparison artifact 中发现。

本版不新增报表家族，不改变 maturity 的总体判定规则，也不修改训练或发布流程。它只做 registry/maturity 层的证据传播和展示。

## tag 说明

`v299.0.0` 的含义是：MiniGPT v299 registry and maturity CI order regression summary。

这一版让 `ci_workflow_order_regression_count` 和 `max_abs_ci_workflow_order_violation_delta` 从 readiness comparison 继续进入 registry 与 maturity 层。

一句话总结：v299 让 CI workflow order regression 成为项目总览级治理信号，成熟度汇总也能感知 smoke-before-coverage 漂移。
