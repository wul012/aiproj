# v271 运行说明

## 本版主题

v271 把 handoff clean batch-review requirement 带到 training-scale promotion acceptance。此前 v270 已经能在执行前 handoff 边界阻断不满足 clean requirement 的 workflow；本版让 promotion acceptance 也读取该 guard，并在 status 不是 `clean` 时阻断 promotion。

## 关键改动

- `build_training_scale_promotion()` 新增 `clean_batch_review_guard` 汇总，读取 handoff 的 clean batch-review 字段。
- `_issues()` 在 handoff 要求 clean batch-review 但 status 不是 `clean` 时增加 blocker。
- promotion CSV/Markdown/HTML 都展示 `handoff_require_clean_batch_review` 与 `handoff_clean_batch_review_status`。
- `scripts/build_training_scale_promotion.py` 打印 `handoff_clean_batch_review_status`，方便自动化日志直接判断。

## Tag 含义

`v271.0.0` 标记 training scale promotion clean batch gate。它表示 clean batch-review requirement 已经从 decision、workflow、handoff 继续推进到 promotion acceptance。

## 一句话总结

v271 让未满足 clean batch-review requirement 的 handoff 不能被误提升为 promotion-ready evidence，把自动化安全边界推进到了 promotion acceptance。
