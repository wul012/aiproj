# v237 portfolio comparison blocker action CLI gate 运行说明

## 本版主题

v237 给 training portfolio comparison 增加 opt-in 的 blocker action CLI gate。

v236 已经让比较报告输出结构化 `review_actions`。v237 让脚本在显式传入 `--fail-on-blocker-action` 时，如果存在 blocker action，就在报告写出后返回非零退出码。

## 本版边界

默认 CLI 行为保持兼容：即使存在 blocker action，不传 `--fail-on-blocker-action` 仍返回 0。

本版不改变 library 里的比较逻辑、评分、排序、baseline 选择和 artifact 渲染。

## Tag 含义

`v237.0.0` 表示 portfolio comparison blocker action CLI gate 版本。

这版不是新模型能力，而是把训练组合比较的结构化 blocker action 接入可选自动化失败语义。

## 一句话总结

v237 让 portfolio comparison 在保留默认兼容的同时，可以被 CI 或后续 workflow 用作 blocker action 门禁。
