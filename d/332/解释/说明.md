# v332 运行截图说明

## 本版主题

v332 在 v331 clean remediation gate 的基础上增加机器可读 issues：

```text
remediation_gate.issue_count
remediation_gate.issues[]
```

严格模式阻断时，summary 不只给出 `remediation_gate_decision=stop`，还会给出稳定 issue code、severity、category、action code、owner scope 和 count。
