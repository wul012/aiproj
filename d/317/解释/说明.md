# v317 运行截图说明

## 本版主题

v317 新增 tiny scorecard comparison smoke：

```text
baseline tiny smoke -> candidate tiny smoke -> benchmark scorecard comparison -> comparison smoke summary
```

它会真实训练两个 tiny checkpoint，各自跑 eval suite、generation quality、same-checkpoint pair batch 和 benchmark scorecard，然后比较两个 scorecard。

本版边界：这是 comparison plumbing evidence，不是模型能力提升证明。summary 中保留 `model_quality_claim=not_claimed`。

## 关键结论

- `scripts/run_tiny_scorecard_comparison_smoke.py` 能真实跑两个 tiny benchmark smoke。
- 两个 scorecard 会进入现有 comparison 模块，生成 JSON、CSV、case delta、Markdown、HTML。
- summary 保留模型能力边界，不把 tiny 1-iteration score delta 包装成模型进步。

## Tag 含义

`v317.0.0` 表示项目已经具备 CPU 级别的双 tiny scorecard comparison smoke：它能验证跨运行 benchmark scorecard comparison 链路，但不宣称 tiny 模型质量优劣。

一句话总结：v317 把 tiny benchmark 证据从单次 run 推进到双 run scorecard comparison，同时继续保持能力结论克制。
