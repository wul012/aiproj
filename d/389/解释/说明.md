# v389 workflow handoff maturity CI reasons 运行说明

## 本版主题

v389 把 training scale run decision 中的 selected/global batch maturity CI regression reason counts，继续带到 workflow 和 handoff。

本版解决的问题是：v388 的 decision 报告已经知道 CI 回归原因，但 workflow/handoff 是执行前审查入口，也应该直接看到这些原因。v389 让执行前报告不只显示 CI regression count，还显示 reason counts。

## 关键改动

- `src/minigpt/training_scale_workflow.py`
  - workflow summary 新增 selected/global reason counts。
  - workflow recommendations 会带上 observed reason-count detail。
- `src/minigpt/training_scale_workflow_artifacts.py`
  - CSV、Markdown、HTML 展示 selected/global reason counts。
- `scripts/run_training_scale_workflow.py`
  - stdout 打印 selected/global reason-count JSON。
- `src/minigpt/training_scale_handoff.py`
  - clean batch-review guard 和 handoff summary 保存 selected/global reason counts。
  - handoff recommendations 带上 observed reason-count detail。
- `src/minigpt/training_scale_handoff_artifacts.py`
  - CSV、Markdown、HTML 展示 selected/global reason counts。
