# v417 运行截图说明：training portfolio suite-design review action

## 本版目标

v417 把 v416 maturity narrative 暴露的 release-readiness suite-design regression 继续带入 training portfolio comparison。目标是让训练组合比较在候选分数更高时，也能因为 suite-design readiness regression 生成明确 blocker，而不是只显示泛泛的 maturity review。

本版不改变训练、benchmark、maturity 的原始生成规则；它只读取 maturity narrative summary 中已有字段，并把它们用于训练组合比较的选择审查。
