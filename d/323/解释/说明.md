# v323 运行截图说明

## 本版主题

v323 接在 v322 的 configurable decision threshold 后面，把 blocked 决策里最常见的 rubric threshold 信息提升到顶层 summary：

```text
decision_first_threshold_candidate
decision_first_threshold_score
decision_first_threshold_min
decision_first_threshold_margin
```

本版边界：不改 decision 规则，不改训练，不声称 tiny 模型变强；只让 tiny comparison smoke 的阻断原因更容易在 CI 和归档里阅读。

## 关键结论

- v323 让 `rubric_avg_score below ...` 不再只藏在 `benchmark_scorecard_decision.json` 的 blockers 里。
- 顶层 summary 直接展示第一个 threshold-blocked candidate。
- `margin` 使用 `rubric_avg_score - min_rubric_score`，负数表示离通过门槛还差多少。
- 这仍然是 smoke/CI 可读性增强，不是模型质量提升。
- 全量 591 个测试通过，source encoding、py_compile 和 diff check 都通过。
