# v385 batch maturity CI reasons 运行说明

## 本版主题

v385 把 v384 的 training portfolio comparison maturity CI regression reason counts 带进 training portfolio batch 的 comparison review summary。

本版解决的问题是：batch 层已经能汇总每个 variant 的 comparison review，但之前只保留 CI 回归数量和名字，不保留原因计数。现在 batch Markdown、HTML 和 CLI 输出都能看到 `maturity_ci_regression_reason_counts`，方便直接判断 batch 中哪些 variant 的 CI 阻断来自什么原因。

## 关键改动

- `src/minigpt/training_portfolio_batch.py`
  - `_comparison_review_summary()` 新增 `maturity_ci_regression_reason_counts`。
  - 该字段从 nested comparison summary 归一化而来。
- `src/minigpt/training_portfolio_batch_artifacts.py`
  - Markdown/HTML 新增 `CI regression reasons`。
  - 比较摘要和比较面板都能展示原因计数。
- `tests/test_training_portfolio_batch.py`
  - 覆盖 reason-count carryover、Markdown、HTML 和 batch 运行路径。
