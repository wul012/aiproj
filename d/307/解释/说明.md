# v307 运行截图说明

## 本版主题

v307 把 execution handoff 中的 batch CI regression 字段继续传递到 training scale promotion。

本版不改变训练模型、不新增 promotion index，也不扩大 promoted seed 链路；重点是让 promotion acceptance 在接收 handoff 时能看到并阻断 CI-regressed batch 证据。

## 关键结论

- promotion summary 现在包含 selected 和 aggregate handoff CI regression 字段。
- promotion CSV/Markdown/HTML/CLI 都能显示 `handoff_batch_maturity_ci_regression_count`。
- clean-required promotion 不再只信 `handoff_clean_batch_review_status=clean`，CI regression count 大于 0 时仍会 blocked。

## 一句话总结

v307 的运行证据说明 CI-regressed handoff batch 风险已经进入 promotion acceptance，promotion 前不会漏掉 v306 传来的 CI 风险信号。
