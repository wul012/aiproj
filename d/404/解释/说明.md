# MiniGPT v404 运行说明

## 本版目标

v404 拆分 `src/minigpt/maintenance_policy.py`。

拆分前它有 648 行，同时负责 maintenance batching、proposal decision、governance stabilization review、proposal routing 和 artifact facade。职责已经偏宽。

本版拆成：

- `maintenance_policy_common.py`
- `maintenance_policy_batching.py`
- `maintenance_policy_governance.py`
- `maintenance_policy.py` facade

## 不做什么

- 不改默认七条治理链。
- 不改 report schema。
- 不新增治理链。
- 不改变模型训练或评估结果。

本版验证时还发现当前环境存在 `HTTP_PROXY/HTTPS_PROXY/ALL_PROXY=http://127.0.0.1:7897`，导致本地 `test_server.py` HTTP 测试走代理返回 502。因此本版也把 `tests/test_server.py` 的本地 HTTP helper 改成无代理 opener，保证 `127.0.0.1` 测试不被代理劫持。
