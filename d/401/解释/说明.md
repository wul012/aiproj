# v401 CI wrapper benchmark-history summary 运行说明

## 本版主题

v401 让 CI tiny scorecard wrapper plan 写入 `benchmark_history_summary`。现在 plan JSON 和 text 不只记录 history artifact 的 hash，也能直接说明它是 `tiny-smoke`、`not_claimed`，并保留 readiness failed reasons。

## 关键改动

- `scripts/run_ci_tiny_scorecard_comparison_smoke.py`
  - 新增 `build_benchmark_history_summary()`。
  - `ci_tiny_scorecard_smoke_plan.json` 新增 `benchmark_history_summary`。
  - `ci_tiny_scorecard_smoke_plan.txt` 新增 benchmark-history 语义行。
- `tests/test_ci_tiny_scorecard_smoke.py`
  - 覆盖 history 缺失、fixture 解析、真实 wrapper smoke 输出。
  - 与 v400 plan checker 测试一起验证 plan 与 checker 的语义一致。
