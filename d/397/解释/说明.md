# v397 governance chain value review 运行说明

## 本版主题

v397 是治理链收口版本。它把“v382-v396 连续传递 CI reason 字段，是否已经开始重复”的判断写入现有 maintenance policy，而不是继续新增一条治理链。

## 关键改动

- `src/minigpt/maintenance_policy.py`
  - 默认七条 governance chains 新增 value、duplicate-risk、recent-expansion 和 guardrail 字段。
  - summary 新增 value/risk/freeze 计数。
  - recommendation 会提示冻结高重叠链的新字段。
- `src/minigpt/maintenance_policy_governance_artifacts.py`
  - CSV、Markdown、HTML 展示新增字段。
- `scripts/check_maintenance_batching.py`
  - stdout 打印 `governance_high_duplicate_risk_count` 和 `governance_freeze_new_fields_count` 等字段。
- `tests/test_maintenance_policy.py`
  - 覆盖默认七链计数、freeze recommendation、渲染输出和 CLI stdout。
