# v463 archived path portability CI gate

## 本版目标

v463 把刚刚修复的 CI 问题沉淀成一个可重复验证的 gate：归档 JSON 中保存的 Windows 风格路径，例如 `d\448\解释\...`，在 Linux runner 上必须能经过统一规范化后解析到真实文件。

本版不改变 v448 归档内容，也不重写旧证据。它只新增检查器、CI workflow 步骤和 hygiene 约束，防止同类路径问题再次在 GitHub Actions 上出现。
