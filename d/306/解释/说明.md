# v306 运行截图说明

## 本版主题

v306 把 training-scale run decision 中的 batch CI regression 字段继续传递到 consolidated workflow 和 execution handoff。

本版不改变训练模型、不新增新报告层，重点是修补下游证据链断点：workflow 和 handoff 现在可以直接看见 `batch_maturity_ci_regression_count`、selected CI regression 计数和 CI-regressed batch 名称。

## 关键结论

- workflow summary 不再只携带 `clean_batch_review_status`，还携带 selected 和 aggregate CI regression 证据。
- handoff guard 不再只相信状态字符串；strict clean-batch 模式会直接检查 CI regression 数量。
- CLI 输出增加机器可读 key/value 字段，方便后续 CI 或脚本读取。

## 一句话总结

v306 的截图证明 CI-regressed batch 风险已经从 decision 层继续进入 workflow 与 handoff 层，执行交接前不会只剩一个模糊的 review 状态。
