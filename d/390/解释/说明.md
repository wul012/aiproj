# v390 promotion maturity CI reasons 运行说明

## 本版主题

v390 把 workflow/handoff 中的 selected/global maturity CI regression reason counts，继续带到 training scale promotion。

本版解决的问题是：promotion 报告原来能看到 handoff CI regression count 和 names，但不能直接解释这些 CI 回归由哪些原因组成。v390 让 promotion review 不再需要回查 handoff 或 decision JSON 才能看到原因。

## 关键改动

- `src/minigpt/training_scale_promotion.py`
  - suite guard 和 clean batch-review guard 读取 selected/global reason counts。
  - promotion summary 保存 selected/global reason counts。
  - promotion recommendations 在 CI regression 路径下带上 observed reason-count detail。
- `src/minigpt/training_scale_promotion_artifacts.py`
  - CSV、Markdown、HTML 展示 selected/global reason counts。
- `scripts/build_training_scale_promotion.py`
  - stdout 打印 selected/global reason-count JSON。
- `tests/test_training_scale_promotion.py`
  - fixture 和断言覆盖 JSON summary、CSV、Markdown、HTML、CLI stdout 和 strict clean-batch recommendation。

## 一句话总结
