# v325 运行截图说明

## 本版主题

v325 把 v324 的 threshold profile 从 tiny smoke wrapper 下沉到真正的 benchmark scorecard decision artifact：

```text
benchmark_scorecard_decision.json
benchmark_scorecard_decision.md
benchmark_scorecard_decision.html
  -> threshold profile
run_tiny_scorecard_comparison_smoke.py
  -> consume decision summary profile
```

本版边界：不改训练、不改 scorecard scoring、不改 promotion 规则；只是让 decision artifact 自己成为 threshold profile 的权威来源。

## 关键结论

- v325 的 threshold profile 是 decision artifact 的一部分。
- tiny smoke 仍能展示 profile，但现在只是消费底层证据。
- profile 只统计非 baseline promotion candidates；baseline 即使低于 rubric threshold，也不会进入 threshold profile。
- 这仍然是 evidence 可读性和治理链路增强，不是模型质量提升。
- 全量 592 个测试通过，source encoding、py_compile 和 diff check 都通过。

## Tag 含义

`v325.0.0` 表示 benchmark scorecard promotion decision 已经能原生输出 threshold blocker profile，tiny smoke 不再是该 profile 的唯一来源。

一句话总结：v325 把 threshold profile 从 smoke 汇总字段提升为 decision artifact 的正式证据结构。
