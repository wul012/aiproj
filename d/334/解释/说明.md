# v334 运行截图说明

## 本版主题

v334 新增 `scripts/check_tiny_scorecard_comparison_smoke.py`，用于读取已经生成的 tiny scorecard comparison smoke summary，并独立判断 summary 是否满足 CI 消费契约。

它检查：

```text
commands[] 是否全部 pass
关键 artifact flags 是否存在
remediation_gate 是否通过或是否为允许的 strict stop
model_quality_claim 是否仍为 not_claimed
```

## 关键结论

- v334 不重新运行 tiny training。
- v334 不改变原 smoke summary 结构。
- checker report 是消费后的验收结论，原 summary 仍是事实来源。
- `model_quality_claim=not_claimed` 被纳入检查，防止把 tiny smoke 误写成真实模型能力证明。

## Tag 含义

`v334.0.0` 表示 tiny scorecard comparison smoke summary 已有独立 checker，可以在 CI 中复查产物完整性、gate 状态、命令状态和质量声明边界。

一句话总结：v334 让 tiny scorecard comparison smoke 从“生成证据”推进到“证据可被独立验收”。
