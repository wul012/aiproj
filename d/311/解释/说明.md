# v311 运行截图说明

## 本版主题

v311 把 promoted baseline decision 中的 handoff batch CI regression 和 `comparison_exclusion_reasons` 继续带入 promoted next-cycle seed。
这样 seed 不只保留 selected baseline，也能说明哪些 rejected decision inputs 因 CI regression 留在 baseline 之外。

本版不执行训练，只加强下一轮 seed 的治理证据。

## 证据链角色

v311 的作用是让 promoted next-cycle seed 成为 decision 之后的证据承接点。
如果 decision 层知道 rejected input 因 handoff batch CI regression 被挡住，seed 层也会继续展示这个事实。

## Tag 含义

`v311.0.0` 表示 MiniGPT 第 311 个可回看版本：promoted next-cycle seed 已经能消费和展示 promoted baseline decision 的 CI regression exclusion context。
