# v384 portfolio maturity CI reasons 运行说明

## 本版主题

v384 把 v383 maturity narrative 的 CI workflow regression reason counts 带入 training portfolio comparison。

本版解决的问题是：训练组合比较层已经能阻断 CI-regressed portfolio，但原先只展示 CI 回归数量，缺少“为什么回归”的原因计数。现在 portfolio summary、comparison summary、review action、recommendation、CSV、Markdown 和 HTML 都能显示 maturity CI reason counts。

## 关键改动

- `src/minigpt/training_portfolio_comparison.py`
  - 从 `maturity_narrative.summary` 读取 `release_readiness_ci_workflow_regression_reasons` 和 `release_readiness_ci_workflow_regression_reason_counts`。
  - 汇总 `maturity_ci_regression_reason_counts`。
  - 在 delta explanation 中展示 reason counts。
- `src/minigpt/training_portfolio_comparison_review.py`
  - best-score CI regression blocker action 的 evidence 现在保留 reasons 和 reason counts。
  - recommendation 文本带上原因计数。
- `src/minigpt/training_portfolio_comparison_artifacts.py`
  - CSV 新增 reason 字段。
  - Markdown/HTML 新增 `CI regression reasons` 和 `ci_reasons=...`。
- `tests/test_training_portfolio_comparison.py`
  - fixture 写入 maturity CI regression reasons。
  - 覆盖 summary、action evidence、recommendation、CSV、Markdown、HTML。
