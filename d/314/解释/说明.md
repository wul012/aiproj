# v314 运行截图说明

## 本版主题

v314 把 promoted seed handoff 的 schema-v2 receipt CI-regression 契约继续向外提升到 handoff assurance 和 CI smoke summary。v313 已经能验证 receipt 与 embedded sidecar 的 CI 字段一致，本版补上最后一层可见性：CI 或交接者只看 assurance/smoke summary，也能直接看到 receipt schema version、selected/aggregate CI regression counts 和 comparison exclusion reasons。

本版不改变训练命令、clean-evidence gate、clean-batch gate 或模型训练行为，只收紧 receipt evidence chain 的外层摘要。

## 关键结论

- embedded receipt check 新增扁平字段：`receipt_schema_version`、`receipt_selected_handoff_batch_maturity_ci_regression_count`、`receipt_handoff_batch_maturity_ci_regression_count`、`receipt_comparison_exclusion_reasons`。
- handoff assurance 新增对应的 `embedded_receipt_check_receipt_*` 字段。
- assurance smoke summary 会断言 schema version 为 2，并检查 selected/aggregate CI regression counts 与 comparison exclusion reasons。
- main handoff CSV、Markdown、HTML 也展示 assurance 层 receipt contract 字段。
