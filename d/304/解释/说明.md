# v304 运行截图说明

## 本版主题

v304 把 training portfolio batch 的 CI regression review context 继续传入 gated training-scale run 和 training-scale run comparison。目标是让 `maturity_ci_regression_count`、`maturity_ci_regression_names` 不只停在 batch 报告里，而能被训练规模下游汇总继续看到。

本版不改变训练、评估、gate 判定或 batch comparison 逻辑，只增强字段传递、报告输出和 CLI 可见性。

## tag 说明

`v304.0.0` 的含义是：MiniGPT v304 training scale run CI regression carryover。

这一版把 batch-level CI regression evidence 继续带到 gated training-scale run 和 run comparison。

一句话总结：v304 让训练规模链路下游也能直接看见 batch 里的 CI 回归风险。
