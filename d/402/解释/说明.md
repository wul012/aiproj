# v402 eval suite design coverage 运行说明

## 本版主题

v402 从 CI wrapper 回到评估设计本身：新增 suite design coverage summary，并把它写入 tiny standard benchmark smoke 的 capped suite payload 和顶层 summary。

## 关键改动

- `src/minigpt/eval_suite_design.py`
  - 新增 `summarize_prompt_suite_design()`。
  - 汇总 task type、difficulty、tags、token budget、seed uniqueness、expected behavior 和 comparison readiness。
- `scripts/run_tiny_standard_benchmark_smoke.py`
  - capped suite JSON 新增 `design_summary`。
  - smoke summary 新增 `suite_design`。
  - text summary 新增 `suite_design_*` 字段。
- `tests/test_eval_suite.py`
  - 覆盖 standard-zh design readiness 和 narrow suite warning。
- `tests/test_tiny_standard_benchmark_smoke.py`
  - 覆盖 capped token budget、rendered summary 和真实 tiny smoke 链路。
