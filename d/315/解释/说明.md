# v315 运行截图说明

## 本版主题

v315 增加 CPU 友好的 tiny standard benchmark smoke。它真实跑通一条小型模型能力证据链：

```text
tiny corpus -> train.py -> checkpoint.pt -> eval_suite.py -> generation quality -> benchmark scorecard -> smoke summary
```

本版的边界很重要：`status=pass` 只表示证据链跑通、产物齐全；不表示 1 iteration tiny 模型已经具备强模型质量。实际 smoke 中 `scorecard_overall_status=fail` 是合理结果，说明 scorecard 没有被包装成“质量通过”。

- `scripts/run_tiny_standard_benchmark_smoke.py` 能在 CPU 上跑一个极小真实训练链路。

- 这版开始把项目重心从纯治理链条回拨到真实模型能力证据链。
