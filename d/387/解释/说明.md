# v387 scale run comparison maturity CI reasons 运行说明

## 本版主题

v387 把 v386 单个 gated scale-run 中的 batch maturity CI regression reason counts 带到 training scale run comparison。

本版解决的问题是：多 run comparison 已经能统计 batch CI 回归数量和名字，但原先没有聚合原因计数。现在 `training_scale_run_comparison.json`、CSV、Markdown、HTML 和 `scripts/compare_training_scale_runs.py` stdout summary 都能看到 `batch_maturity_ci_regression_reason_counts`。

## 关键改动

- `src/minigpt/training_scale_run_comparison.py`
  - run row 新增 `batch_maturity_ci_regression_reason_counts`。
  - comparison summary 新增聚合后的 `batch_maturity_ci_regression_reason_counts`。
- `src/minigpt/training_scale_run_comparison_artifacts.py`
  - CSV 新增同名列。
  - Markdown/HTML 新增 `Batch CI regression reasons` / `CI regression reasons`。
- `tests/test_training_scale_run_comparison.py`
  - 覆盖 reason-count carryover、聚合、CSV、Markdown、HTML 输出。
