# v388 scale run decision maturity CI reasons 运行说明

## 本版主题

v388 把 scale-run comparison 中已经聚合的 `batch_maturity_ci_regression_reason_counts` 带到 training scale run decision。

这版也顺手把训练治理链里重复的 reason-count 解析和格式化 helper 收到 `report_utils.py`，让 batch、comparison、review、scale-run 和 decision 层使用同一套规则。

## 关键改动

- `src/minigpt/training_scale_run_decision.py`
  - summary 新增 `selected_batch_maturity_ci_regression_reason_counts`。
  - summary 新增 `batch_maturity_ci_regression_reason_counts`。
- `src/minigpt/training_scale_run_decision_artifacts.py`
  - CSV、Markdown、HTML 展示 selected/global reason counts。
- `scripts/decide_training_scale_run.py`
  - stdout 打印 selected/global reason-count JSON。
- `src/minigpt/report_utils.py`
  - 新增 `positive_int_mapping()` 和 `format_mapping()`。
  - 训练治理链多个模块改为复用共享 helper。
