# v305 运行截图说明

## 本版主题

v305 把 batch CI regression context 接入 training-scale run decision。目标是让 `--require-clean-batch-review` 不只检查 batch review actions、blockers 和 coverage regressions，也检查 CI-regressed batch evidence。

本版不改变训练执行、gate 策略或 readiness score，只增强 decision 层的字段传递、渲染和严格 clean gate。

## tag 说明

`v305.0.0` 的含义是：MiniGPT v305 training scale decision CI regression gate。

这一版把 batch-level CI regression evidence 接入 training-scale run decision 和 clean batch-review execution gate。

一句话总结：v305 让 CI 回归风险不只是显示在 comparison 中，也能参与是否允许 clean execute automation 的判断。
