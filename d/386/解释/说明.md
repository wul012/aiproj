# v386 scale run maturity CI reasons 运行说明

## 本版主题

v386 把 v385 batch review summary 中的 maturity CI regression reason counts 带到 gated training scale run 的 `batch_summary`。

本版解决的问题是：scale-run 报告已经能保存 batch CI 回归数量和名字，但原先没有保存原因计数。现在 `training_scale_run.json`、CSV、Markdown、HTML 和 `scripts/run_training_scale_plan.py` stdout 都能看到 `maturity_ci_regression_reason_counts`。

## 关键改动

- `src/minigpt/training_scale_run.py`
  - `_batch_summary()` 新增 `maturity_ci_regression_reason_counts`。
  - CSV 新增同名列。
  - Markdown/HTML Batch 区域新增 `CI regression reasons`。
- `tests/test_training_scale_run.py`
  - 覆盖 reason-count carryover、Markdown 和 HTML 输出。
