# v395 promoted seed section split 运行说明

## 本版主题

v395 是维护性拆分版本，把 `promoted_training_scale_seed_artifacts.py` 中的 HTML 页面渲染和 section helper 拆到 `promoted_training_scale_seed_sections.py`。

本版解决的问题是：v394 后 seed artifact 文件已经接近 500 行，后续如果继续推进 seed handoff reason context，会继续压大 artifact writer。v395 先把视图层拆出去，保持产物编排层更轻。

## 关键改动

- `src/minigpt/promoted_training_scale_seed_sections.py`
  - 新增 seed HTML renderer、baseline section、command section、source table、list section、style 和 card helpers。
- `src/minigpt/promoted_training_scale_seed_artifacts.py`
  - 继续负责 JSON、CSV、Markdown、HTML 文件写入和 output path 编排。
  - 通过导入 section module 保持 `render_promoted_training_scale_seed_html()` 对外可用。
- `tests/test_promoted_training_scale_seed.py`
  - 既有测试覆盖 render/write/export 行为不变。
