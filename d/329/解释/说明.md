# v329 运行截图说明

## 本版主题

v329 给 remediation plan 增加可执行 metadata：

```text
remediation_plan[].action_code
remediation_plan[].severity
remediation_plan[].owner_scope
remediation_plan[].target_artifacts
```

本版边界：不改训练、不改评分、不改 promotion 判定；只是让 v327/v328 的 remediation 建议更适合脚本消费和人工分工。

## 关键结论

- v329 让 remediation action 从自然语言建议升级为可执行任务条目。
- action code 稳定，适合后续 CI、dashboard 或计划脚本消费。
- severity 和 owner scope 让人工审阅知道优先级和职责边界。
- target artifacts 明确下一步应该查看哪些证据。
- 全量测试、source encoding、py_compile 和 diff check 都通过。
