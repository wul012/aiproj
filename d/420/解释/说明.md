# v420 运行截图说明：training scale promotion suite-design guard

## 本版目标

v420 把 v419 handoff 中的 suite-design regression context 带入 training scale promotion。目标是让 promotion acceptance 阶段也能看到 `handoff_batch_maturity_suite_design_regression_count` 和 affected portfolio names，避免 strict clean-batch promotion 只检查 CI regression 而漏掉 benchmark-suite blocker。

本版不执行训练，不改变 handoff 生成逻辑，也不重新判断 suite-design regression；只在 promotion 层消费 handoff 已有字段，并让 strict clean-batch promotion guard 识别这类风险。

## 命令摘要

本轮用最小 handoff fixture 生成 promotion HTML：

```powershell
python - <<'PY'
from minigpt.training_scale_promotion import build_training_scale_promotion
report = build_training_scale_promotion("runs/v420-training-scale-promotion-suite-design/handoff")
print(report["summary"]["promotion_status"])
print(report["summary"]["handoff_batch_maturity_suite_design_regression_count"])
print(report["recommendations"])
PY
```

关键输出：

```text
promotion_status=blocked
handoff_clean_batch_review_status=clean
handoff_batch_maturity_suite_design_regression_count=1
handoff_batch_maturity_suite_design_regression_names=["suite-design-risk"]
```
