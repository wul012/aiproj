# v424 运行截图说明：promoted training scale seed suite-design carryover

## 本版目标

v424 把 v423 promoted baseline decision 中的 suite-design regression context 继续带入 promoted training scale next-cycle seed。目标是让下一轮训练规模计划种子既知道选中的 baseline 是 clean 的，也保留 rejected promoted decision inputs 中仍有 suite-design regression 的事实。

本版不执行训练，不改变 baseline decision 的选择逻辑，也不新增新的治理链。它只让 seed 层消费 decision summary 里的 suite-design 字段，并写入 JSON/CSV/Markdown/HTML、CLI stdout、recommendations 和运行截图证据。
