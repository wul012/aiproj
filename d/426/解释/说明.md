# v426 运行截图说明：promoted seed handoff receipt suite-design contract

## 本版目标

v426 把 v425 handoff 报告里的 suite-design regression context 写入 automation receipt schema v3，并让 receipt check、embedded receipt check 和 handoff assurance 都校验这些字段。目标是让最终自动化收据不只证明 CI regression 字段一致，也证明 suite-design rejected evidence 没有在 sidecar 合同里丢失。

本版不执行训练，不改变 seed handoff 的 baseline，也不新增治理链。它只是把既有 suite-design 字段提升到 receipt/assurance 合同层。
