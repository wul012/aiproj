# MiniGPT Project Overview

MiniGPT is a from-scratch learning project for tokenizer, dataset, training, evaluation, and small GPT-style model inspection. In its later stages it also became an AI governance project: publication receipt reports, contract checks, indexes, reviews, and no-promotion boundaries record what each model or evidence artifact may be used for.

The project has two important layers:

- The model layer covers tokenizer behavior, tiny training runs, holdout evaluation, scorecards, and capability diagnostics.
- The governance layer covers publication receipt evidence, lookup-only consumption, contract checks, artifact indexes, and release/readiness style reports.

The maintenance goal after v1130 is readability. Historical files remain stable, but new work should use shorter names, clearer docs, and layered script directories.
