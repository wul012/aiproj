# Artifact Map

The artifact map is a maintenance view over `f/<version>` evidence directories. It helps reviewers find screenshots, summaries, and machine-readable reports without manually opening every folder.

The v1134 map records:

- version
- evidence directory
- whether `解释/说明.md` exists
- screenshot count
- report directory count
- JSON/CSV/Markdown/HTML artifact counts
- completion status

Use this map when reviewing late-stage maintenance or publication receipt versions. It does not replace the JSON reports; it points readers to where those reports live.
