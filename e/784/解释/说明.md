# v784 training portfolio summary split

## 目标

v784 是维护拆分版本，不新增 training portfolio 功能。目标是把 `training_portfolio_comparison.py` 中的 portfolio artifact 读取与摘要归一化逻辑拆到独立模块，让主比较模块只保留横向比较、delta、baseline 选择和推荐决策。

## 关键变化

- 新增 `src/minigpt/training_portfolio_comparison_portfolio.py`。
- `src/minigpt/training_portfolio_comparison.py` 从 641 行降到 403 行。
- 新模块承接 `build_training_portfolio_summary`，负责读取 run manifest、benchmark scorecard、dataset card、maturity narrative、generation quality 等 artifact。
- 主模块继续输出原来的 comparison report schema，不改变 CLI、artifact writer 或测试入口。
