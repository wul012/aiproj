# v564 required-term pair no-pair-id loss-balanced first-token stability 运行说明

## 本版目标

v563 把 v562 missed seeds 定位到 first-token ranking gap。v564 因此新增一个 first-token repair corpus mode：在 v561 no-pair-id loss-balanced 目标上加入 `fixed=f/fi/fix` 和 `loss=l/lo/los` 前缀目标，测试它是否能提升三 seed 稳定性。

本版不扩大模型、不改 decode、不做 held-out。它只验证 explicit prefix rows 是否能把 v562 的 `1/3` 推高。

## 结论

v564 说明 first-token prefix rows 会改变 missed seed 形态，但没有提升总体稳定性。下一步更合理的是比较 v562/v564 的 seed 迁移，再决定是否做 seed-specific policy、objective mix，或重新设计更稳的首 token 训练目标。
