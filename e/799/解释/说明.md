# v799 route promotion decision index contract check 运行说明

## 本版目标

v799 接在 v798 route promotion decision index 后面，验证 v798 index 是否能从记录的 v797 source decision 重新推导出来。

这不是新的训练或新的模型质量判断，而是给 accepted route ledger 增加可复核入口：如果 index 被手工篡改、source decision 路径丢失、route id 或 boundary/claim 变化，本版 check 会失败。

## 运行命令

```powershell
python -B scripts\check_model_capability_route_promotion_decision_index.py --decision-index e\798\解释\model-capability-route-promotion-decision-index --out-dir e\799\解释\model-capability-route-promotion-decision-index-check --require-pass --force
```
