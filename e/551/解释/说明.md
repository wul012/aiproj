# v551 required-term pair seed config held-out gap diagnostic 运行说明

## 本版目标

v551 承接 v550 的 held-out replay 结果，不再泛泛说“还有一个缺口”，而是把缺口拆成可复核的诊断行：prompt surface、seed、selected config、missed term、profile 级 hit/miss 和生成预览。

本版不训练新 checkpoint，也不改 selected policy。它只读取 v550 主报告和内嵌 replay sidecar，把剩余 `equals + 1535` 的失败原因固定成后续修复入口。
