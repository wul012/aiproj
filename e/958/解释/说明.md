## 边界说明

v958 是 review，不是 publication 本身，也不是 promotion。它只批准 v957 packet index 进入 lookup-only publication；模型质量声明仍限定为 `bounded_randomized_target_hidden_holdout_claim_only`，`promotion_ready` 与 `approved_for_promotion` 仍为 false。

如果 v957 index 失败、next step 不匹配、granted use 漂移、source packet/check 不存在、source evidence 丢失、source packet row 或 packet index row 被删除，review 会失败。

一句话总结：v958 把 v957 lookup-only packet index 从“可索引”推进到“可发布前 review 通过”，但仍然不提升模型生产质量声明。
