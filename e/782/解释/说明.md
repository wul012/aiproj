# v782 promoted comparison summary split

## 目标

v782 是维护拆分版本，不新增 promoted comparison 功能。目标是把 `promoted_training_scale_comparison.py` 中 200 多行的 summary 聚合逻辑拆到独立模块。

## 关键变化

- 新增 `src/minigpt/promoted_training_scale_comparison_summary.py`。
- `src/minigpt/promoted_training_scale_comparison.py` 从 661 行降到 420 行。
- 新模块承接 comparison status、handoff clean-batch counts、CI regression reason counts、suite mismatch totals、comparison-ready metrics 等 summary 字段。
- 主 comparison 模块保留 promotion index loading、row normalization、comparison input resolution、comparison execution、blockers 和 clean-batch guard。

## 验证

- `python -m py_compile src\minigpt\promoted_training_scale_comparison.py src\minigpt\promoted_training_scale_comparison_summary.py`
- `python -m pytest tests\test_promoted_training_scale_comparison.py -q -o cache_dir=runs\pytest-cache-v782-focused`
- 结果：`13 passed`
