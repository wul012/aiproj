# v929 randomized holdout publication registry entry

## 目标

v929 读取 v928 publication decision index，生成一个面向下游查找的 registry entry。

它把 v928 的索引信息压缩成一个登记条目，保留关键边界：

- bounded publication 已接受。
- accepted claim 仍然只有 1 个。
- blocked claim 仍然有 3 个。
- candidate case count 仍然是 20。
- allowed use 仍然是 `bounded_model_capability_governance_only`。
- model quality claim 仍然是 `bounded_randomized_target_hidden_holdout_claim_only`。
- promotion 仍然是 false。

## 本版不做什么

- 不重新训练模型。
- 不重新 replay。
- 不生成完整 registry manifest。
- 不批准 production promotion。

```powershell

```
