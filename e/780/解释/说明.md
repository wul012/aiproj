# v780 release readiness panel split

## 目标

v780 是维护拆分版本，不新增 release readiness 功能。目标是把 `release_readiness.py` 中的各类 panel builder 拆出，避免 dashboard 主流程继续混杂 panel 细节文本。

## 关键变化

- 新增 `src/minigpt/release_readiness_panels.py`。
- `src/minigpt/release_readiness.py` 从 701 行降到 391 行。
- 新模块承接 registry、bundle、audit、gate、request history、benchmark history、maturity、CI workflow、test coverage panel。
- 主模块保留路径解析、JSON 读取、dashboard assembly、summary、actions、evidence 和 artifact export。

## 验证

- `python -m py_compile src\minigpt\release_readiness.py src\minigpt\release_readiness_panels.py`
- `python -m pytest tests\test_release_readiness.py -q -o cache_dir=runs\pytest-cache-v780-focused`
- 结果：`14 passed`
