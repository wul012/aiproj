## 边界说明

v952 只审核 lookup-only index，不批准生产 promotion。它会确认 v951 index 的 source evidence 仍存在且 pass，source publication 与 source publication check 仍存在，published use 仍是 `downstream_governance_lookup_only`。

如果 index 失败、source evidence 缺失、source publication/check 缺失、published use 漂移、或 no-promotion 字段被打开，本版 review 会失败。

一句话总结：v952 把 v951 publication index 从“可查阅索引”推进到“下游 receipt 可接收”的审核状态。
