# v781 maturity release context split

## 目标

v781 是维护拆分版本，不新增 maturity 功能。目标是把 `maturity.py` 中的 release readiness comparison context 和 trend status 判定拆到独立模块。

## 关键变化

- 新增 `src/minigpt/maturity_release_context.py`。
- `src/minigpt/maturity.py` 从 664 行降到 494 行。
- 新模块承接 `build_release_readiness_context` 和 `release_readiness_trend_status`。
- 主 maturity 模块保留版本发现、能力矩阵、summary、phase timeline、registry/request contexts 和 recommendations。

## 验证
