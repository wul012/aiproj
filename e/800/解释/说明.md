# v800 route promotion governance snapshot 运行说明

## 本版目标

v800 接在 v798 decision index 和 v799 contract check 后面，把 accepted route ledger 与可重建校验合成治理快照。

本版输出的 `route_cards` 是后续模型能力治理模块可消费的状态卡：它说明 route 是否 accepted、是否 contract verified、是否仍在 tiny pair-probe boundary 内，以及允许的 downstream 使用范围。

本版不做重新训练，不新增模型质量结论，不把 tiny pair-probe 的 route promotion 解释成生产级能力。

## 运行命令

```powershell
python -B scripts\build_model_capability_route_promotion_governance_snapshot.py --decision-index e\798\解释\model-capability-route-promotion-decision-index --index-check e\799\解释\model-capability-route-promotion-decision-index-check --out-dir e\800\解释\model-capability-route-promotion-governance-snapshot --require-ready-snapshot --force
```
