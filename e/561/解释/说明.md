# v561 required-term pair equals-surface no-pair-id loss-balanced repair 运行说明

## 本版目标

v560 证明去掉 `pair=01` 后 `fixed=` 可以恢复，但 `loss=` 仍未命中。v561 在不带回 numeric pair id 的前提下，给 loss 分支增加轻量权重，验证这个剩余缺口是否能被目标平衡修复。

本版不扩大模型、不做 held-out 泛化声明，也不把单 seed 结果当成最终模型能力。它只确认 seed `1535` 在 no-pair-id + loss-balanced 语料下是否能达到 pair-full。
