# v785 registry release-readiness summary split

## 目标

v785 是维护拆分版本，不新增 registry 功能。目标是把 `registry_data.py` 中的 release-readiness comparison 读取、状态判定和 benchmark requirement delta 统计拆到独立模块，降低 registry 数据主模块的职责宽度。

## 关键变化

- 新增 `src/minigpt/registry_release_readiness_summary.py`。
- `src/minigpt/registry_data.py` 从 634 行降到 507 行。
- 保留既有 `src/minigpt/registry_release_readiness.py` 给 rankings 使用，避免模块撞名影响已有导入。
- 保留 `registry_data._as_optional_float` 私有兼容入口，避免 `registry_artifacts.py`、`registry_render.py`、`registry_leaderboards.py` 的历史导入断裂。
