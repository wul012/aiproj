# v550 required-term pair seed config held-out replay 运行说明

## 本版目标

v550 在 v549 selected-config replay 通过后，继续测试这些 selected checkpoints 是否能迁移到未直接训练的 prompt surface。

本版不重新训练，只把 selected seed/config 接到 held-out prompt specs：

- `colon-spaced`: `fixed: ` / `loss: `
- `equals`: `fixed=` / `loss=`
- `arrow`: `fixed -> ` / `loss -> `
