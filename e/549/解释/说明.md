# v549 required-term pair seed config replay 运行说明

## 本版目标

v549 承接 v548 的 per-seed config-selection policy，把策略重新接回真实源 stability report 和 checkpoint，验证 selected config 是否能重新 replay 出 pair-full。

本版不重新训练，只对 selected seed/config 做 generation-profile replay。

```powershell
python -B scripts\run_model_capability_required_term_pair_seed_config_replay.py `
```
