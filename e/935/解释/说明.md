# v935 randomized holdout publication registry lookup packet check 运行说明

## 本版目标

v935 消费 v934 lookup packet，并从其 source manifest review 重新构建 lookup packet：

```text
v934 lookup packet
  + source v933 manifest review
  -> v935 lookup packet contract check
```

它用于防止 lookup packet 被手工篡改，例如 lookup key 被改成 production 前缀、`rejected_use` 被移除、或 promotion 字段被打开。

明确不做：

- 不重新训练。
- 不重新 replay。
- 不改变 v934 lookup packet。
- 不批准 production promotion。
