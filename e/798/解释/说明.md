# v798 route promotion decision index 运行说明

## 本版目标

v798 接在 v797 route promotion review decision 后面，把最终接受的 bounded route promotion 决策写成可复用索引。

这个索引用于后续模型能力治理模块读取 accepted route、boundary、claim 和 source decision 路径。它不是新的训练结果，也不把 tiny pair-probe 结果升级成生产模型质量结论。

## 运行命令

```powershell
python -B scripts\build_model_capability_route_promotion_decision_index.py e\797\解释\model-capability-route-promotion-review-decision --out-dir e\798\解释\model-capability-route-promotion-decision-index --require-ready-index --force
```
