# v803 route promotion bounded benchmark suite 运行说明

## 本版目标

v803 接在 v802 consumer plan 后面，把 route promotion 的下游计划落实为具体 bounded benchmark suite。

本版新增 5 个 prompt cases，每个 case 都带有 PromptCase 兼容字段、expected terms 和 scoring contract。它仍然不重新训练模型，不运行生成，只固定后续评估要跑什么。

## 运行命令

```powershell
python -B scripts\build_model_capability_route_promotion_bounded_benchmark_suite.py --consumer-plan e\802\解释\model-capability-route-promotion-consumer-plan --out-dir e\803\解释\model-capability-route-promotion-bounded-benchmark-suite --require-ready-suite --force
```
