# v797 route promotion review decision 运行说明

## 本版目标

v797 接在 v796 route promotion release packet review 后面，把“packet 可进入 bounded review”的结论推进为最终的受限路线推广决策。

本版不重新训练模型，不扩大模型质量声明，也不把 tiny pair-probe 结果解释成生产级能力。它只确认 v796 review 的证据链足够干净，可以接受 `accept_bounded_route_promotion`。

## 运行命令

```powershell
python -B scripts\decide_model_capability_route_promotion_review.py --release-packet-review e\796\解释\model-capability-route-promotion-release-packet-review --out-dir e\797\解释\model-capability-route-promotion-review-decision --require-pass --force
```
