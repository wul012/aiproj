# v872 运行截图和解释：bounded objective loss signal bridge target-only memory patch

## 本版目标

v872 接在 v871 后面。v871 证明 single-line surface 训练后仍然有两个 `answer:` 回声和一个 `answeti` 标签碎片。v872 的目标是构造一个新的 no-anchor patch corpus，把修补重心从“单行标签 surface”转向“target-only completion memory”。

本版不训练，不 replay，也不声称模型能力提升。它只生成下一轮训练输入。

## 样本设计

v872 有三类核心样本：

```text
target_only_completion_memory
prompt_target_memory
label_target_memory
```

示例：

```text
fixed loss

Answer with exactly two tokens: fixed loss fixed loss

answer:
fixed loss
```

这版不是完全删除标签，而是降低标签密度：大部分样本只强化 `fixed loss`，少量保留最小 label -> target 桥接，避免下一轮训练完全忘记 replay prompt 里的 `answer:` / `completion:`。
