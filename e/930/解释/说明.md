# v930 randomized holdout publication registry entry contract check

## 目标

v930 读取 v929 registry entry，并从 entry 记录的 v928 source index 重新构建 registry entry，比较稳定字段是否一致。

它防止的风险是：

- registry entry 被手动篡改。
- source index 路径丢失。
- allowed use 被扩大。
- consumer boundary 被改成 production 用途。
- promotion 被误打开。

## 本版不做什么

- 不重新训练模型。
- 不重新 replay。
- 不生成完整 registry manifest。
- 不批准 production promotion。

## 运行命令

```powershell
python scripts\check_randomized_holdout_publication_registry_entry.py `
  e\929\解释\randomized-holdout-publication-registry-entry `
  --out-dir e\930\解释\randomized-holdout-publication-registry-entry-check `
  --require-pass `
  --force
```
