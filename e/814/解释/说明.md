# v814 运行截图与解释

## 本版目标

v814 用 v813 revised corpus 训练一轮新的 bounded repair checkpoint，并把训练产物收成 evidence。它解决的是“新的 revised seed 是否真的能产出可复核 checkpoint”的问题。

本版仍不宣称模型能力提升。训练 loss 下降和 checkpoint 存在只能说明训练产物完成，下一步必须用同一套 bounded replay 重新比较。

## 真实训练命令摘要


## 关键修复

本版顺手修复了 training evidence CLI 的 `--force` 行为：当 `--out-dir` 是包含 `run/` 的父目录时，旧逻辑会把 `run/` 一起删除。本版让 `--force` 保留嵌套 `run_dir`，只清理旧 report 文件。

这个修复已经由测试 `test_cli_force_preserves_nested_run_dir` 覆盖。
