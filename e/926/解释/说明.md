# v926 randomized holdout acceptance publication packet review 运行说明

## 本版目标

v926 读取 v925 publication packet，并做 bounded publication review。

它只批准：

```text
bounded_model_capability_governance_only
```

它不批准：

```text
production_promotion
general_model_quality
larger_corpus_transfer
```

本版不重新训练、不重新 replay，也不新增模型能力声明。

```text
e/925/解释/randomized-holdout-acceptance-publication-packet
```

## 运行命令

```powershell
python scripts\review_randomized_holdout_acceptance_publication_packet.py `
  --publication-packet e\925\解释\randomized-holdout-acceptance-publication-packet `
  --out-dir e\926\解释\randomized-holdout-acceptance-publication-packet-review `
  --require-review-ready `
  --require-publication-approval `
  --force
```


## 检查范围

v926 检查：

- v925 publication packet 必须 pass。
- v925 decision 必须 ready。
- packet 和 summary 都必须 ready。
- handoff 状态必须是 `ready_for_bounded_acceptance_publication_review`。
- accepted claim 必须正好 1 个。
- blocked claim 必须至少 3 个。
- allowed use 必须保持 bounded governance only。
- promotion 和 approved_for_promotion 必须保持 false。
- contract check 必须 ready。
- evidence rows 必须存在。
- v925 自身 failed check count 必须是 0。

v926 把 v925 publication packet 推进到 bounded downstream review 通过状态，但仍然明确阻止 production promotion。
