# v936 randomized holdout publication registry lookup index 运行说明

## 本版目标

v936 消费 v934 lookup packet 和 v935 lookup packet contract check，生成合并后的 governance lookup index：

```text
v934 lookup packet
  + v935 lookup packet contract check
  -> v936 lookup index
```

它让后续消费者不用分别读取 packet 和 check，就能拿到 lookup keys、lookup entries、contract check ready 状态和 promotion rejection 边界。

明确不做：

- 不重新训练。
- 不重新 replay。
- 不把 lookup index 改成 production promotion。
