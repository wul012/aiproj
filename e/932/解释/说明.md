# v932 randomized holdout publication registry manifest 运行说明

## 本版目标

v932 在 v931 registry packet 之后补上只读 manifest 层：

```text
v931 verified publication registry packet
  -> v932 bounded publication registry manifest
```

这一版做两件事：

- 生成 `randomized_holdout_publication_registry_manifest`，把已登记、已复核、可公开为 bounded claim 的 entry 汇总成 manifest。
- 把 randomized holdout publication 链路里的 `allowed_use`、`model_quality_claim`、`consumer_boundary`、`entry_id`、`next_step` 等重复常量收敛到 `randomized_holdout_publication_constants.py`。

明确不做：

- 不重新训练模型。
- 不重新 replay checkpoint。
- 不把 bounded publication 变成 production promotion。
- 不扩大 tiny checkpoint 的模型质量结论。
