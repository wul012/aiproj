# v906 运行截图与解释

## 本版目标

v906 根据 v905 review 的结论，构造 semantic paraphrase target-hidden holdout suite。

v905 已确认 v904 的 5/5 是强信号，但 5 个 prompt 都带有 `pair/target/answer_terms` 等任务提示。本版的目标是保留 tokenizer-covered 和 `fixed/loss` scoring contract，同时移除已知 task-hint terms。

本版只构造 suite，不运行模型，不声明 promotion。

## Prompt 变化

v906 用更少任务暗示的 prompt 替换 v902 的 `pair/target` prompt：

- `answer from memory\nanswer:`
- `write stored result\noutput:`
- `complete learned route\nresult:`
- `return final words\nanswer:`
- `self check memory result\nfinal:`

这些 prompt：

- 不含 `fixed/loss`。
- 不含已知 task hint terms：`learned pair`、`target pair`、`answer_terms`、`requires target pair`、`pair`。
- 仍然被 v890 tokenizer 覆盖。
- scoring contract 仍要求 continuation 同时包含 `fixed` 和 `loss`。
