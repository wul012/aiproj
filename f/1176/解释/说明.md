# v1176 运行解释：completion mask helper dedup

本版是 v1175 PTQ 之后的合同保持型维护版，不改变 PTQ 量化结论，也不重新训练模型。

## 目标

- 把 `(full_token_ids, prompt_length)` 样本的 padded `X` / completion-masked `Y` 构造抽到中性模块 `src/minigpt/completion_masking.py`。
- 保留 `distill_common._build_xy` 兼容入口，避免破坏 v1174/v1172 的显式导入合同。
- 让 `ptq_v1175.py` 不再维护自己的 `_padded_xy` 副本，和 distillation 共用同一 completion-mask 合同。
- 把用户要求的 `项目通俗说明/README.md` 接入根 README 文档地图。

## 关键文件

- `src/minigpt/completion_masking.py`：新增 `build_completion_xy`，唯一负责 completion-token mask 构造。
- `src/minigpt/distill_common.py`：把 `_build_xy` 改为 common helper 的兼容别名。
- `src/minigpt/ptq_v1175.py`：删除局部 `_padded_xy`，改用 `build_completion_xy`。
- `tests/test_completion_masking_v1176.py`：验证 prompt 忽略、completion 监督、distill alias 和 EOS-free 单位置合同。
- `README.md`：新增 `项目通俗说明/README.md` 文档地图入口。

## 验证

```text
python -m py_compile src\minigpt\completion_masking.py src\minigpt\distill_common.py src\minigpt\ptq_v1175.py tests\test_completion_masking_v1176.py tests\test_distill_common_v1174.py tests\test_ptq_v1175.py
python -m pytest tests\test_completion_masking_v1176.py tests\test_distill_common_v1174.py tests\test_ptq_v1175.py -q -o cache_dir=runs\pytest-cache-v1176-focused
```

结果：`26 passed`。

## 边界

本版不改变 v1175 的量化 verdict，不新增模型能力声称。它只把已经由 distillation 和 PTQ 两个实验家族共同使用的 completion-mask 逻辑单一来源化，降低后续 completion-only loss、KL、CE、scorecard 等实验出现 mask 分歧的风险。
