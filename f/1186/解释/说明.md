# v1186 运行说明：使用 canonical grokking checkpoint（推理 demo）

v1185 训练并保存了标准 grokking checkpoint。v1186 闭合产品化回路：提供一个极小的推理 API，把保存的 checkpoint **加载回来并真正用它算 `a + b (mod p)`**，再用 demo 从磁盘上的 `.pt` 独立复现 held-out 准确率、并解码出学到的模数加法表。无需训练，无需 GPU。

## 推理细节

训练时模型看的是 `[a, +, b, =, c]`，损失在 `=` 位置预测 `c`。使用时只喂 4 个 token 的 prompt `[a, +, b, =]`，读最后一个位置的 next-token logits，argmax 就是预测的 `c`——标准自回归解码。

```python
from minigpt.grok_checkpoint_v1185 import load_checkpoint
from minigpt.grok_predict_v1186 import predict
model, meta = load_checkpoint("f/1185/解释/grok_checkpoint_v1185/grok_checkpoint_v1185.pt")
predict(model, 36, 37, meta.p)   # -> 73   ((36+37) mod 97)
```
