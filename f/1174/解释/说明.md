# v1174 运行解释：distillation common helper dedup

本版是 v1172/v1173 蒸馏实验之后的合同保持型维护版，不新增模型能力结论，不重新解释暗知识实验结果。

## 目标

- 把 v1172 deterministic distillation 和 v1173 stochastic dark-knowledge distillation 共用的底层训练原语移到 `src/minigpt/distill_common.py`。
- 让 v1173 不再通过 `distill_v1172` 间接复用 helper，避免后续版本回改旧版本实验模块。
- 保留 v1172 原有显式导入合同，使已有测试和潜在调用方不需要迁移。

## 关键文件

- `src/minigpt/distill_common.py`：新增共享层，集中 `_build_xy`、`kl_term`、`shuffle_residual_mass`、`train_student`、`teacher_logit_stats`、`make_distill_model`。
- `src/minigpt/distill_v1172.py`：删除本地 helper 定义，改为从 common 导入并继续 re-export。
- `src/minigpt/distill_v1173.py`：改为直接依赖 common，不再从 v1172 模块穿透导入训练 helper。
- `tests/test_distill_common_v1174.py`：新增 common 合同测试，验证 re-export、mask、KL identity、模型工厂。
- `tests/test_distill_v1173.py`：mask/shuffle 测试改用 common 入口。

## 验证

```text
python -m py_compile src\minigpt\distill_common.py src\minigpt\distill_v1172.py src\minigpt\distill_v1173.py tests\test_distill_common_v1174.py tests\test_distill_v1172.py tests\test_distill_v1173.py
python -m pytest tests\test_distill_common_v1174.py tests\test_distill_v1172.py tests\test_distill_v1173.py -q -o cache_dir=runs\pytest-cache-v1174-focused
```

结果：`35 passed`。

## 边界

本版不修改 v1172/v1173 的实验结论，不新增训练轮次，不改变 distillation verdict。它只把已经被两个版本共同使用的训练原语从版本模块中移出，形成后续蒸馏实验可以安全复用的维护边界。
