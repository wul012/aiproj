# v1198 运行说明：归纳头的 OV 电路是一个"拷贝"电路（OV 半边）

v1197 因果验证了归纳机制的 **QK / 前缀匹配半边**（归纳头**注意到**后继位置）。v1198 验证**另一半边**——**OV（value→output）电路把被注意到的 token 拷贝进输出 logit**。两者合起来 = 教科书的二段式归纳头机制（Elhage 2021；Olsson 2022）。这镜像了 grokking 弧：一个权重级结果（FFT v1188）+ 一个因果结果（消融 v1191）；这里是 v1197 的因果 QK + v1198 的 OV 拷贝。

## 两条证据（**都**要满足才给正向 verdict；两者**不一致**→ review）

**Prong 1 —— 权重级 OV 拷贝。** OV 映射 `M = W_E @ W_V^T @ W_O^T @ W_E^T`（忽略 LayerNorm，Elhage 2021）对归纳头应当**对角占优**：注意到 token t 就抬高 logit t。逐 head 打分：`copy_z`（逐行 (M_ii − 行均值)/行标准差 的平均；随机≈0，正=拷贝，负=反拷贝）与 `diag_is_max`（对角元是该行 argmax 的比例；随机≈1/V）。

**Prong 2 —— 激活级 Direct Logit Attribution（LN 诚实的真值）。** 一个 head 对残差流的贡献 `c_h =（它的拼接 head 切片）@ W_O_h^T`，过**真实的 ln_f 缩放** + tied unembed，在 inductable 位置上对**正确答案** token 的 logit 加成。归纳头应当 `DLA_gap = logit(正确) − 平均 logit(错误) ≫ 0`，prev-token 头（为 L1 服务、不直接写输出）和非归纳 L1 对照≈0。DLA 用**真实注意力 + 真实 value**，补上了权重级理想化丢掉的东西（LayerNorm、layer-0 的 value 输入）。

## 真实结果（K=20，T=64，n_head=4，8 seed，全部学会归纳）

```text
verdict = induction_ov_is_copying_circuit  (status=pass)
base 0.9995（unigram 0.142，chance 0.050）；classifiable 7/8；τ 网格一致 100%

PRONG 1（权重级 OV copy_z，成对对照）：
  induction  +2.52   diag_is_max 0.89   <- 拷贝
  L1 对照    -1.66   （同一个 W_E！只是 W_V/W_O 不同）-> 反拷贝
  prev-token -0.45
  raw Gram   +3.83   （tied-embedding 基，见 lens-1）
  -> copy_ok = True
PRONG 2（激活级 DLA_gap）：
  induction  +0.566  （logit 正确 − 错误）
  L1 对照    -0.043
  prev-token -0.006
  -> dla_ok = True
```

**结论**：归纳头的 OV 电路是**拷贝**电路。**权重级**——它的 `M` 对角占优（copy_z +2.52，diag_is_max 0.89），而同数量的非归纳 L1 对照（**用同一个 W_E**）不对角（−1.66，反而反拷贝）、prev 头也不（−0.45）。**激活级**——它的实际输出在 inductable 位置上特异地抬高正确答案的 logit（DLA_gap +0.566），对照≈0。两条证据**一致**。

## lens-1：tied-embedding 的 Gram 混淆（不处理就是致命错）

模型把 unembed 与 embedding **绑定**，所以 `M` 两边是同一个 `W_E`，而**裸 Gram `W_E @ W_E^T` 本身就强对角占优**（copy_z +3.83）。所以 verdict **必须是成对/相对的**，绝不能是孤立地"归纳 OV 对角"。关键观察：**对照头共享同一个 `W_E`、只在 `W_V/W_O` 上不同，却不拷贝**——若对角是从 Gram 继承的，对照也该拷贝；它们没有（甚至 −1.66 反拷贝）。所以归纳 OV 的对角是 `W_V/W_O` **学**出来的（去**利用**那个拷贝友好的基），不是从嵌入 Gram 白捡的。图 C 把这点画出来：induction / control / prev 同基，只有 induction 拷贝。

## 诚实的闸（status=pass 只认有效测量）

- **逐 seed base 闸**：base<成功线 S 的 seed 没学会归纳、没有电路可解剖 → 排除（本版 8/8 都过）。
- **可分类闸**：≥80% 训练成功的 seed 要同时有"≥1 归纳头"和"≥1 非归纳 L1 对照头"。seed 1343 四个 L1 头全是归纳头 → 没有对照 → 不可分类（7/8=0.875 仍过）。
- **两条证据必须一致**：copy_ok≠dla_ok → review（`ov_dla_disagree`）。这是诚实机制（镜像 v1197 的 mean/zero 一致）。一致的零结果（两条都否）→ pass `induction_ov_not_copying`。
- **正号要求**：copy 不只要 beat 对照，还要 `copy_z ≥ 0.5` 且 `diag_is_max ≥ 0.30`（排除"反拷贝但 beat 一个更反的对照"的假阳）。
- **τ 稳健性扫描**：verdict 在 tau_ind 网格上不变（本版 100%）。

## 为什么 n_head=4（一个被诚实处理的工程事实）

v1197 用 n_head=8（head size 8）以便凑出"数量匹配的非电路对照"。但 head size 8 训练归纳**不可靠**：CPU 探针里 n_head=8 width-64 的模型在 2500 步都没越过归纳相变（base 0.50，没有归纳头形成）。换成 **n_head=4（head size 16）**——v1196 证明这是**可靠**学会归纳的基底——同一个 seed 600 步就到 0.997。归纳是经典的**相变**能力，head size 太小会把相变推到预算之外。v1198 因此用 n_head=4，并诚实记录这点。对照头改用"非归纳的 L1 头"（n_head=4 下通常 1 个），再加 prev 头作第二对照 + 8 seed，足够稳。

## 范围

2 层宽 64 MiniGPT、n_head=4、K=20/T=64。OV 拷贝是教科书结论（Elhage 2021），**新贡献**是在这个模型上做了**受控的多 seed、Gram-混淆感知（成对）+ DLA 互证**的诚实演示。**不是**关于 LLM 归纳头的断言。与 v1197（QK）合起来，在我们自己的 MiniGPT 上凑齐了二段式归纳头机制。

## 工程

Phase A（`run_induction_ov_v1198.py`）每 seed 训一个 2 层 n_head=4 模型，缓存逐 head 的 prev/induction 分数、逐 head 权重级 OV 拷贝分数（copy_z、diag_is_max）、逐 head 激活级 DLA gap、以及 tied-embedding Gram 基线。Phase B（`analyze_induction_ov_v1198.py`）纯 CPU 重判两条证据 + τ 扫描，零重训。DLA 用 c_proj 的 forward-pre-hook 抓拼接 head 输出、ln_f 的 forward-pre-hook 抓归一化前残差，按真实 ln_f 缩放折叠。复用 v1197 的 `head_scores`/`train_model`、v1196 的任务与 `make_model`。11 个测试。

一句话总结：v1198 在自己的 2 层模型上证明归纳头的 OV 电路是拷贝电路——权重级 OV 对角占优（copy_z +2.52 vs 同-W_E 对照 −1.66，diag_is_max 0.89）且激活级 DLA 特异地抬高正确答案（+0.566 vs 对照 ≈0），两条证据一致；成对对照钉死了 tied-embedding 的 Gram 混淆（对照同基却不拷贝），逐 seed base 闸 + 可分类闸 + 两证一致闸 + 正号要求 + τ 稳健保证 status=pass 只认有效测量；用 4 个 CPU 探针发现并改正了 n_head=8 训练不可靠（→ n_head=4）、校准了 DLA 阈值。与 v1197（QK）合成二段式归纳头机制。
