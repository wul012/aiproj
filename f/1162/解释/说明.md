# v1162 学习式绝对位置 vs RoPE：超过训练长度后的行为

## 目标

诚实测量「学习式绝对位置嵌入」与「RoPE」在**超过训练长度**时的唯一真实差异。两者都在短 `train_block_size=32` 上训练，再在 `{32,48,64,96,128}` 的更长窗口上评估留出集逐位置交叉熵，跑 5 个随机种子。本版**刻意不**声称「RoPE 能外推到更长序列」——在本语料上这个说法站不住脚（见下「诚实边界」）。

> 本版方向由一个对抗式设计评审（v1162 诚实性面板）重塑：它跑了真实代码，证伪了几个直觉但错误的说法，下面的结论都按面板要求落地。

## 运行命令

```powershell
python scripts\run_rope_length_extrapolation_v1162.py --device cuda --seeds 5
```

## 关键结果（真实 GPU 运行，RTX 4060，5 seeds）

- `status=pass`、`decision=length_extrapolation_measured`、`verdict=absolute_index_definedness_gap_confirmed`（结论由数据判定，不预设赢家）
- 长度 128 处：学习式「超范围 − 范围内」损失上升 **+0.1223 ± 0.0191** nats；RoPE 仅 **+0.0042 ± 0.0009**（基本持平）
- 范围内对照（L=32）：学习式 0.7776 ≈ RoPE 0.7783（同分布打平，与 v1160 一致）
- **滑动窗口（学习式现实部署）= 0.7767，与 RoPE 0.7783 打平** —— 证明 naive 长前向的差距是**推理方式的产物**，不是位置方案本身的缺陷
- 超范围 token 数下限 18960（统计充分）；`beyond_range_token_count_min` 远超 ~1000 门槛

## 诚实边界（面板要求，逐条落地）

1. **不是「学习式崩溃 / RoPE 存活」。** `seq_len>block_size` 的 ValueError 对两种方案**完全一样**地触发（受 `block_size` 与 `block_size×block_size` 掩码缓冲约束）；`block_size=32` 的 RoPE 模型同样在长度 128 处报错。证据：`both_schemes_raise_beyond_block_size=True`、`both_schemes_finite_when_rebuilt=True`。真正差别是**参数预算**：学习式要 `block_size` 行可训练参数（超范围那些行拿不到梯度），RoPE 需要**零**位置参数，其旋转对任意位置都有定义。
2. **不是相对距离外推。** 本语料是 ~11 字的独立句子，超范围位置上一切带信号的相对偏移（≤~11）训练时都见过；唯一新的是**绝对索引**，而 RoPE 按构造对其平移等变。
3. **不是「更长上下文有帮助」。** 句子相互独立，没有跨句依赖，任何 token 的有效历史不超过 ~11 字。本版测的是**能否在更长长度上保持有定义/可用**，不是上下文长度的收益。
4. **「灾难性」被证伪。** 退化是真实但温和的（128 处 +0.12 nats），不是 3–4 nats。
5. **零化尾行诊断（更精确的归因）：** 随机尾 0.8961 → 零化尾 0.7915 → 范围内 0.7776。分解为：噪声注入分量 **+0.1046**（把未训练行清零即可回收）vs 残余缺信号分量 **+0.0139**。即学习式超范围退化**主要来自随机未训练行注入的噪声**，而非单纯缺位置信号。
6. **未训练尾确实在 init 上：** `weight_decay=0.0` 使那些行梯度为零、衰减为零，`untrained_tail_l2_drift_max=0.0`（逐种子全 0）。若用 AdamW 默认 `weight_decay=0.01`，这些行每步会被解耦衰减，就不再「在 init 上」——故本版显式关掉衰减。
7. **naive RoPE 本身会在未见相对距离上退化**（Llama 形式、base=10000、无 NTK/YaRN）。长度 128 处约 **39.4%** 的注意力质量落在相对距离 >32 的区域，但本语料这些距离不带信号，所以暴露不出该退化——**不可**把「这里 RoPE 行」推广到自然长程文本。
8. **参数口径：** RoPE 模型仍分配了一张**未使用**的位置表，所以总参数相等只是巧合，RoPE 有效位置参数为 0。

## 一句话总结

v1162 用 5 种子、长度扫描、逐位置曲线和两条诊断臂（滑动窗口、零化尾），把学习式绝对位置与 RoPE 在超训练长度处的差异**诚实地**测清楚：差异是「学习式有未训练的位置行、RoPE 零位置参数且对任意位置有定义」，而**不是**「RoPE 外推更长依赖」或「学习式崩溃」——现实部署（滑窗）下学习式与 RoPE 在本语料上打平。
