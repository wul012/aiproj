# v1167 experiment_utils 去重保养

## 目标

按 AGENTS.md 的节奏（3-4 个功能版本后做 1 个契约保持的重构版本），把 v1164(SFT)、v1165(base→SFT 迁移)、v1166(DPO) 三个连续能力版本之后攒下的实验脚手架重复债还掉。这三个多 seed 实验驱动反复出现同样三个原语：把"每个 seed 一条记录"的列表聚合成 `(mean, std)`、从实验 config 新建一个 `MiniGPT`、给模型权重拍快照供之后 `load_state_dict`。

本版**不新增模型能力、不改训练超参、不改任何报告产物结构**，是一次纯保养（同 [[1171-v1159-minigpt-train-lm-dedup]] 与 v1163 的做法）。

## 改了什么

- 新增 `src/minigpt/experiment_utils.py`，提供三个原语：
  - `mean_std(values)`：忽略 `None` 与 `NaN`，空列表返回 `(nan, 0.0)`、单值 std=0；取 v1166 的 NaN-鲁棒版本（对 v1164/v1165 这种不含 NaN 的输入完全等价，对 v1166 的 `last_loss` 列必要）。
  - `build_minigpt(vocab_size, config, *, dropout=0.0)`：从带 `block_size/n_layer/n_head/n_embd/use_rope` 字段的 config 鸭子类型构建 `MiniGPT`，返回 CPU 模型（调用方 `.to(device)`），**RNG 流与原内联构建逐字节一致**，故初始化权重不变。
  - `clone_state(model)`：`{k: v.detach().clone() ...}` 的状态快照。
- 迁移三个能力模块到共享实现：
  - `sft_instruction_v1164.py`：删本地 `_mean_std`、内联 `MiniGPT(GPTConfig(...))`；改用 `mean_std` / `build_minigpt`。删掉因此不再用的 `import statistics` 与 `GPTConfig`。
  - `sft_pretrain_transfer_v1165.py`：删 `_build` / `_mean_std` / 内联 clone；改用 `build_minigpt` / `mean_std` / `clone_state`。
  - `dpo_preference_v1166.py`：删 `_build` / `_clone_state` / `_mean_std`；改用三个共享原语。
- 新增 `tests/test_experiment_utils.py`：覆盖 `mean_std`（基本统计、单值、空、忽略 None/NaN）、`build_minigpt`（按 config 字段构建 + 同 seed 确定性）、`clone_state`（取值相等且为独立副本）。
- 范围边界（刻意不动）：`_significant`（gap-减-合并标准差 的显著性判定）**只保留在 v1166 本地**——它当前只有一个使用方；v1164/v1165 是把同一判据内联写进各自的 `_gap`（且那里还要 gap 数值本身），强行抽取会制造单用户导入或改动其结构，得不偿失。`_param_l2_delta` 同理留在 v1166 本地。

## 契约保持的证据

- v1164/v1165/v1166 的既有模块测试**未做任何修改**仍全部通过——这是"行为没变"的关键证据（同 v1163 的判据）。
- `build_minigpt` 的确定性由 `test_experiment_utils` 钉死：同 seed 两次构建权重逐键相等；把构造器包进函数不改变 RNG 消耗。
- 全量测试：通过（v1166 的 3236 + 本版新增 experiment_utils 测试）。

- 结构图：`f/1167/图片/v1167-experiment-utils-dedup.png`（左 BEFORE：3 模块各自重复 mean_std×3 / build×3 / clone×2；右 AFTER：1 个共享模块，3 个模块导入）
- 运行证据：重构版本的运行证据就是全量测试套件全绿。

## 一句话总结

v1167 把 v1164/65/66 三个实验驱动里逐字节重复的 `mean_std` / 模型构建 / 状态快照三个原语收进新的 `experiment_utils.py`，删三处重复、迁三个模块、既有测试零改动全绿——一次干净的契约保持保养，给后续偏好/对齐实验留好可复用的脚手架。
