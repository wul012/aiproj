# v1118 运行截图和解释

## 本版目标

v1118 对 v1117 lookup-only receipt 做重建式 contract check。它从 v1117 receipt 中读取 source review，再重新构建一份 receipt，并与原始 receipt 对比。

本版不训练模型，不修改 v1117 receipt，也不批准 production promotion。

## 输入与输出

输入目录：

```text
f/1117/解释/receipt-v1117
```

输出目录：

```text
f/1118/解释/receipt-check-v1118
```

核心输出包括 JSON、CSV、TXT、Markdown 和 HTML。JSON 是 v1119 index 的机器可读输入，HTML 和截图用于人工复核 contract summary。
