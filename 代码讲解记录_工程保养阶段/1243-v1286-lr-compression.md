# 1243 - v1286 lr compression：平台压缩 8 倍后，F 守恒吗？

## 这一版回答什么问题

v1285 定量了经典深平台的雕刻：val 离开静息带前，最终回路功率增量的 ~60% 已经
就位（F ≈ 0.6）。v1281 又早已证明同一批格子在更高 lr 下 grok 快 8 倍（t_gen
11,400 → ~1,400）。两者拼起来是一个统一假说：**建造速率由有效 lr 设定，泛化在
建造完成时触发**——若为真，F 应在时间压缩下守恒（压缩的是钟表，不是机制）。
若压缩后的格子反而晚建造（F ≈ 0.2，像 v1284 的边界），则压缩改变的是建造与
泛化的关系本身，不只是速度。

## P1 与 bar 的第三次语义适配

P1（纯缓存）读 v1281 的八个 α=1 格子（lr ∈ {2, 4, 8}×1e-3）：t_mem 100–300、
t_gen 1,000–2,800，且**压缩平台静息在 val ≈ 0**（min 0.000–0.002；与 1e-3 深
平台的 0.11–0.21 带完全不同），末端起跳在几百步内扫过 0.1→0.9。同一条披露
规则（"bar 刚好高于静息带"）第三次应用：t_pre bar = {0.10, 0.15, 0.20}
（主 0.15）。F 公式、0.2/0.5 分箱、快照机制、前缀门全部沿用 v1284/v1285 原样。

## 实验设计：八个 v1281 格子、密尾梯子、逐位参照

- **格子（8）**：d=128、α=1、lr=2e-3 × {1337,1338}、4e-3 × 三 seeds、8e-3 ×
  三 seeds——就是 v1281 的 α=1 格子，带梯子重跑；G0 要求逐位复现 v1281 参照
  （t_mem/t_gen 相等、heldout 差 < 1e-6）+ 前缀门 + v1283 相位 = delayed。
- **梯子**：k ∈ {100…2600} 13 级、尾部加密（压缩起跳快）；每格 ≈ 全跑
  （≤3k）+ 梯子（≤13.2k）；8 格 ≈ 8–10 万步（预算 ≤112 次 / 13 万步）。
- 阶梯（预注册 `2b31c62d`）：三个 lr 的中位 F 全部 ≥0.5 →
  `construction_completion_invariant`；全部 ≤0.2 →
  `compression_switches_to_late_construction`；全部居中 →
  `partial_under_compression`；lr 间分箱不一致 → review（mixed_lrs）；lr 内
  离散（一 seed ≤0.2 且另一 ≥0.5）→ review（mixed_seeds）；G2 三 bar 一致。
- lr 经 `dataclasses.replace(cfg, lr=lr)` 传入 v1284 的 `run_cell`——继 v1285
  之后又一个"新代码只有配置 + 参照 G0 + 阶梯"的薄模块。
- P2 探针 = (4e-3, 1337) 整条梯子；双止损 = 前缀 / 参照。

## 结果：不变量被证伪——lr 解耦了建造与泛化

八格全部干净（前缀门 76/76、v1281 参照逐位复现、相位 8/8 delayed、预算
84/112 次 / 63,400 步）。F（bar 0.15）：2e-3 → {0.315, 0.076}（中位 0.196）、
4e-3 → {0.270, 0.281, 0.226}（0.270）、8e-3 → {0.380, 0.280, 0.233}（0.280）。
2e-3 组的中位恰好骑在 late/partial 分箱线上（0.196 vs 0.2），三个 bar 给出
不同判决 → **`review`（bar_instability）**——G2 兜底防护对真实骑线数据
恰好履职，预注册路由未改一字。

review 里的确定内容分两层。第一层：**建造完成不变量死了**——压缩下 F 从
1e-3 的 0.56–0.72 掉到 ~0.25 一带，泛化不再等建造接近完成。第二层（相对
时间图揭示）更微妙：压缩格在 grok 时刻的**绝对**份额其实更高（0.45–0.55 vs
慢格的 0.29）——F 低是分母的功劳：**高 lr 的回路在 grok 之后继续大幅纯化**，
终点份额随 lr 单调上升（0.31@1e-3 → 0.55–0.59@2e-3 → 0.62–0.72@4e-3 →
0.71–0.81@8e-3）。lr 不只是把钟表拨快：建造被加速的程度超过泛化，两者解耦，
而且同宽度下终点回路纯得多——v1285 刚闭合的"宽度-纯度梯度"原来是
lr 条件性的（第几次了？这条弧上每个"梯度"最后都通向 lr）。

三区域收束图景：慢深 grokking（F≈0.6，grok 后不再纯化）/ 压缩 grokking
（F≈0.25，grok 后强纯化）/ 相位边界（F≈0.25，v1284）。只有慢深 grokking
里"泛化时刻"等于"建造完成时刻"。
