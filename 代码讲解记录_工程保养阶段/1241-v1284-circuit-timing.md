# 1241 - v1284 circuit timing：grokking 的平台期在造回路，还是在等？

## 这一版回答什么问题

v1283 把基底切成两个动力学相：耦合相（w≤20 与 w=24 的一个 seed——train/val
同升、延迟精确为 0、从无记忆平台）与延迟/grokking 相（w≥28）。这一版的 P1 探针
先从 v1277 提交缓存解决了终点问题：**耦合相到达同一类解——傅里叶频率回路——
而且浓度更高**（w=16 的 top-5 功率份额 0.85–0.92，w=32 只有 0.68–0.72）。剩下
的就是时机问题，它区分 grokking 平台期的两个机制故事：

- **慢雕刻**（经典叙事）：平台期就是回路的渐进建造——傅里叶结构在记忆期间
  累积，回路建成时泛化到来。预测：val 开始动之前，最终回路的大部分功率已经
  就位。
- **等待后速成**：平台期是空转；回路在末尾快速成形，与耦合相同一个日程。
  预测：两个相里 val 动之前都几乎没有回路功率。

## 方法：确定性截断重训 = 零改动的权重快照

训练是种子确定性的（v1279 跨会话逐位复现过）。用更小的 max_steps 重跑
`train_to_grok` 会复现同一轨迹的前缀——返回的模型就是第 k 步的快照，经典训练
代码零修改。方法自带验证：每个截断跑的最后一行曲线必须与完整跑在该步的行
逐位一致（**前缀确定性门**；失败即方法失效 → review）。这个门把"重复训练的
冗余"变成了证据。

## 实验设计：6 条轨迹、一把梯子、一个分数

全部 α=1、lr=1e-3（经典配方）。

- **轨迹格（6）**：(w=20, 1337/1338) 与 (w=24, 1337) 预期耦合；(w=24, 1338) 与
  (w=28, 1337/1338) 预期延迟——w=24 这对是宽度匹配的跨相比较，同一宽度、同一
  配方，只有 seed 决定相。
- **每格**：一次完整跑（定义 steps_run、最终谱与其 top-5 频率集），然后在梯子
  k ∈ {100, 200, 400, 700, 1000, 1400, 1900, 2600, 3400, 4400, 5600}（k <
  steps_run）上逐点截断重跑。每快照缓存 48 维功率谱与 C(k) = 最终回路频率集
  的功率份额；C₀ = 种子化初始处该集合的份额（CPU 免费）。
- **判决量**：预泛化结构分数 F = (C(t_pre) − C₀)/(C_final − C₀)，t_pre = val ≤
  0.1 的最后一个快照，截到 [0,1]。
- 阶梯（预注册 `74970587`）：F_delayed ≥ 0.5 且 F_coupled ≤ 0.2 →
  `plateau_is_circuit_construction`（慢雕刻成立）；两者 ≤ 0.2 →
  `construction_is_rapid_in_both`（平台在等待，经典慢雕刻叙事在本基底证伪）；
  两者 ≥ 0.5 → `structure_precedes_in_both`；其余 → review。终点门：耦合相
  最终份额中位 < 0.5 → `coupled_phase_uses_different_solution`（P1 预示不会
  触发，但保留）。G0 = 前缀确定性 + 相位匹配 v1283 + heldout；G2 = t_pre bar
  ∈ {0.05, 0.1, 0.2} 判决一致。
- P2 探针 = (w=28, 1337) 整条梯子（最清晰的延迟格，延迟 3,200），止损条件 =
  前缀门失败。预算 ≤ 76 次运行 / 26 万总训练步（预计 ~10 万步）。

## 实现要点

`src/minigpt/grok_circuit_timing_v1284.py`：训练走 v1185 `train_to_grok` +
v1279 `scaled_init`（与 v1283 的格子同一初始化路径，保证相位可比）；谱走
v1188 的 `number_embedding`/`embedding_spectrum`；相位分类直接 import v1283
的 `classify_phase`。测试 7 项：F 数学（截断、缺 t_pre、退化分母）、配置校验、
G0 三条路径 + 完整性、四个判决分支 + 终点覆盖、注入快照函数 + P2 预载编排
（假快照函数第一版把 val 跳变写成相对 run 长度、被自己的前缀门当场抓住——
门的第一次立功发生在测试里）、字节稳定契约、图像落盘。

## 结果：平台不拥有建造——两相共享一条回路生长曲线

P2 探针先验证了方法（前缀确定性 10/10 快照逐位通过），并预示了走向：w=28/1337
的回路份额从第 100 步就开始连续攀升（0.16 → 记忆点 0.41 → 0.65 → 终点 0.76），
val 在"平台"期间从 0.08 漏爬到 0.53——边界附近的平台不是 d=128 那种死平台。
全部 6 格出来后（前缀门 43/43 全过、相位 6/6 匹配 v1283、heldout 0.963–0.988、
预算 49/76 次运行、60,600/260,000 步）：

预泛化结构分数 F：耦合 {0.275, 0.472, 0.251}（中位 0.275），延迟 {0.240,
0.308, 0.219}（中位 0.240）——**两个相几乎相同**，全部落在预注册的 (0.2, 0.5)
中间带。没有任何分离模式触发，判决按阶梯路由 **`review`（mixed_fractions）**，
三个 t_pre bar 下一致（G2 过）；终点门也未触发（耦合相最终份额中位 0.883）。

review 里的确定内容比任何预注册分支都干净：**两相在同一个相对日程上建造
回路**。val 越过 0.1 时两相都已就位约四分之一的最终回路功率；C(t) 曲线形状
相同，延迟相只是时间拉伸——w=24 跨相对把混杂钉死（同宽度同配方同初始化
分布，F 0.251 vs 0.240，延迟 seed 的建造曲线 ~3× 膨胀）。经典"慢雕刻"叙事
在本基底需要修正：结构确实在平台期间增长，但它在平台之前、在没有平台的
耦合相里，都以同样的相对斜率增长——**平台不拥有建造；区分两相的是 train
accuracy 相对建造曲线的行为，不是建造本身**。

描述性副产物：最终 top-5 份额随宽度平滑下降（0.88–0.91 @20 → 0.81–0.85 @24
→ 0.76 @28，与 v1277 缓存的 0.85–0.92 @16、0.68–0.72 @32 无缝衔接）——越窄
的模型回路越"纯"；边界附近记忆本身也慢（t_mem 1600–2300 vs d=128 的
100–300），平台是"漏"的。

## 诚实边界与自审

范围：own grokked substrate、toy scale、经典配方、6 个预注册格（全部在相位
边界附近 w∈[20,28]）；**未测 d=128 的经典深平台**（val 钉在 chance 数千步的
那种）——F 在深平台是否也 ≈0.25 是 banked 的直接后续（梯子成本 ~40k+ 步/格）；
F 的分辨率受梯子密度限制；"同一日程"基于 3+3 格的中位数。decide() 自审：
预注册 `74970587` 后判据与代码**零改动**（三连清洁链）；F 阈值在新数据前
冻结于 0.2/0.5；P2 唯一止损（前缀门）未触发——但前缀门在测试阶段抓住过假
快照函数把 val 跳变写成相对 run 长度的错误，方法的自验证在上线前就立了功。
mixed_fractions → review 是预注册路由的正式产出，均匀性本身是可报告的发现。
