# 1245 · v1288 spike anatomy：谁在把 grok 后的解推下山脊？

## 一、版本定位：从"现象"到"元凶"

v1287 的底座防护抓到了比原问题更大的东西：grok 后继续训练时，解是**亚稳**
的——11 格里 9 格出现 train+val 同时崩塌又在 100–300 步内自愈的尖峰，两个
"死亡格"只是恰好在视界终点被截尾在尖峰中间。v1288 执行 banked 的尖峰解剖，
问最锋利的因果问题：**是什么在反复破坏已经泛化的解？**

头号嫌疑人是 weight decay：wd·lr 在解处持续收缩权重、纯化在推进（v1287），
自然的机理故事是"wd 雕刻到解滑出稳定边缘 → 优化器弹射 → 重新 grok"。
若为真，在解处**移除 wd** 应当消灭尖峰——并且同时**冻结纯化**（v1287 的
两个现象是同一个 wd 过程的两张面孔，一个预注册的绑定预测）。

## 二、设计：成对分支臂（paired branch arms）

对 9 个尖峰易发格，确定性重跑到一个**分支点**（v1287 快照梯子上健康度已被
提交缓存实测的步：P1 逐格核验九个分支点 val 0.99–1.0，并把 8e-3/1338 的
1.4× 点——它恰好落在尖峰中间 val=0.567——换成已验证健康的 1.8× 点），
抓取权重态，然后**两条臂延续到 v1287 视界**：wd=1.0 与 wd=0.0。

设计要点：
1. **共享混淆**：两臂都在分支点重置 AdamW 动量——optimizer-reset 混淆
   两臂对称，臂间唯一差异就是 wd。若连 wd=1 臂都不再尖峰，说明尖峰需要
   积累的优化器历史（`spikes_need_optimizer_history`，同样有信息量的出口）。
2. **训练代码零改动**：分支 = `train_to_grok(init_state=...)`；全批次训练
   不消耗 RNG，分支确定性成立；早停仍用 grok_stop_val=2.0 关闭。
3. **G0 逐位锚定**：每格分支重跑的整条曲线必须与已提交 v1287 缓存逐位相等
   （1e-9）——分支态因此就是 v1287 轨迹上的那个态，不是"近似复刻"。
4. **终点 heldout 明确不作门**（brief 披露的偏离）：臂可以恰好结束在尖峰
   里，那是数据不是底座故障——v1287 censoring 教训的直接制度化。
5. **反截尾跑（2 个，描述性）**：把两个死亡格整跑延长 1,000 步（观测自愈
   ≤300 步，3 倍余量），检验"截尾"解释。

判据：尖峰 = val < 0.9（网格 {0.8, 0.85, 0.9}）的极大连续段；形态防护
（非截尾段 >500 步 = 不是被研究的现象）。S1/S0 = 两臂各自有尖峰的格数：
S1≥5 且 S0=0 → `spikes_are_wd_driven`；S0≥3 → `spikes_persist_without_wd`；
S1=S0=0 → `spikes_need_optimizer_history`；其余 review。次级（不驱动判决）：
纯度冻结与范数方向。预算 29 runs / 117,000 步，预注册 `b07f1b1f`；
CI-parity 闸门（v1287 教训升格的新规）在预注册前全绿。

## 三、P2 探针与 Phase A

探针（4e-3/1337，6,400 步）四条止损全过，预览已见雏形：wd1 臂两个尖峰
（0.90/0.74，各 ≤100 步自愈，恰好对应连续跑同窗的 2 个），wd0 臂零尖峰。
Phase A 29 runs 精确命中预算；G0 全绿（9 格分支曲线逐位复现 v1287 缓存、
分支健康最低 0.991、反截尾前缀逐位）。

## 四、结果：因果收束

**判决 `spikes_are_wd_driven`（G2 三 bar 稳定）——自 v1280 以来第二个
非 review，本弧第一个正面因果判决。**

- **S1=8/9、S0=0**：wd=1 臂复现 14 个尖峰（最深 0.311，全部自愈）；
  wd=0 臂九格干净（min val 全 ≥0.909）。尖峰计数与连续跑同量级（14 vs
  15）但逐格不重合（canonical 臂 0/连续 2；8e-3/1337 臂 1/连续 0）——
  尖峰是该动力学区域的统计性质，不是固定日程。
- **反截尾**：两个"死亡格"都在旧视界后 **100 步内**满血复活
  （0.431→0.996、0.373→0.999）。4e-3/1338 的 preregistered 旗标读 False
  只因它的 +1000 新端点又恰好撞进一个浅涡（0.881）——端点抽签的讽刺
  递归重演，本身就是亚稳论断的再确认。
- **范数方向**：wd1 中位比 0.953（收缩）、wd0 **1.753**（无 wd 一路增长）。
- **纯度冻结**：中位数是 wash（两臂 ≈0）——天花板效应，压缩格分支点纯度
  已 0.73–0.93；唯一有余量的 canonical 格给出教科书解离：wd1 爬 0.372→
  0.574、wd0 冻在 0.373（探索性标注，如实报告 wash 不藏）。

机理拼图闭合：**纯化与亚稳性是 wd 雕刻的一体两面**——wd 在 grok 后继续
收缩并雕刻回路，同一只手周期性把解推下稳定边缘，数百步后重新 grok；
移除 wd 则尖峰消失、范数反转、纯化冻结。v1287 的"早停恰好保护了 canonical
配方"由此有了机理解释：它停在了 wd 还没来得及推下山脊的时刻。

## 五、纪律与 banked

预注册后判据与代码**零改动**（第七连清洁链）；次级读数未混入判决；
中位数 wash 与旗标 False 都如实报告并给出全曲线证据。banked：尖峰的
微观触发（范数/锐度在尖峰前的斜坡，需要稠密范数采样）；canonical 频率集
旋转动力学（v1287 另一条 banked 未动）。
