# 1242 - v1285 deep plateau：经典深平台在雕刻，还是在等？

## 这一版回答什么问题

v1284 在相位边界（w=20–28）发现两相共享一条回路建造曲线（F ≈ 0.25），"平台
不拥有建造"。但那些格子的平台都又短又漏。慢雕刻叙事唯一可能幸存的地方是它
最初被讲述的地方：**d=128 经典配方的深平台**——t_mem=100、t_gen 10,500–12,700、
一万步的间隔。这一版把同一把尺子搬过去：F 还是 ≈0.25（v1284 的修正推广到经典
grokking），还是 ≥0.5（深平台真的在雕刻，v1284 的发现是边界效应）？

## P1 与一个诚实的度量适配

P1（纯缓存，免费）给了两个设计输入：(1) 三个 v1279 格子的精确参照值（t_mem
全部 =100——第一次 eval 就完成记忆；t_gen 11,400/12,700/10,500）；(2) 一个
出乎意料的事实：**经典平台不在 chance 处休息**——val 从第 100 步起就 ≥0.11，
在 ~0.11–0.21 带内漂移，6k 处下探 0.14，然后渐进爬升（0.38@8k、0.52@10k）再
跳变。v1284 的 t_pre bar {0.05, 0.1, 0.2} 低于这条静息带，会人为制造 F=0。
语义常数是"刚好高于静息带"，因此本版 bar 改为 {0.25, 0.30, 0.35}（主 0.30），
在预注册**之前**披露；F 公式、阈值（0.2/0.5）、梯子机制、前缀门全部沿用
v1284 原样。

- **格子（3）**：d=128、α=1、lr=1e-3、seeds {1337, 1338, 1339}——就是 v1279
  网格的那三格，带完整梯子重跑。G0 要求完整跑**逐位复现** v1279 参照
  （t_mem、t_gen 相等，heldout 差 < 1e-6）+ 前缀门 + v1283 相位分类 =
  delayed。max_steps 60000（常数 lr 的 max_steps 不变性已由 v1279/v1283/v1284
  三次跨配置复现证实）。
- **梯子**：k ∈ {100 … 12400} 15 级，每格 ~60–73k 步，3 格 ≈ 200k 步
  （预算上限 50 次 / 240k 步）。
- 阶梯（预注册 `46fc9734`）：F 中位 ≥0.5 → `deep_plateau_sculpts`；≤0.2 →
  `construction_is_late_everywhere`；中间 → `partial_early_construction`；
  离散防护（一格 ≤0.2 且另一格 ≥0.5）→ review（mixed_seeds）；G2 = 三个 bar
  下判决一致。
- P2 探针 = seed 1337 整条梯子；止损 = 前缀门失败或参照不复现。

## 结果：深平台真的在雕刻——自 v1280 以来第一个非 review 判决

P2 探针（seed 1337）就把方向定了：前缀门 15/15、参照逐位复现、F ≈ 0.72。
全部三格出来后：F = {0.720, 0.556, 0.600}，中位 **0.600 ≥ 0.5**，三个 bar 下
判决一致、离散防护未触发、G0/G1/G2 全过——**`deep_plateau_sculpts`**。预算
46/50 次、191,600/240,000 步；三格的完整跑全部逐位复现 v1279 参照
（t_mem=100、t_gen 11,400/12,700/10,500、heldout 差 < 1e-6）。

数字说话：val 离开静息带之前，最终回路功率增量的 56–72% 已经就位；C(t) 沿
整条一万步的平台缓慢单调爬升（0.15 → 0.30），没有末端速成。**慢雕刻叙事在
它最初被讲述的地方幸存**——而 v1284 在相位边界测得的 F ≈ 0.25（"平台不拥有
建造"）由此正式定性为边界效应，resolution note 已回写 v1284 简报。两版合璧：
回路建造在相位门附近晚到且与泛化锁定，在深 grokking 里早启且平台主导。同一
把尺子（同一 F 公式、同一前缀验证的快照法）量出两个区域的真实机制差异。

意外的奖励：三个终点份额 0.305/0.312/0.305 把 v1188 的历史测量（top-5 share
0.307）三重复现，宽度-纯度梯度闭合为 0.92@16 → 0.88@20 → 0.85@24 → 0.76@28
→ 0.71@32 → 0.31@128。经典平台"静息在 val 0.11–0.21 而非 chance"这个 P1
发现也值得记住：记忆解从第一步起就带着少量泛化成分，纯粹的"死平台"在这个
基底上并不存在。
