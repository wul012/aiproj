# v47 运行截图和解释

## 本版主题

v47 在 run registry 里新增跨 run 的 pair delta 聚合视图。v46 已经能让 registry 打开每个 run 的 pair batch/trend 报告；v47 进一步回答“多个 run 里，哪些 prompt case 的左右 checkpoint 生成长度差异最大”。

## 关键改动

- `src/minigpt/registry.py`：新增 `pair_delta_summary` 和 `pair_delta_leaderboard`。
- registry JSON：保存跨 run 的 case delta 统计、最大 generated/continuation delta 和 top case 列表。
- registry HTML：新增 `Pair deltas` 统计卡和 `Pair Delta Leaders` 面板，显示 run、case、abs generated delta、原始 delta、equal 状态、task、checkpoint pair 和报告链接。
- `tests/test_registry.py`：覆盖跨 run delta 聚合、排序、summary 字段、HTML 面板和相对链接。
- `README.md`、`b/README.md`、`代码讲解记录_评估基准阶段/README.md`：同步 v47 能力、截图索引、tag 和下一步。

## tag 含义

`v47.0.0` 表示 MiniGPT 第四十七个可复查学习版本：registry 不只索引 pair 报告，还能跨 run 聚合出最值得复查的 pair delta case。

## 一句话总结

v47 把“多个 run 可以打开 pair 报告”推进到“多个 run 的 pair 差异可以被集中排序和解释”。
