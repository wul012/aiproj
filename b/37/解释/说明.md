# v37.0.0 标记说明

v37 的主题是 `baseline model comparison`。本版把 `scripts/compare_runs.py` 从基础多 run 对比升级为带 baseline 的模型比较报告，用来比较 tokenizer、模型大小、训练步数和数据版本是否一致。

## 本版新增/修改

- `src/minigpt/comparison.py`
  - `RunComparison` 新增 `dataset_version`、`dataset_fingerprint`、token count 和 `model_signature`
  - `build_comparison_report` 新增 `baseline` 参数
  - 新增 `baseline_deltas`、`summary` 和 `recommendations`
  - 新增 `render_comparison_markdown` / `render_comparison_html`
  - `write_comparison_outputs` 现在输出 JSON/CSV/SVG/Markdown/HTML

- `scripts/compare_runs.py`
  - 新增 `--baseline`
  - CLI 输出 baseline、summary 和新增产物路径

- `tests/test_comparison.py`
  - 覆盖 baseline 选择、delta 计算、模型签名、dataset version、Markdown/HTML 输出和 HTML 转义

- `代码讲解记录_评估基准阶段/52-v37-baseline-model-comparison.md`
  - 说明目标、边界、字段语义、测试链路和截图证据

## 标记含义

`v37.0.0` 表示 MiniGPT 已经具备 baseline model comparison 的工程雏形。它不只是列出多个 run，而是能围绕一个 baseline 解释候选模型的 loss 是否改善、参数量代价、tokenizer/model signature 是否变化，以及 dataset version 是否保持一致。
