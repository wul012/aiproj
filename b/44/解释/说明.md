# v44.0.0 标记说明

## 本版主题

v44.0.0 的主题是 pair batch trend comparison：在 v43 已经能生成单份 `pair_generation_batch.*` 的基础上，新增趋势比较脚本，把多份已保存的 pair batch 报告放在一起，输出跨版本或跨 checkpoint pair 的 JSON、CSV、Markdown 和 HTML 趋势报告。

## 关键改动

- `src/minigpt/pair_trend.py`：新增 pair batch trend report builder，汇总 report-level 指标、case-level equality variants、最大字符 delta 和 CSV/Markdown/HTML 渲染。
- `scripts/compare_pair_batches.py`：新增命令行入口，读取多份 `pair_generation_batch.json`，支持 `--name` 命名，并写出 `pair_batch_trend.*`。
- `src/minigpt/__init__.py`：导出 pair trend 报告构建与写入函数。
- `tests/test_pair_trend.py`：覆盖趋势 summary、case 变化检测、报告加载、name 数量校验和四类输出文件。
- `README.md`、`b/README.md`、`代码讲解记录_评估基准阶段/README.md`：同步 v44 能力、命令、截图和讲解索引。
- `代码讲解记录_评估基准阶段/59-v44-pair-batch-trends.md`：解释本版在评估链路中的位置、字段语义、CLI 流程、输出证据和测试覆盖。
