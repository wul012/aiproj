# v51 运行截图和解释

## 本版主题

v51 给 benchmark scorecard 增加 rubric-style per-prompt correctness scoring。v50 已经能按 task type 和 difficulty 找到弱项分组；v51 继续向下，把每个 prompt 的基本正确性检查写成 rubric 分数，避免 benchmark 只看长度、多样性和 pair 稳定性。

## 关键改动

- `src/minigpt/benchmark_scorecard.py`：新增 `rubric_scores`、schema v3、`rubric_correctness` 组件、required/forbidden terms、长度检查、任务形态检查、weakest rubric case 和 rubric CSV。
- `scripts/build_benchmark_scorecard.py`：CLI 输出新增 rubric status、average score 和 weakest rubric case。
- `tests/test_benchmark_scorecard.py`：覆盖 schema v3、rubric component、rubric CSV、weakest case、missing terms、Markdown/HTML rubric 区块和 drilldown rubric 字段。
- `README.md`、`b/README.md`、`代码讲解记录_项目成熟度阶段/README.md`：同步 v51 能力、tag、截图索引和后续方向。
- `代码讲解记录_项目成熟度阶段/66-v51-benchmark-rubric-scoring.md`：解释 rubric scoring 如何承接 v50 drilldown。

## tag 含义

`v51.0.0` 表示 MiniGPT 第五十一个可复查学习版本：benchmark scorecard 不再只评估输出形状，也能用轻量 rubric 表达每个 prompt 的基本正确性。

## 一句话总结

v51 把“benchmark 能定位弱项分组”推进成“benchmark 能解释每个 prompt 的基本正确性得分”。
