# v40.0.0 标记说明

v40 的主题是 `checkpoint comparison shortcuts`。本版在 v39 checkpoint selector 基础上增加 `/api/checkpoint-compare` 和 playground 对比表，让用户在本地页面里快速看到候选 checkpoint 的文件状态、tokenizer、参数量、dataset version 和相对 default 的差异，并能直接切换或打开 model-info。

## 本版新增/修改

- `src/minigpt/server.py`
  - 新增 `build_checkpoint_compare_payload`
  - 新增 `/api/checkpoint-compare`
  - `/api/health` 返回 `checkpoint_compare_endpoint`
  - checkpoint 比较行新增 `size_bytes`、`metadata_run_dir`、`model_info_endpoint`、`parameter_delta`、`same_dataset_version` 等字段
  - `/api/model-info?checkpoint=<id>` 能优先读取候选 checkpoint 父目录中的 metadata

- `src/minigpt/playground.py`
  - 新增 `Checkpoint Compare` 区块
  - 页面加载 `/api/checkpoint-compare`
  - 表格展示 status、size、params、dataset、delta
  - 每行提供 `Use` 和 `Model info` 快捷动作

- `scripts/serve_playground.py`
  - 启动时打印 `checkpoint_compare=http://.../api/checkpoint-compare`

- `tests/test_server.py`
  - 覆盖 comparison payload、metadata run dir、参数差异、dataset 差异、HTTP compare endpoint

- `tests/test_playground.py`
  - 覆盖 compare table、`/api/checkpoint-compare` 加载和 `selectCheckpoint` 动作 wiring

- `代码讲解记录_评估基准阶段/55-v40-checkpoint-comparison-shortcuts.md`
  - 说明目标、边界、字段语义、API 行为、前端动作、测试链路和截图证据

## 标记含义

`v40.0.0` 表示 MiniGPT 的本地 playground 已经支持 checkpoint 快速比较：服务端能输出候选 checkpoint 的状态和差异，前端能显示对比表，并把对比结果转成选择 checkpoint 或查看 model-info 的直接动作。
