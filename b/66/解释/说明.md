# v66 运行截图说明：release-quality maturity narrative

## 本版主题

v66 新增 release-quality maturity narrative。它读取 maturity summary、registry release readiness trend、request history summary、benchmark scorecard 和 dataset card，把分散证据合成一份 JSON/Markdown/HTML portfolio 叙事报告。

本版不训练模型、不扩大数据集、不重新计算 benchmark 分数，也不替代 release gate。它的作用是把已有证据链整理成更适合答辩、展示、交接和版本回顾的一页入口。

## 关键命令

```powershell
python -m unittest tests.test_maturity_narrative
python -m unittest discover -s tests
python -m compileall src tests scripts
python scripts/build_maturity_narrative.py --project-root <fixture> --maturity <maturity_summary.json> --registry <registry.json> --request-history-summary <request_history_summary.json> --benchmark-scorecard <benchmark_scorecard.json> --dataset-card <dataset_card.json> --out-dir <out>
powershell -ExecutionPolicy Bypass -File scripts/playwright_chrome_smoke.ps1 -UrlOrPath <maturity_narrative.html> -Out b/66/图片/05-playwright-maturity-narrative-html.png
```

## Tag 含义

`v66.0.0` 标记 “MiniGPT v66 release-quality maturity narrative”。

这个 tag 的简单解释是：本版把分散的成熟度、发布趋势、请求历史、benchmark 和数据卡证据合成一份面向 portfolio 的成熟度叙事报告，并补齐 CLI、测试、README、代码讲解和运行截图归档。

## 一句话总结

v66 把项目从“有很多独立证据报告”推进到“能用一份成熟度叙事报告统一解释证据链、状态和下一步”。
