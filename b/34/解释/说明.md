# v34.0.0 标记说明

v34 的主题是 `configurable release gate delta baseline`。v33 已经能解释 compared profile 相对 baseline profile 新增或移除了哪些 failed/warned checks；本版继续向后，把 baseline 从“默认第一个 selected profile”扩展为可以通过 `--baseline-profile` 显式选择。

## 本版新增/修改

- `src/minigpt/release_gate_comparison.py`
  - `build_release_gate_profile_comparison` 新增 `baseline_profile`
  - 新增 `_resolve_baseline_profile`
  - report 顶层写入 `baseline_profile`
  - summary 写入 `baseline_profile`
  - `_build_profile_deltas` 按显式 baseline 选择 baseline row
  - Markdown/HTML summary 显示 baseline profile

- `scripts/compare_release_gate_profiles.py`
  - CLI 新增 `--baseline-profile`
  - 输出新增 `baseline_profile=...`

- `tests/test_release_gate_comparison.py`
  - 覆盖默认 baseline 兼容性
  - 覆盖显式 `baseline_profile="review"`
  - 覆盖 baseline 不在 selected profiles 里的错误提示
  - 覆盖 JSON/Markdown/HTML 的 baseline 输出

- `代码讲解记录_发布治理阶段/49-v34-configurable-release-gate-baseline.md`
  - 说明目标、边界、baseline 解析规则、CLI 行为、输出产物、测试覆盖和归档证据

## 标记含义

`v34.0.0` 表示 MiniGPT 的 release gate profile comparison 不再只能靠 profile 顺序隐含 delta baseline，而是可以显式选择 baseline profile，并把这个参照系写入 JSON、Markdown、HTML、CLI 日志和归档截图。
