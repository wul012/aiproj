# v50 运行截图和解释

## 本版主题

v50 给 benchmark scorecard 增加 task type / difficulty drilldown。v49 已经能把一个 run 的 eval suite、generation quality、pair batch 和 registry context 合成总分；v50 继续往下解释总分，把 case-level 证据按任务类型和难度分组，指出最弱的 benchmark 切片。

## 关键改动

- `src/minigpt/benchmark_scorecard.py`：新增 `drilldowns` 字段、schema v2、分组评分、最弱 task/difficulty 摘要、Markdown/HTML drilldown 区块和独立 drilldown CSV。
- `scripts/build_benchmark_scorecard.py`：CLI 输出新增 task/difficulty 分组数量和最弱分组。
- `tests/test_benchmark_scorecard.py`：覆盖 schema v2、task/difficulty group count、weakest group、drilldown CSV 和 HTML/Markdown drilldown 区块。
- `README.md`、`b/README.md`、`代码讲解记录_项目成熟度阶段/README.md`：同步 v50 能力、tag、截图索引和后续方向。
- `代码讲解记录_项目成熟度阶段/65-v50-benchmark-scorecard-drilldowns.md`：解释 drilldown 如何承接 v49 scorecard。

## tag 含义

`v50.0.0` 表示 MiniGPT 第五十个可复查学习版本：benchmark 不再只有总分，也能解释到任务类型和难度分组。

## 一句话总结

v50 把“run 级 benchmark 总分”推进成“能定位任务切片短板的 benchmark drilldown”。
