# v52 运行截图和解释

## 本版主题

v52 把 v51 的 benchmark rubric correctness 分数带入 run registry。单个 scorecard 已经能说明每个 prompt 的 rubric 得分；本版让多个 run 可以按 rubric 平均分排名，看到 correctness regression，并在 registry CSV/HTML 中直接打开 scorecard。

## 关键改动

- `src/minigpt/registry.py`：读取 `benchmark-scorecard/benchmark_scorecard.json`，新增 rubric status、avg score、rank、delta、weakest case、leaderboard 和 regression summary。
- `scripts/register_runs.py`：CLI 输出新增 `benchmark_rubric_counts` 和 `benchmark_rubric_summary`。
- `tests/test_registry.py`：覆盖 rubric 字段读取、registry summary、CSV 字段、HTML Rubric 列、Rubric Leaderboard、scorecard 链接和排序选项。
- `README.md`、`b/README.md`、`代码讲解记录_项目成熟度阶段/README.md`：同步 v52 能力、tag、截图索引和后续方向。
- `代码讲解记录_项目成熟度阶段/67-v52-registry-benchmark-rubric-tracking.md`：解释 registry rubric tracking 如何承接 v51 rubric scoring。

## tag 含义

`v52.0.0` 表示 MiniGPT 第五十二个可复查学习版本：benchmark rubric correctness 从单 run scorecard 进入多 run registry。

## 一句话总结

v52 把“单个 run 的 rubric 正确性分数”推进成“多个 run 的正确性排名和退化证据”。
