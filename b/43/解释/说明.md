# v43.0.0 标记说明

## 本版主题

v43.0.0 的主题是 fixed prompt pair generation batches：在 v42 已经能保存单次左右 checkpoint 生成对比的基础上，新增一套批量脚本，把固定 prompt suite 同时跑在两个 checkpoint 上，并输出可横向扫描的 JSON、CSV、Markdown 和 HTML 报告。

## 关键改动

- `src/minigpt/pair_batch.py`：新增 pair batch report builder，记录每个 prompt 的左右输出、相等标记、字符长度 delta、task/difficulty 汇总和报告渲染。
- `scripts/pair_batch.py`：新增命令行入口，加载 `data/eval_prompts.json` 或自定义 suite，调用左右 checkpoint，并写出 `pair_generation_batch.*`。
- `src/minigpt/__init__.py`：导出 pair batch 报告构建与写入函数。
- `tests/test_pair_batch.py`：覆盖 case result、report summary、JSON/CSV/Markdown/HTML 输出。
- `README.md`、`b/README.md`、`代码讲解记录_评估基准阶段/README.md`：同步 v43 能力、命令、截图和讲解索引。
- `代码讲解记录_评估基准阶段/58-v43-pair-batch-comparison.md`：解释本版在评估链路中的位置、字段语义、CLI 流程、输出证据和测试覆盖。

## 标签含义

`v43.0.0` 表示 MiniGPT 已经从“单次 pair generation 可以留档”推进到“同一套固定 prompts 可以批量比较两个 checkpoint，并生成稳定的本地对比报告”。
