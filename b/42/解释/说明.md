# v42.0.0 标记说明

## 本版主题

v42.0.0 的主题是 persisted pair generation artifacts：在 v41 已经支持 `/api/generate-pair` 的基础上，新增 `/api/generate-pair-artifact`，把同一个 prompt 下左右 checkpoint 的生成结果保存成可复查的本地 JSON/HTML 报告。

## 关键改动

- `src/minigpt/server.py`：新增 `write_pair_generation_artifacts`、`render_pair_generation_html`、`/api/generate-pair-artifact`，并把 artifact JSON/HTML 路径写入 `inference_requests.jsonl`。
- `src/minigpt/playground.py`：在 side-by-side 区块增加 `Generate & Save Pair` 按钮、保存状态和 HTML artifact 链接。
- `scripts/serve_playground.py`：启动输出新增 `generate_pair_artifact` 端点。
- `tests/test_server.py`：覆盖 artifact 文件写入、record schema、HTML 内容、日志 artifact 路径和保存端点健康检查。
- `tests/test_playground.py`：覆盖保存按钮、保存状态和 `/api/generate-pair-artifact` 前端 wiring。
- `代码讲解记录_评估基准阶段/57-v42-pair-generation-artifacts.md`：解释本版在评估链路中的位置、保存端点、artifact schema、HTML 报告、日志证据和测试覆盖。

## 标签含义

`v42.0.0` 表示 MiniGPT 已经从“能实时左右对比两个 checkpoint”推进到“能把一次左右对比保存成可打开、可复查、可被日志追踪的本地实验证据”。
