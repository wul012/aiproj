# v41.0.0 标记说明

v41 的主题是 `side-by-side checkpoint generation`。本版在 v40 checkpoint comparison shortcuts 基础上增加 `/api/generate-pair` 和 playground 双栏生成视图，让同一个 prompt 可以同时发给左右两个 checkpoint，并返回输出相等性、长度 delta 和 pair request 日志。

## 本版新增/修改

- `src/minigpt/server.py`
  - 新增 `GenerationPairRequest`
  - 新增 `parse_generation_pair_request`
  - 新增 `/api/generate-pair`
  - `/api/health` 返回 `generate_pair_endpoint`
  - pair response 返回 `left`、`right` 和 `comparison`
  - `inference_requests.jsonl` 新增 `/api/generate-pair` 事件

- `src/minigpt/playground.py`
  - 新增 `Side-by-Side Generate` 区块
  - 新增左右 checkpoint selector
  - 新增 `Generate Pair` 按钮
  - 左右输出分别显示 `data.left.generated` 和 `data.right.generated`
  - 状态栏显示 `generated_equal` 和字符长度 delta

- `scripts/serve_playground.py`
  - 启动时打印 `generate_pair=http://.../api/generate-pair`

- `tests/test_server.py`
  - 覆盖 pair request 解析、缺失 right checkpoint 拒绝、HTTP pair generation、左右 checkpoint 路由和 pair request log

- `tests/test_playground.py`
  - 覆盖左右 selector、pair generate 按钮、`/api/generate-pair` 和 payload 字段

- `代码讲解记录_评估基准阶段/56-v41-side-by-side-generation.md`
  - 说明目标、边界、字段语义、API 行为、前端动作、日志证据、测试链路和截图证据

## 标记含义

`v41.0.0` 表示 MiniGPT 的本地 playground 已经支持双 checkpoint 现场生成对照：同一个 prompt 和同一组采样参数可以同时调用左右 checkpoint，返回两个输出和一组简短比较字段，并把 pair request 写入 JSONL 证据。
