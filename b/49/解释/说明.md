# v49 运行截图和解释

## 本版主题

v49 做 benchmark scorecard 收口。前面已经有 eval suite、generation quality、pair batch、pair delta leaders 和 registry context，本版把这些分散证据合成一个 run 级评分报告，输出 JSON/CSV/Markdown/HTML，避免继续只拆 links/trends/dashboard。

## 关键改动

- `src/minigpt/benchmark_scorecard.py`：新增 scorecard 构建、评分、case 合并和 JSON/CSV/Markdown/HTML 渲染。
- `scripts/build_benchmark_scorecard.py`：新增命令行入口，读取一个 run 目录并写出 `benchmark-scorecard`。
- `tests/test_benchmark_scorecard.py`：覆盖组件评分、registry context、输出文件和 HTML 转义。
- `README.md`、`b/README.md`、`代码讲解记录_项目成熟度阶段/README.md`：同步 v49 能力、tag、截图索引和后续方向。
- `代码讲解记录_项目成熟度阶段/64-v49-benchmark-scorecard.md`：解释 scorecard 在项目成熟度阶段的位置。

## tag 含义

`v49.0.0` 表示 MiniGPT 第四十九个可复查学习版本：项目把分散的评估证据合成为一个统一 benchmark scorecard，能更直接说明某个 run 的评估状态。

## 一句话总结

v49 把“评估证据到处都有”推进成“一个 run 可以被统一打分和解释”。
