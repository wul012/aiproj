# v67 运行截图说明：training portfolio pipeline

## 本版主题

v67 新增 training portfolio pipeline。它把数据准备、真实训练、固定 prompt 评估、生成质量分析、benchmark scorecard、dataset card、registry、maturity summary 和 maturity narrative 串成一个可 dry-run 或 `--execute` 的端到端训练组合跑。

本版不声称 smoke 模型能力已经变强。它解决的是“如何从一个数据源稳定生产一整套训练和评估证据”的工程问题。

## 关键命令

```powershell
python -m unittest tests.test_training_portfolio
python -m unittest discover -s tests
python -m compileall src tests scripts
python scripts/run_training_portfolio.py <sources> --out-root <out> --dataset-name <name> --dataset-version <version>
python scripts/run_training_portfolio.py <sources> --out-root <out> --dataset-name <name> --dataset-version <version> --execute --device cpu --max-iters 2
powershell -ExecutionPolicy Bypass -File scripts/playwright_chrome_smoke.ps1 -UrlOrPath <training_portfolio.html> -Out b/67/图片/05-playwright-training-portfolio-html.png
```

## Tag 含义

`v67.0.0` 标记 “MiniGPT v67 training portfolio pipeline”。

这个 tag 的简单解释是：本版新增可 dry-run/execute 的训练组合跑入口，把数据、训练、评估、质量分析、scorecard、数据卡、registry、成熟度 summary 和成熟度叙事连接成一条可复现链路，并补齐测试、README、代码讲解和运行截图归档。

## 一句话总结

v67 把项目从“成熟度叙事能读已有证据”推进到“一个命令能重新生产训练 portfolio 证据链”。
