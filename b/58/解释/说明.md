# v58 运行截图和解释

## 本版主题

v58 在 v57 的 Request History 表格基础上补“筛选和导出”。v57 已经能把 `inference_requests.jsonl` 读成最近请求列表；v58 让这个列表可以按 `status`、`endpoint`、`checkpoint` 筛选，并通过 `format=csv` 导出当前筛选结果。

## 关键改动

- `src/minigpt/server.py`：新增 request history 过滤参数、匹配计数、CSV 字段列表、`request_history_to_csv` 和 `format=csv` 响应。
- `src/minigpt/playground.py`：新增 status/endpoint/checkpoint/limit 控件、Export CSV 链接、筛选 query 构造、匹配数量展示和错误提示。
- `tests/test_server.py`：覆盖过滤组合、checkpoint 过滤匹配 pair 请求、CSV header/行内容、HTTP CSV 响应类型。
- `tests/test_playground.py`：覆盖筛选控件、Export CSV 链接、query 构造和匹配数量展示。
- `README.md`、`b/README.md`、`代码讲解记录_项目成熟度阶段/73-v58-request-history-filters-export.md`：同步 v58 能力、截图和讲解索引。

## tag 含义

`v58.0.0` 表示 MiniGPT 第五十八个可复查学习版本：本地推理请求历史从“能回看最近记录”推进到“能筛选当前关注的请求并导出 CSV”。

## 一句话总结

v58 把 Request History 从浏览表格推进成可筛选、可导出的本地推理复查入口。
